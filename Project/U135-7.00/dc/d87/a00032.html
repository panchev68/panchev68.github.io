Watchdog.hpp File Reference

Detailed Description

This file contains the declaration of the Watchdog class for handling the Independent Watchdog (IWDG) peripheral.

The Watchdog class provides a singleton interface to initialize and refresh the IWDG peripheral. It ensures that the system is periodically reset if the software becomes unresponsive.
