Client

Detailed Description

Handles RS232 communication with the server.

The Client class is responsible for handling RS232 communication with the server. It inherits from the BSP::ComPort and Protocol classes, providing functionalities for message receiving and sending callbacks, character received event handling, message parsing, and message building.
