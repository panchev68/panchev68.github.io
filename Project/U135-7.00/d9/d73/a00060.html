Protocol

Detailed Description

Handles RS232 communication, including parsing and building messages.

The Protocol class is responsible for handling RS232 communication, including parsing and building messages. It supports commands such as ECHO, UPDATE_IO, and RESTART.

The class uses FreeRTOS for real-time operating system functionalities and includes helper utilities for CRC16 checksum calculation and byte swapping.
