Detailed Description

Communication port interface utilizing UART with DMA.

The ComPort class is a communication port interface utilizing UART with DMA. It inherits from the Lib::HAL::Uart_DMA class and provides methods to open the port and send data. The class is designed to be extended by derived classes that implement the handleCharReceivedEvent method to process received characters.
