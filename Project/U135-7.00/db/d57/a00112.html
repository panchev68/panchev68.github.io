SerialPortRepeater

Detailed Description

A class that extends Lib::USB::SerialPortRepeaterBase to handle serial port repeating functionality.

This class provides methods to open and close the serial port, check if it is powered by USB, enable or disable the USB port, and set the module reset pin. It also manages timeouts for receive and transmit LEDs.

#include <SerialPortRepeater.hpp>
