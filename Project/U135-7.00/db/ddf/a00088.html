Leds::Led

Detailed Description

A class to control an LED using a pin handle.

The Led class provides methods to set the LED state and update it based on a user-defined function.

◆ tick()

void tick ( )
inline

Updates the state of the LED pin based on the onUpdate callback.

This function is called periodically to update the state of the LED pin. If the onUpdate callback is not null, it calls the callback and writes the returned value to the LED pin.
