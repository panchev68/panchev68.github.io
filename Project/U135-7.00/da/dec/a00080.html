Leds

Detailed Description

The Leds class for managing LED operations.

The Leds class is a part of the BSP (Board Support Package) and inherits from the Lib::BSP::DriverBase class. It provides functionality to control and update the state of various LEDs on the board.

The class includes the following nested classes:

  • Config: A nested class for configuring LED pins and update period.
  • Led: A nested class representing an individual LED with methods to set its state and update it.

The Leds class uses FreeRTOS for task management and includes methods for initialization and handling tick events.
