Button

Detailed Description

Button class for handling button press, release, and hold events.

The Button class is responsible for handling button press, release, and hold events. It inherits from the DriverBase class and uses FreeRTOS for timing functionalities.

◆ handleTickEvent()

void handleTickEvent ( )
inlineoverride

Handles the tick event for the button.

This function is called periodically to check the state of the button pin. It updates the button state based on the pin reading and triggers the appropriate callbacks for press, release, and hold events.

  • If the pin state changes, it updates the internal state and triggers the onPress or onRelease callback.
  • If the button is in the pressed state and the hold period has elapsed, it transitions to the hold state and triggers the onHold callback.
Note
This function overrides a virtual function from the base class.
