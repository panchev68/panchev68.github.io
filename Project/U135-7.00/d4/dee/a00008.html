Detailed Description

Header file for the RS232 Client class.

This file contains the declaration of the Client class, which is responsible for handling RS232 communication. The Client class inherits from BSP::ComPort and Protocol classes, providing functionalities for message receiving and sending callbacks, character received event handling, message parsing, and message building.
