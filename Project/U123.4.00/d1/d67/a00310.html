Lib::BSP::DigitalPeripheralBase::Test Class Reference

Detailed Description

Test class.

This class provides functionality for testing the digital peripheral outputs. It includes methods to start and stop the test, set the test mode, and set the change period. The test mode determines the pattern of output values, while the change period determines the time interval between changes.
