App::ApplicationBase Class Referenceabstract

Detailed Description

Base class for application components.

The ApplicationBase class serves as a base class for various application components. It provides a pure virtual function that must be overridden by derived classes to return the component's name.

Note
This class is intended to be used as a base class only.
