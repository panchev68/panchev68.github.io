HAL::LcdDataPort Class Reference

Detailed Description

A class for interfacing with the LCD data port using GPIO.

The LcdDataPort class is a singleton that extends the GPIO::Port class from the Lib::HAL::GPIO namespace. It provides an interface for writing data to the LCD in half-byte and full-byte formats.
