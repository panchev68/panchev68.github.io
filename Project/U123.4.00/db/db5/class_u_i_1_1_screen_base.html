UI::ScreenBase Class Reference

Detailed Description

A base class for screen components in the UI.

The ScreenBase class provides a foundation for creating screen components in the UI. It includes callback functions for handling button click and hold events, as well as virtual methods for managing screen activation, deactivation, and tick events. Derived classes can override these virtual methods to implement specific behavior.

Note
This class is intended to be used as a base class and should not be instantiated directly.
