Align.hpp File Reference

Detailed Description

This file contains the definition of the Align class, which is used to align strings to the left, center, or right within a fixed number of characters per line.

The Align class provides an enumeration for alignment modes (LEFT, CENTER, RIGHT) and calculates the starting position for the string based on the selected mode and the fixed number of characters per line.
