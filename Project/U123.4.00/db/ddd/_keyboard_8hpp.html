Keyboard.hpp File Reference

Detailed Description

This file contains the declaration of the Keyboard class for handling keyboard input and button events.

The Keyboard class is responsible for handling keyboard input, detecting button presses, and managing hold button events. It utilizes FreeRTOS for task management and queues for event handling.

The Keyboard class interacts with the following components:

  • HAL::EXTI7: External interrupt handler for detecting button presses.
  • FreeRTOS::Queue: Queues for handling keycode and button detection events.
  • FreeRTOS::Timer: Timer for managing hold button events.
  • FreeRTOS::Timeout: Timeout for debouncing button presses.

The Keyboard class provides the following functionalities:

  • Setup and initialization of the keyboard and its components.
  • Sampling of keycodes from the keycode queue.
  • Updating the keyboard state based on button events.
  • Handling hold button events and generating corresponding keycodes.
Note
This implementation assumes the use of FreeRTOS for task and queue management.
