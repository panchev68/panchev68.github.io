UI::Screen::PeripheralOutputs Class Reference

Detailed Description

A singleton class derived from ScreenBase that handles peripheral output screens.

The PeripheralOutputs class manages the peripheral output screens and provides functionality to handle tick events and update the screen. It uses FreeRTOS for timeout management.
