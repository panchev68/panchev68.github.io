UI::Screen::ResetState Class Reference

Detailed Description

A singleton class for managing the reset state screen.

The ResetState class is a singleton that inherits from ScreenBase and provides functionality to manage the reset state screen in the UI.

virtual void UI::ScreenBase::clearButtonsFunction ( )

Clears the all button functions.

This method clears all button functions, setting them to nullptr. It can be called to remove any existing button functions from the screen. This is useful when switching between screens to ensure that no residual button functions are active.
