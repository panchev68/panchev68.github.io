ScreenBase.hpp File Reference

Detailed Description

Defines the ScreenBase class for UI components.

This file contains the definition of the ScreenBase class, which serves as a base class for different screens in the UI. It includes various callback functions for button click and hold events, and provides virtual methods for handling screen activation, deactivation, and tick events.

This file is part of the U123-Firmware project.
