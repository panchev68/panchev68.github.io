BSP::Lcd Class Reference

Detailed Description

The Lcd class provides an interface to control an LCD display. It uses a configuration structure to set the number of characters per line and the number of lines on the display.

The class offers methods to setup the display, update its content, retrieve the buffer, get the buffer size, and clear the display.
