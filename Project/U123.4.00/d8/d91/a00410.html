UI::EditDateBase Class Reference

Detailed Description

A base class for editing date values in the UI.

The EditDateBase class provides an interface for editing date values in the UI. It inherits from the EditValueBase class and provides functionality for incrementing and decrementing date values, as well as handling button events for date editing.
