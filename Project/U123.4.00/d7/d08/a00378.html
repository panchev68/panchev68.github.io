System Class Reference

Detailed Description

A class that represents the system and provides various functionalities.

The System class is responsible for managing the overall system operations. It provides a singleton instance to ensure that only one instance of the System class exists throughout the application. The class also contains a nested Parameters class that handles system parameters.

Note
This class inherits from Lib::System::System.
