Detailed Description

The RotationTableSerial class is responsible for managing the rotation table in serial mode.

Rotation table serial application

Introduction

The RotationTableSerial application is responsible for managing the rotation table in serial mode. It includes the Control class, which handles the control logic of the application, and the Parser class, which parses incoming data and processes commands.

Command format

The RotationTableSerial application uses a simple command format to control the rotation table. The command format consists of a command byte, a port byte, and a payload of four bytes. The command byte specifies the type of command, the port byte specifies the port number, and the payload contains additional data for the command.

The command format is as follows:

Example:

`P0FFFF` - Set port 0 to 0xFFFF
`P1FFFF` - Set port 1 to 0xFFFF
`P10000` - Set port 1 to 0x0000
`P00000` - Set port 0 to 0x0000
