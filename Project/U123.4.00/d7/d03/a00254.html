App::RemoteIO Class Reference

Detailed Description

The RemoteIO class is responsible for managing remote I/O operations.

The RemoteIO class extends the ApplicationBase class and provides functionality for handling remote I/O operations. It includes a constructor to initialize the class with a specific baud rate, a destructor, and a method to get the name of the remote I/O instance.

This constructor initializes the RemoteIO object with a specific baud rate. It is responsible for setting up the remote I/O communication. The baud rate is used to configure the communication speed for the remote I/O. The constructor takes the baud rate as a parameter and initializes the remote I/O communication with the specified baud rate.
