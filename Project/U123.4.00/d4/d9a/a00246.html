App::RemoteIOBase Class Reference

Detailed Description

The RemoteIOBase class is responsible for managing remote I/O operations.

The RemoteIOBase class extends the ApplicationBase class and provides functionality for handling remote I/O operations. It includes a constructor to initialize the class with a specific baud rate, a destructor, and a method to get the name of the remote I/O instance.
