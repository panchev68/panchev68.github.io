UI::Screen::HardwareVersion Class Reference

Detailed Description

A screen class for displaying the hardware version.

The HardwareVersion class inherits from ScreenBase and provides functionality to display the hardware version on the screen. It includes methods to handle button events and activate the screen.

Note
This class follows the singleton design pattern.
