MenuBase.hpp File Reference

Detailed Description

Declaration of the MenuBase class, which represents a base class for menu screens in the UI.

This file contains the definition of the MenuBase class, which inherits from ScreenBase and provides functionality for handling menu items and button events. The class uses an initializer list to initialize menu items and provides methods to activate, deactivate, and set the position of the menu.
