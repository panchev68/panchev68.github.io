EditValueBase.hpp File Reference

Detailed Description

This file contains the declaration of the EditValueBase class, which serves as a base class for editing values in the UI.

The EditValueBase class provides an interface for handling value editing, including incrementing and decrementing positions and values, as well as handling button events. It also defines the necessary enums, structs, and pure virtual functions that derived classes must implement.

The class includes:

  • A constructor and destructor.
  • Methods for getting and setting the edit mode.
  • Methods for getting separators.
  • Pure virtual functions for incrementing and decrementing positions and values, and for getting a string representation of the value.
  • A method for handling button events.

The class uses the BSP::Keyboard::Keycode type for handling key events.

This class is part of the UI namespace.
