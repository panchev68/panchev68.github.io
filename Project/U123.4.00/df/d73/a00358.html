BSP::Led Class Reference

Detailed Description

A class for controlling an LED.

The Led class provides methods to set the LED to different modes such as OFF, ON, and BLINK. It is implemented as a singleton to ensure only one instance of the LED control exists. The class also handles the setup and tick events for the LED.
