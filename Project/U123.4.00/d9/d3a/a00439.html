UI::Screen::EditProduceDate Class Reference

Detailed Description

A screen class for editing the produce date.

This class inherits from ScreenBase and provides functionality to edit the produce date. It includes methods to handle button events and activate the screen.

Note
This class follows the singleton design pattern.
