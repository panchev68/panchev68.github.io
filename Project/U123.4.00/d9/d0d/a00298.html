Lib::BSP::DigitalPeripheralBase::Outputs Class Reference

Detailed Description

Manages the digital outputs and SPI communication for a peripheral.

This class handles the configuration, setup, and updating of digital outputs through SPI communication. It provides mechanisms to handle value changes, status updates, and SPI errors.

Note
This class is designed to work with the MC33879 SPI port expander.
