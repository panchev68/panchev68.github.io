UI::EditValueBase Class Referenceabstract

Detailed Description

A base class for editing values with keyboard interaction.

The EditValueBase class provides a framework for editing values using keyboard interactions. It defines the basic structure and functionality required for editing values, including handling separators, modes, and button events.
