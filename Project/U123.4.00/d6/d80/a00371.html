HAL::EXTI7 Class Reference

Detailed Description

A class for handling external interrupts on pin 7 of GPIOC.

The EXTI7 class is a singleton that extends the EXTIx class from the Lib::HAL::GPIO namespace. It provides an interface for handling external interrupts on pin 7 of GPIOC.
