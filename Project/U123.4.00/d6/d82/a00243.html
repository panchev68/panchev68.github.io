App::ApplicationManager Class Reference

Detailed Description

The ApplicationManager class is responsible for managing the lifecycle of applications within the project.

The ApplicationManager class is a singleton class that manages the lifecycle of applications within the project. It provides methods to run applications and retrieve the name of the active application.
