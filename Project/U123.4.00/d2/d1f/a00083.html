ComPort.hpp File Reference

Detailed Description

This file contains the declaration of the ComPort class and its associated classes and methods.

The ComPort class provides an interface for managing communication over a serial port. It includes methods for opening and closing the port, setting a parser for incoming data, setting the baud rate, and sending data.

The ParserBase class is an abstract base class for implementing custom parsers for incoming data.
