App::RemoteIO57600 Class Reference

Detailed Description

A class that represents a remote I/O interface with a baud rate of 57600.

This class inherits from RemoteIOBase and initializes the base class with a baud rate of 57600. It is used to manage remote I/O operations at the specified baud rate.
