Detailed Description

ADC Port implementation for Block Mode.

Provides a concrete implementation of the PortBase interface for block mode ADC operation, supporting open, close, conversion control, and value reading. This class ensures correct configuration and management of the ADC hardware in block mode, facilitating integration with STM32 embedded applications that require precise analog input handling and robust error checking.
