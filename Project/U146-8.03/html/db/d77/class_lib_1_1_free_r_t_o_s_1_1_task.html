Detailed Description

Represents a FreeRTOS task and provides methods for task lifecycle management and control.

This class encapsulates the creation, execution, suspension, resumption, and deletion of FreeRTOS tasks, offering both static and instance-level interfaces for flexible task management in embedded applications.
