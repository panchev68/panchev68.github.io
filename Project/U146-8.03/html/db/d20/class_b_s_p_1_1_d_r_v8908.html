Detailed Description

DRV8908 motor driver abstraction.

This class encapsulates register operations, error management, and output control for DRV8908 chips. It provides high-level access to device features, including register read/write, output mapping, and error/status monitoring for embedded motor control applications. The class is intended for use in systems requiring robust and efficient management of DRV8908 devices, supporting both configuration and runtime diagnostics.

See also
https://www.ti.com/product/DRV8908-Q1

enum class ErrorState : uint8_t
strong

DRV8908 error states.

Enumerates the possible error states reported by the DRV8908 device. These states indicate fault conditions such as power-on reset, overvoltage, undervoltage, overcurrent, open load, or overtemperature errors detected by the chip. The enumeration is used for error reporting and diagnostics in embedded applications.

◆ Register

enum class Register : uint8_t
strong

DRV8908 register addresses.

Enumerates the register addresses used for DRV8908 device communication and configuration. Each value corresponds to a specific register in the chip, enabling control and monitoring of device features such as output control, PWM, open load detection, and status reporting.
