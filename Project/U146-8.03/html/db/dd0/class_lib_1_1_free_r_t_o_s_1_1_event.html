Detailed Description

template<typename T>
class Lib::FreeRTOS::Event< T >

Encapsulates a FreeRTOS single-item event queue for type-safe signaling between tasks and ISRs.

This class template provides methods for sending and receiving events of type T, supporting both task and ISR contexts, and manages the underlying FreeRTOS queue resource.
