Detailed Description

TestPeripheralOutputs class declaration for peripheral output test sequencing.

This file declares the TestPeripheralOutputs class, which provides an interface for running, controlling, and sequencing test patterns on peripheral outputs in the application. It enables starting, stopping, and configuring test patterns for hardware outputs, as well as querying and updating the test state.
