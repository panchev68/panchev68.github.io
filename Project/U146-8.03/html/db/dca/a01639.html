Lib::HAL::SPI::Master::Port< DMA > Class Reference

SPI Master DMA port specialization. More...

Description

Implements SPI master communication using DMA for STM32 HAL. Provides open, close, init, deinit, send, receive, sendReceive, and event callback support. All operations use the SPI handle provided at construction.
