Detailed Description

SPI master port in interrupt (IT) mode.

Provides non-blocking transmit, receive, and transceive operations with event hooks for SPI master in IT mode. This class wraps STM32 HAL SPI interrupt API, enabling asynchronous SPI communication and event-driven callbacks. It supports initialization, deinitialization, abort, and completion event handling for SPI transactions. All operations are performed in interrupt mode, allowing efficient, non-blocking data transfers.
