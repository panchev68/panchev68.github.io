Detailed Description

template<typename T>
class Lib::Helper::Container< T >

Container template for managing collections of pointer elements.

This class provides a type-safe, flexible interface for storing, adding, removing, and processing pointers to objects of type T. It supports batch operations, access to the underlying vector, and efficient management of object pointers for embedded and performance-critical applications.

Template Parameters
TType of the objects pointed to by the container elements.
