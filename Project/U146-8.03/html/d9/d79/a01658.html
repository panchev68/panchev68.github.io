Detailed Description

Timer specialization for encoder mode.

Specializes the Timer class for STM32 TIM encoder mode. Provides methods for initialization, shutdown, and value change event notification for quadrature encoder decoding. Integrates with STM32 HAL and supports interrupt-driven operation.
