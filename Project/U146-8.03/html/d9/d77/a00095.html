Detailed Description

Declares the Outputs class for output device management and communication.

Defines the Outputs class, which provides an interface for controlling output devices, handling SPI communication, FreeRTOS task scheduling, and error event binding in an STM32 embedded system. The class supports daisy-chained SPI output devices, manages device initialization and deinitialization, monitors error states, and allows binding of error handler functions for robust output control and diagnostics.
