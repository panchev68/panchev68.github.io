Detailed Description

ModelXbee class for XBee module interaction.

Provides interfaces for signal strength, connection, transmit time, LEDs, module parameters, and spectrum analysis. Encapsulates logic for managing XBee state, supporting UI display, hardware interaction, and event-driven updates in STM32 embedded UI applications.
