Description

template<size_t T_PIN_NUM>
class Lib::BSP::DigitalPeripheral::InputsBase< T_PIN_NUM >

Manages SPI communication, input mapping, debouncing, and change detection for T_PIN_NUM digital inputs. Intended for extension by concrete input port classes.

Template Parameters
T_PIN_NUMNumber of input pins managed.
