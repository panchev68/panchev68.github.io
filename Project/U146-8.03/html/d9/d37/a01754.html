Detailed Description

template<typename T = uint8_t *>
class DataHandle< T >

Lightweight handle for a pointer and its size.

The DataHandle template class wraps a pointer and its size, supporting move, clear, and safe access for embedded and memory-constrained systems. It does not manage ownership or lifetime of the underlying data.

Template Parameters
TPointer type managed by the handle (defaults to uint8_t *).
