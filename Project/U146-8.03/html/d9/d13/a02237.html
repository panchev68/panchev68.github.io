Lib::Helper::Delay

Example usage of Delay classes for precise timing.

Demonstrates how to use the nanosecond and microsecond delay classes for precise timing in embedded applications using the DWT cycle counter.

Example usage:
#include "Delay.hpp"
int main() {
delay_us delayMicro(10); // Delay for 10 microseconds
delay_ns delayNano(100); // Delay for 100 nanoseconds
volatile uint32_t t1 = delayMicro.get();
volatile uint32_t t2 = delayNano.get();
return 0;
}
