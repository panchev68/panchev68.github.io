Detailed Description

template<typename... Args>
class Lib::Helper::Event< Args >

Event class for managing and emitting events with listeners.

Implements a type-safe event system supporting binding, unbinding, and emitting events to multiple listeners. Listeners are stored as std::function objects and can be invoked with arbitrary arguments. Provides efficient memory management through vector storage and supports move semantics for optimal performance in embedded environments.

ArgsTypes of arguments passed to listeners when event is emitted.
