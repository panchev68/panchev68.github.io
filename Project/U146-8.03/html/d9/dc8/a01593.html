Detailed Description

I2C master port DMA mode specialization for asynchronous STM32 operations.

Provides DMA-based I2C master operations using STM32 HAL for event-driven communication. This class enables non-blocking I2C transfers with DMA and supports user-defined callbacks for transfer events.
