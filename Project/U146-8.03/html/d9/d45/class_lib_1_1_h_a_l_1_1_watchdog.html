Detailed Description

STM32 IWDG watchdog interface.

This class encapsulates the configuration and refresh operations for the STM32 independent watchdog (IWDG) peripheral, providing methods to initialize the watchdog with specific prescaler and reload values and to periodically refresh the watchdog counter to prevent system resets. It ensures robust system operation by integrating with the STM32 HAL and supporting safe parameter validation and error handling.
