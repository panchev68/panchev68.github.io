Detailed Description

OutputsListener class for SPI message handling.

The OutputsListener class manages SPI message transmission and reception for digital peripheral outputs. It provides a singleton interface for queue-based communication, encapsulating message structure, transmit and receive queues, and setup logic for FreeRTOS-based embedded systems. The class ensures safe and efficient message passing between hardware drivers and application logic, supporting robust and maintainable firmware design.
