Detailed Description

Specializes the Pin template for input pins, providing state reading and configuration for GPIO input pins.

This class enables robust and type-safe abstraction of GPIO input functionality, supporting flexible initialization, state reading, and configuration of mode, pull, and speed for STM32-based embedded systems.
