Detailed Description

Voltmeter base class for ADC voltage measurement.

This file defines the Base class for ADC-based voltmeter implementations, providing configuration, initialization, periodic update, value integration, and event notification. It is intended for use as a foundation for voltmeter drivers that require ADC sampling, filtering, and voltage conversion.
