Detailed Description

Minimal stdlib replacements for embedded environments.

Provides basic string-to-number and number-to-string conversions for use in resource-constrained systems.

long int std::atol ( const char * input)
inline

Converts a string to a long integer.

Parses a null-terminated string and returns its value as a long integer. Handles optional leading whitespace and sign characters. Stops parsing at the first non-digit character.

inline

Converts an integer to a string in the specified base.

Converts the given integer value to a null-terminated string representation in the specified base (2-36). The result is stored in the provided buffer, which must be large enough to hold the result and the terminating null character. Handles negative values only for base 10.

Parameters
valueInteger value to convert.
bufferBuffer for the resulting string.
sizeLength of the buffer.
baseNumerical base for conversion (2-36).
Returns
Pointer to the resulting string within the buffer.
Precondition
buffer must not be nullptr and size > 0.
Postcondition
buffer contains a null-terminated string representation of value.

inline

Converts a string to a double.

Parses a null-terminated string and returns its value as a double. Handles optional leading whitespace, sign, and fractional part. Parsing stops at the first non-numeric character. If endPosition is not nullptr, it is set to the character after the last parsed character.

Parameters
