Detailed Description

Manages rotary encoder pins and event processing.

This class handles rotary encoder channel and button pins, debouncing, event queueing, direction reversal, and pulse division for robust rotary encoder signal processing. It supports detection of button press and release events, configurable pulse division for noise immunity, and direction reversal for flexible integration. The class uses a FreeRTOS queue for event buffering and provides methods for event polling, debounced button state reading, and interrupt-driven event handling.
