Detailed Description

Utility for hex, BCD, and ASCII conversions.

The HexHelper class provides static methods for converting values to hexadecimal ASCII and BCD formats. It is intended for use in embedded applications where efficient and type-safe conversion between numeric and character representations is required. The class cannot be instantiated.

Converts a 4-bit value (0-15) to its ASCII hexadecimal character (uppercase).

The conversion maps values 0-9 to '0'-'9' and 10-15 to 'A'-'F'.

inlinestaticconstexpr

Converts a 16-bit value to packed BCD (Binary-Coded Decimal).

Each decimal digit of the input value is packed into 4 bits, starting from the least significant digit. For example, 1234 becomes 0x1234 in BCD (0x0001 0x0002 0x0003 0x0004).

Converts an 8-bit value (0-99) to packed BCD.

The upper nibble contains the tens digit, and the lower nibble contains the units digit. For example, 45 becomes 0x45 in BCD (0x40 | 0x05).
