Detailed Description

Defines the OutputVersion enumeration and the Outputs class template for digital output peripherals.

This file provides the OutputVersion enum class to distinguish between different hardware versions of the digital output driver (such as DRV8908 and DRV8908B). It also declares the Outputs class template, which is intended to be specialized for each supported hardware version. The template enables type-safe, version-specific implementation of digital output control logic for STM32-based embedded systems, ensuring compatibility and maintainability across hardware revisions. The design supports compile-time selection of the appropriate driver logic, facilitating efficient and reliable digital output management in resource- constrained embedded environments.
