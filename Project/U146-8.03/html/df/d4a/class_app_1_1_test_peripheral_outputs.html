Detailed Description

TestPeripheralOutputs class for peripheral output test sequencing and control.

This class provides methods to start, stop, and configure test patterns for hardware outputs. It supports querying the current test index, number of available tests, running state, and update period. The implementation details are encapsulated in a private Impl class.
