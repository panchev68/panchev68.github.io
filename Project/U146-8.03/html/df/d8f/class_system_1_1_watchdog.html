Detailed Description

System watchdog timer for fault prevention.

The Watchdog class inherits from Lib::HAL::Watchdog and manages the hardware watchdog timer for the system. It ensures that the watchdog is periodically refreshed to avoid system resets caused by software errors, hangs, or unexpected conditions. This mechanism is critical for maintaining system reliability and detecting faults in embedded applications.
