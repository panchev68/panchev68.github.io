Detailed Description

template<UpdateMode T_MODE>
class Lib::BSP::DigitalPeripheral::Base< T_MODE >

Base class template for digital peripheral control.

Implements a FreeRTOS task loop for managing input/output ports in either sequential or parallel update modes. Provides hooks for pre/post processing and output enable control, and manages tick timing for real-time operation. The class is intended to be subclassed for specific peripheral implementations.

Template Parameters
T_MODEThe update mode (sequential or parallel) for the digital peripheral.
