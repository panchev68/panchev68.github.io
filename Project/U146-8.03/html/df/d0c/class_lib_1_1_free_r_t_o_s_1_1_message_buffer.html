Lib::FreeRTOS::MessageBuffer Class Reference

Description

Type-safe FreeRTOS message buffer for inter-task and ISR communication.

Manages the underlying FreeRTOS message buffer resource and provides methods for sending, receiving, and querying buffer state. Ensures safe creation, manipulation, and destruction of message buffers, supporting both task and ISR contexts. Validates resource usage, prevents leaks, and provides atomic synchronization primitives for inter-task communication.
