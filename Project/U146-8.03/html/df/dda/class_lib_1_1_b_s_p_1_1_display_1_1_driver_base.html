Detailed Description

Abstract base class for display driver operations.

This class defines the interface for initializing the display driver, transmitting display blocks, handling transfer completion, and managing transfer state for derived display driver implementations. It is designed to support singleton usage and facilitate integration with C callback mechanisms in embedded display systems.
