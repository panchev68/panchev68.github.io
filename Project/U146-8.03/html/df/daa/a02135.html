Lib::Xbee::Core::Frame0x01 Class Reference

Query builder for Xbee TX Request (16-bit address) frames. More...

Description

Constructs a TX Request frame with configurable frame ID, destination address, options, and RF data payload. Supports conversion to Frame::Data for transmission through the Xbee API.
