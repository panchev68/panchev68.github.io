Detailed Description

Delay classes for precise time delays in nanoseconds and microseconds.

This file provides base and specialized classes for implementing precise delays using the DWT cycle counter hardware. It supports both nanosecond and microsecond delays, enabling accurate timing operations in embedded systems where precise control over execution timing is required. The design leverages the STM32 DWT peripheral for high-resolution timing and is suitable for real-time and resource-constrained environments.
