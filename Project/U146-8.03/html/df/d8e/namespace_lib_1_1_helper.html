Detailed Description

Integrator class for running average calculation over a fixed-size window.

This template class maintains a buffer of the most recent SIZE values and computes their average. It is suitable for smoothing sensor readings or other streaming data in embedded and application code. The buffer uses a std::vector with reserved capacity for efficiency. The class supports adding new values, retrieving the current average, resetting, and checking if the buffer is full.

Template Parameters
TType of the values to integrate (must support arithmetic operations).
SIZENumber of samples to average over (window size).

This enumeration specifies the available time units for delay operations, allowing selection between nanosecond and microsecond precision. It enables template-based specialization of delay classes for different timing requirements in embedded applications.
