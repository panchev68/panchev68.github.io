Detailed Description

Counting semaphore wrapper for FreeRTOS.

This class encapsulates the creation, management, and operations of a FreeRTOS counting semaphore, allowing control over multiple resource instances with a configurable maximum and initial count, supporting thread and ISR synchronization, and providing methods for querying and manipulating the semaphore state in embedded applications.
