Detailed Description

EEPROM driver for M24C64W device.

Encapsulates logic for communicating with the M24C64W I2C EEPROM, providing initialization, read, write, and device presence detection methods. Designed for STM32 HAL integration, it supports configurable chip and page sizes for flexible hardware adaptation and robust non-volatile memory management in embedded systems.
