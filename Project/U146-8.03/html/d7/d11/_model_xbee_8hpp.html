Detailed Description

Declares the ModelXbee class and related types for XBee module interaction in TouchGFX.

This file defines ModelXbee and its nested types for signal strength, connection, transmit time, LEDs, module parameters, and spectrum analysis in a TouchGFX application. It provides interfaces for querying and updating XBee state, supporting UI display, hardware interaction, and event-driven updates in STM32 embedded UI applications.
