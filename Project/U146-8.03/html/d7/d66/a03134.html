Detailed Description

DMA mode ADC port implementation for STM32 ADC peripherals.

This file provides a DMA-based ADC port implementation for STM32 microcontrollers, utilizing FreeRTOS queues for safe value transfer between ISR and tasks. The class encapsulates DMA start/stop logic, value queuing, and ISR callback handling for ADC peripherals in DMA mode.
