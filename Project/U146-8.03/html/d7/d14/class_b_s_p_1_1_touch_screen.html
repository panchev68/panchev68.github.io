Detailed Description

Touch screen controller integrating XPT2046 driver with TouchGFX.

This class bridges the low-level XPT2046 touch controller driver with the TouchGFX UI framework, providing initialization, touch sampling, and rotation handling for STM32-based embedded systems. It manages hardware pin configuration, calibration, and abstracts touch input for the UI layer.
