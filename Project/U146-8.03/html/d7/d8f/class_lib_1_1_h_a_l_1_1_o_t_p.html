Detailed Description

OTP Flash Memory class.

Provides access to the STM32 OTP (one-time programmable) flash memory region for user data. The OTP area is 0x1FFF7800 - 0x1FFF7A0F (528 bytes), with 16 bytes reserved for block locking.
