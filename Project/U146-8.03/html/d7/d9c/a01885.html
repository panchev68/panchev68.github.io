Detailed Description

MenuScroller for arranging and synchronizing menu items in a scrollable container with a slider.

The MenuScroller class sorts, positions, and manages visibility and scrolling logic for menu buttons in TouchGFX UI. It provides setup, slider synchronization, and access to the arranged children.
