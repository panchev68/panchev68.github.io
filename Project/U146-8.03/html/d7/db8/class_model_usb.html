Detailed Description

ModelUsb class for USB power state and module reset control.

The ModelUsb class enables querying the USB power status and controlling the module reset pin for STM32 embedded applications using TouchGFX. It abstracts hardware-specific details, providing a clean interface for the UI and application logic to interact with USB-related features.
