Detailed Description

UART port type aliases and handle for STM32 HAL.

Provides type aliases for UART port implementations in DMA and interrupt-driven modes, and defines a type alias for the UART handle used in the HAL layer. Enables flexible UART configuration and usage in STM32-based applications.
