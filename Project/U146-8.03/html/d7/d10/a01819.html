Lib::Helper::StringHelper Class Reference

Utility class for ASCII character and string array operations. More...

Description

The StringHelper class provides static methods for checking ASCII printability, digit and hex digit classification, and searching for a string within a null-terminated array of string pointers. All methods are designed for efficiency and type safety in embedded environments.
