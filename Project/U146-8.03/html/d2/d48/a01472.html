Detailed Description

Abstract base class for I2C master operations.

Defines the interface for opening, closing, sending, receiving, and checking readiness of I2C devices, enabling robust and extensible I2C master communication for embedded systems. This class provides a foundation for implementing concrete I2C master drivers, supporting device management and data transfer operations with configurable timeouts and error handling.
