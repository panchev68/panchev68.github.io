Lib::Helper::RingBuffer< T, SIZE > Class Template Reference

Description

template<typename T, size_t SIZE>
class Lib::Helper::RingBuffer< T, SIZE >

Fixed-size circular buffer supporting FIFO push and pop.

Provides constant-capacity, wrap-around indexing for efficient, deterministic memory usage. Suitable for sensor data, communication queues, and real-time streaming where predictable behavior is required.

Template Parameters
TType of elements stored.
SIZEBuffer capacity (must be > 1).
