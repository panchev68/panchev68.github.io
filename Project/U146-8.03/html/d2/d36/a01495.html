Lib::HAL::ADCPort::Port< DriverMode::DMA > Class Reference

DMA mode ADC port implementation. More...

Description

Implements an ADC port using DMA for STM32, with FreeRTOS queue for value transfer. Provides methods for opening/closing the port, starting/stopping DMA conversions, reading ADC values, and handling DMA completion in ISR context.
