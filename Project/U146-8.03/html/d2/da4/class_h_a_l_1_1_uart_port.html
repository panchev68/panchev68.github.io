HAL::UartPort Class Reference

Description

UART port controller for STM32 HAL.

Manages UART peripheral initialization, open/close, send/receive operations, and event handling. Tracks instances for callback routing and supports asynchronous communication. Provides a high-level interface for UART communication, including error handling and event-driven operations.
