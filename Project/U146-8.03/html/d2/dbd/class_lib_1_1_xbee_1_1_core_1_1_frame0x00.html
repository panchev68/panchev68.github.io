Detailed Description

Frame0x00 query builder for Xbee TX Request (64-bit address) frames.

This class constructs a TX Request frame with configurable frame ID, destination address, options, and RF data payload. It supports conversion to Frame::Data for transmission through the Xbee API, ensuring correct frame formatting and data encapsulation for embedded communication.
