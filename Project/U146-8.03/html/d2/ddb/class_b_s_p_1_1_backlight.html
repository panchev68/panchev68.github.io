Detailed Description

Display backlight controller using STM32 timer PWM.

Inherits from Lib::BSP::Backlight and initializes it with the STM32 timer and channel configuration specific to the U146 hardware. Provides methods for setting and retrieving the backlight brightness using PWM, abstracting hardware details for application-level use.
