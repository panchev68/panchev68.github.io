Lib::HAL::UART::Base Class Referenceabstract

Description

Abstract base class for UART operations.

Provides a unified interface for UART initialization, deinitialization, baudrate management, and send/receive operations. Serves as the foundation for UART driver implementations in the hardware abstraction layer.
