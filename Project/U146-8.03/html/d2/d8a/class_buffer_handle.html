Detailed Description

BufferHandle provides non-owning access to a byte buffer for manipulation and memory operations.

This class enables referencing external memory regions as byte buffers, allowing for flexible assignment, copying, and filling operations without managing memory ownership, suitable for embedded and real-time systems.
