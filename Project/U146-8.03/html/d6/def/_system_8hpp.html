Description

System global objects and startup state definitions.

This file declares inline singleton instances for system configuration, identification, and state management. It centralizes access to build information, hardware configuration, startup synchronization, and persistent parameters. All objects are statically allocated for deterministic embedded operation and real-time safety. The System namespace encapsulates global objects for startup state, device identification, and configuration.
