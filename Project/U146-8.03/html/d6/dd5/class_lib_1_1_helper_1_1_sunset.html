Detailed Description

Calculates sunrise, sunset, and moon phase for a given location and date.

Provides methods for official, civil, nautical, and astronomical sun events. The class is optimized for embedded environments, minimizing floating-point operations and memory usage. It supports setting the geographic position, time zone, and date, and offers conversion utilities for time representation.
