Detailed Description

template<typename T>
class Lib::FreeRTOS::Event< T >

Type-safe FreeRTOS single-item event queue.

Provides a strongly-typed, single-slot event signaling queue for safe communication between tasks and ISRs. The class manages the underlying FreeRTOS queue resource and ensures correct usage patterns for embedded and real-time systems.
