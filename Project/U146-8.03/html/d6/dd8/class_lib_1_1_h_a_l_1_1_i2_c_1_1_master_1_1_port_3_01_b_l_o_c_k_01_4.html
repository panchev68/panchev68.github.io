Detailed Description

I2C master port block mode specialization.

This class provides blocking (polling) I2C master operations for STM32 embedded systems, enabling synchronous data transfer using the STM32 HAL. It implements the Port template specialization for the BLOCK driver mode, supporting open, close, send, and receive operations in a blocking context for deterministic communication.
