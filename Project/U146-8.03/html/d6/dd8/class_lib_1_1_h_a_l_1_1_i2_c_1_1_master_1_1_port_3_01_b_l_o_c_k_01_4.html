Lib::HAL::I2C::Master::Port< BLOCK > Class Reference

Description

I2C master port block mode specialization for STM32 synchronous operations.

Implements blocking (polling) I2C master operations using STM32 HAL for deterministic, synchronous communication. Supports open, close, send, and receive operations for I2C master devices.
