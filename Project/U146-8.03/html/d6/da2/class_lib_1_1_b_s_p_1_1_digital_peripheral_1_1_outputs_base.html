Detailed Description

template<size_t T_PIN_NUM>
class Lib::BSP::DigitalPeripheral::OutputsBase< T_PIN_NUM >

Template base for digital output port arrays using SPI and mapping.

The OutputsBase class provides output grouping, SPI communication, and error handling for digital output ports. It manages SPI transfers, output value/status tracking, and supports grouping logic for efficient output switching. This class is intended for embedded systems that require robust digital output control with grouping and debouncing features.
