Detailed Description

template<typename T>
class Lib::Helper::Value< T >

Value wrapper for FreeRTOS queue.

Stores a single value and provides thread-safe access, update, and notification mechanisms using a FreeRTOS queue. Supports overwrite and receive operations from both tasks and ISRs. Designed for deterministic, lock-free value exchange in embedded and real-time systems.

Template Parameters
TType of the value to store.
