Lib::FreeRTOS::StreamBuffer Class Reference

StreamBuffer RAII wrapper for FreeRTOS stream buffer API. More...

Description

Manages creation, deletion, and safe access to a FreeRTOS stream buffer, providing methods for sending and receiving data in both task and ISR contexts. Ensures deterministic resource management and type safety for embedded real-time applications.
