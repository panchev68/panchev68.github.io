Lib::Helper::CityHash Class Reference

Description

Compact CityHash-inspired hash for small buffers in embedded systems.

Efficient, non-cryptographic hash for hash tables and checksums. Designed for speed and minimal resource usage in embedded environments. Not suitable for cryptographic purposes.
