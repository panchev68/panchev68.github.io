Detailed Description

Digi API frame buffer for XBEE communication.

Manages a fixed-size buffer for Digi API frames, supporting assignment from vectors or raw data, appending single values or blocks, and clearing the buffer. Ensures buffer bounds and provides access to the underlying data and size.
