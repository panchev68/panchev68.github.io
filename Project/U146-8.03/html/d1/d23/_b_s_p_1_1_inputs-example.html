BSP::Inputs

Example usage of the Inputs digital input port controller class.

Example usage of the Inputs digital input port controller class.Demonstrates initialization and configuration of the Inputs controller with SPI and GPIO handles, enabling management of multiple digital input ports in an embedded system.

Example usage:
#include "Inputs.hpp"
#include <Lib/HAL/SPI.hpp>
#include <Lib/HAL/GPIO.hpp>
int main() {
Lib::HAL::SPI::Handle spiHandle;
Lib::HAL::GPIO::Handle nssPinHandle;
BSP::Inputs::Config config{ &spiHandle, nssPinHandle };
BSP::Inputs inputs(config);
inputs.init();
// Now inputs controller is initialized and ready for use.
return 0;
}
