Description

Precise delay classes for nanoseconds and microseconds using DWT cycle counter hardware.

This file provides type-safe, allocation-free delay classes for embedded systems, utilizing the DWT (Data Watchpoint and Trace) cycle counter for high-precision timing. It defines a base class for DWT initialization and core delay logic, and two specializations for nanosecond and microsecond delays. The implementation is efficient, does not use dynamic memory, and is suitable for real-time embedded applications.
