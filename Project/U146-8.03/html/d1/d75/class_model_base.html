Detailed Description

Aggregates subsystem model interfaces for the TouchGFX UI layer.

Provides unified access to CAN bus, diagnostic, display, peripheral, system, USB, and Xbee models, and manages periodic tick events and system beeper control for embedded STM32 applications.
