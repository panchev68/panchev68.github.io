Detailed Description

I2C master abstract base class.

Provides the interface for opening, closing, sending, receiving, and checking readiness of I2C devices. Serves as the foundation for implementing concrete I2C master drivers with configurable timeouts and error handling. All derived classes must implement the pure virtual methods for device communication.
