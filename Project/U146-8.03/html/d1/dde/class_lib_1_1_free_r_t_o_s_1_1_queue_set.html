Detailed Description

Encapsulates FreeRTOS queue set creation, member management, and selection.

Provides a C++ interface for creating queue sets, adding/removing members, and selecting ready members.

Warning
configUSE_QUEUE_SETS must be set to 1 in FreeRTOSConfig.h.
