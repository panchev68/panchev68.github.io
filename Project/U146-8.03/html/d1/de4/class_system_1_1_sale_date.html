System::SaleDate Class Reference

Description

SaleDate type for product sale date representation.

The SaleDate class inherits from ProduceDateBase and provides a fixed-width storage mechanism for product sale dates. It is used to store and manage the sale date of products in the system, ensuring that the date information is consistently formatted and stored at a specific flash address.
