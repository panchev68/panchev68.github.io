Detailed Description

DRV8908 motor driver interface class.

Defines the DRV8908 class, providing a high-level abstraction for the DRV8908 motor driver chip. Enables register access, error handling, and output control for DRV8908 devices in embedded systems. Supports SPI-based communication, manages output state changes, and processes chip status for error reporting. Designed for integration with OutputsListener for message-based SPI transactions and supports error notification via callback mechanisms. This file provides detailed documentation and type safety for robust motor control applications.
