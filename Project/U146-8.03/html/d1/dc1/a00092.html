Detailed Description

DRV8908 motor driver interface class.

This file defines the DRV8908 class, which provides a high-level abstraction for the DRV8908 motor driver chip. It enables register access, error handling, and output control for DRV8908 devices in embedded systems. The class supports SPI-based communication, manages output state changes, and processes chip status for error reporting. It is designed for integration with OutputsListener for message-based SPI transactions and supports error notification via callback mechanisms.
