Detailed Description

Rotary encoder and button driver using STM32 timer in encoder mode.

Handles position tracking, button interrupts, and event generation for rotary encoders with pushbutton.

enum class Lib::BSP::EncoderTimer::EventCode

Event codes for encoder and button actions.

Enumerates all possible events detected by the encoder and button, including rotation and button states.

Enumerator
BUTTON_PUSH  

Button pressed event.

BUTTON_RELEASE  

Button released event.

BUTTON_HOLD  

Button held for threshold duration.

ENCODER_CW  

Encoder rotated clockwise.

ENCODER_CCW  

Encoder rotated counter-clockwise.

NONE -1 

No event detected.
