Detailed Description

ModelDisplay class for TouchGFX UI display parameter management.

The ModelDisplay class provides mechanisms to query and update display parameters, including rotation and backlight level, for the TouchGFX UI. It ensures synchronization between the UI and the hardware abstraction layer, supports persistent configuration, and validates the presence of parameter storage for robust STM32 embedded applications.
