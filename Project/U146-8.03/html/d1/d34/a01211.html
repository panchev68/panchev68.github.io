Lib::BSP::EncoderTimer::Config Struct Reference

Configuration for EncoderTimer. More...

Description

The Config structure holds the hardware resources required by the EncoderTimer, including the STM32 timer handle configured for encoder mode and the pin handle for the encoder's pushbutton. This structure is passed to the EncoderTimer constructor to initialize the driver.
