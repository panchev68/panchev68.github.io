Lib::Helper::Event< T_ARGS > Class Template Reference

Type-safe event system supporting multiple listeners. More...

Description

template<typename... T_ARGS>
class Lib::Helper::Event< T_ARGS >

This class enables binding, unbinding, and emitting events to multiple listeners using std::function. Listeners can be added or removed at runtime, and events can be triggered with arbitrary arguments. The implementation is optimized for embedded and desktop environments, providing predictable memory usage and efficient listener management.
