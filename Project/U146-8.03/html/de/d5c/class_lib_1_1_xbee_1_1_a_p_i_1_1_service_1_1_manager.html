Detailed Description

Singleton manager for Xbee API services.

The Manager class is responsible for maintaining a collection of Xbee API services derived from ServiceBase. It provides methods to add, remove, and update services, ensuring centralized management through a singleton instance. The class inherits from Lib::Helper::Container<ServiceBase> to leverage container functionality for service storage and manipulation.
