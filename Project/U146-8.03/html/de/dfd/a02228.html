Lib::FreeRTOS::MessageBuffer

Example usage of MessageBuffer for inter-task communication.

Demonstrates creation, sending, and receiving of messages using the MessageBuffer class.

Example usage:

#include "MessageBuffer.hpp"
void exampleUsage() {
Lib::FreeRTOS::MessageBuffer buffer;
const size_t bufferSize = 128;
if (buffer.create(bufferSize)) {
const char* message = "Hello, FreeRTOS!";
size_t sent = buffer.send(message, 17, 10);
char recv[32] = {};
size_t received = buffer.receive(recv, sizeof(recv), 10);
// Use recv and received as needed
buffer.release();
}
}
