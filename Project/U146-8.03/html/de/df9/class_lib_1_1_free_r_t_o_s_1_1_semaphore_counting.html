Lib::FreeRTOS::SemaphoreCounting Class Reference

Description

Counting semaphore for FreeRTOS.

Extends SemaphoreBase to support counting semaphores. This class provides a wrapper for FreeRTOS counting semaphore APIs, enabling creation with a maximum and initial count. It supports standard semaphore operations such as take, give, and querying the current count, as well as ISR-safe variants. The class disables the default create method to enforce explicit configuration of maximum and initial counts.

Creates a counting semaphore.

Allocates and initializes a FreeRTOS counting semaphore with the specified maximum and initial count. The semaphore is not created if one already exists.

Parameters
maxCountMaximum count value for the semaphore; determines the upper bound of the semaphore.
initCountInitial count value for the semaphore; sets the starting count.
Returns
true if creation was successful, false otherwise.
Warning
Asserts if called when a semaphore already exists or if creation fails.
