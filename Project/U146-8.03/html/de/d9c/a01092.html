Detailed Description

Configuration for SPI and GPIO peripherals.

This class encapsulates the configuration required for digital peripheral ports, including the SPI handle, NSS pin, and mapping fields. It is intended to be used for initializing and managing digital output ports that rely on SPI communication and GPIO control, ensuring that all necessary hardware references are provided in a single structure.
