Lib::BSP::EEPROM::I2C::DriverBase Class Referenceabstract

DriverBase base class for I2C EEPROM drivers. More...

Description

Encapsulates common logic for I2C EEPROM access, including configuration, open/close, and page-aware read/write helpers. Derived classes must implement read/write methods.
