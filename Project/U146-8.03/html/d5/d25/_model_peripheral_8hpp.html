Detailed Description

Declares the ModelPeripheral class and related types for TouchGFX model layer peripheral management.

This file defines the ModelPeripheral class and its nested types, enabling monitoring and control of peripheral node states, test output sequences, and event-driven updates. It provides interfaces for handling input/output state changes, node activity, and test output management, encapsulating implementation details using the PIMPL idiom for maintainability and modularity in STM32 embedded UI applications.
