Detailed Description

RAII wrapper for delaying the current FreeRTOS task.

The Delay class provides a simple and type-safe way to delay the execution of the current FreeRTOS task for a specified duration in milliseconds. It is intended for use in embedded systems where deterministic task timing is required. The class encapsulates the vTaskDelay function and ensures that the delay duration is always specified in milliseconds.
