Detailed Description

FreeRTOS stream buffer wrapper for safe creation, transfer, and management.

This class encapsulates FreeRTOS stream buffer operations, providing safe initialization, cleanup, and access to the underlying buffer, supporting both task and ISR context operations for robust data transfer in embedded systems.
