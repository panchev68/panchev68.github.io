Detailed Description

Touch screen driver for XPT2046 controller.

Provides touch detection, position calculation, debouncing, and rotation handling for XPT2046-based touch screens. Integrates with FreeRTOS for asynchronous touch processing.

Implements touch detection, position calculation, debouncing, and rotation handling for XPT2046-based touch screens. Integrates with FreeRTOS for asynchronous touch processing.
