Detailed Description

template<class T_DEVICE, typename T_HANDLE>
class Lib::HAL::DeviceManagerBase< T_DEVICE, T_HANDLE >

Device manager base template for embedded device abstraction.

Manages a collection of device objects and provides methods for adding, removing, and retrieving devices by handle. Enables generic device and handle types for flexible hardware abstraction.

Template Parameters
T_DEVICEDevice class type.
T_HANDLEDevice handle type.
