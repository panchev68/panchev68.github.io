Description

Type-safe wrapper for FreeRTOS event groups.

This file provides a type-safe C++17 class encapsulating a FreeRTOS event group for bitwise event signaling and synchronization between tasks and ISRs. It ensures safe creation, manipulation, and destruction of event groups, supporting both task and ISR contexts, and provides atomic synchronization primitives for inter-task communication.
