Lib::HAL::GPIO::Pin< Type::OUTPUT > Class Reference

GPIO output pin specialization. More...

Description

Specializes the Pin template for output pins, providing methods for writing and toggling the output state. Inherits initialization and handle management from the input pin specialization, ensuring consistent interface and type safety for STM32 GPIO output operations.
