BSP::DigitalPeripheral

Example usage of DigitalPeripheral for digital IO management.

This example demonstrates how to initialize the DigitalPeripheral, check for errors, and access input/output interfaces.

Example usage:
#include "DigitalPeripheral.hpp"
int main() {
BSP::DigitalPeripheral digitalPeripheral;
if (digitalPeripheral.init()) {
// Initialization successful
if (!digitalPeripheral.getErrorOutput()) {
// No error present, proceed with IO operations
}
} else {
// Initialization failed, handle error
}
return 0;
}
