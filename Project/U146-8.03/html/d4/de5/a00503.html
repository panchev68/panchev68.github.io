Detailed Description

DateTime, Date, and Time classes for date and time management in embedded systems.

This file defines the DateTime class and its nested Date and Time classes, providing comprehensive functionality for representing, manipulating, and formatting date and time values. The classes support construction, comparison, validation, string conversion, and packing/unpacking for efficient storage. Designed for embedded and performance-critical systems, the implementation ensures type safety, minimal overhead, and compatibility with standard C++ and C time structures. The Date class includes day-of-week and leap year calculations, while the Time class provides validation and formatting utilities. The overall design enables robust and flexible date/time handling without dynamic memory allocation.
