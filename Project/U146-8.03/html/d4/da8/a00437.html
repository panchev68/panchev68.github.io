Detailed Description

I2C memory address size enumeration for STM32 embedded systems.

This file defines the MemAddrSize strongly typed enumeration, which specifies the memory address size (8-bit or 16-bit) for I2C master memory operations. It enables type-safe selection of address size when performing I2C memory read or write transactions, supporting robust and flexible communication with external memory devices in embedded applications.
