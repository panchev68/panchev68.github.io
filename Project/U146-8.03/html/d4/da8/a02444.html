Detailed Description

Beep class for U146 device beep output control.

Declares the BSP::Beep class, which manages beep output using a hardware timer and provides methods to trigger beep sequences with configurable duration, pause, and count. Integrates with the Lib::BSP::Beep base class for hardware abstraction and consistent beep control across different platforms.
