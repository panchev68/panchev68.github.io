Detailed Description

template<typename T, size_t T_SIZE = (sizeof(T) * 8U)>
class Lib::Helper::BitArray< T, T_SIZE >

Fixed-size array of boolean values representing the bits of an integral type.

This class template provides bitwise access and manipulation for a given integral type, allowing conversion between boolean arrays and their corresponding integral values. It supports efficient bit-level operations for embedded and performance-critical applications, ensuring type safety and compile-time sizing.

Template Parameters
TIntegral type to represent as a bit array.
T_SIZENumber of bits in the array, defaulting to the bit width of T.
