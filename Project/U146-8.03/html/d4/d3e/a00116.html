Detailed Description

STM32 BSP LED control class.

Manages ready and error LEDs, supporting ON, OFF, and BLINKING states with hardware output and periodic updates.
Enables initialization, state changes, and tick-based blinking for LED indicators in STM32 embedded systems.
The Leds class provides a concrete implementation for controlling LED indicators on STM32 boards, including
ready and error LEDs. It supports ON, OFF, and BLINKING states, hardware output management, and periodic tick
updates for deterministic blinking behavior. The class is designed for integration with the BSP layer and can
be extended for custom LED patterns or additional indicators if required.
