Lib::Helper::ValueInfo

Example usage of ValueInfo.

Demonstrates how to use the ValueInfo class for type-safe variable pointer storage and runtime type checking.

Example usage:

#include "ValueInfo.hpp"
#include <iostream>
int main() {
int value = 42;
Lib::Helper::ValueInfo info(&value, typeid(value));
if (info.getTypeInfo() == typeid(int)) {
int * ptr = static_cast<int *>(info.getData());
std::cout << "Value: " << *ptr << std::endl;
}
return 0;
}
