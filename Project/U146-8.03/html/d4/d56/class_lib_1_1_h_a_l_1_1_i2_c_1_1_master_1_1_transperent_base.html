Detailed Description

Transparent I2C master communication base class.

Implements blocking and non-blocking I2C transfer logic, event emission, and device registration for STM32 embedded systems. Derived classes must provide hardware-specific transfer and event emission implementations. The class manages FreeRTOS mutexes and semaphores for thread safety and integrates with a device manager for callback registration and resource tracking.

Note
Derived classes must implement event emission and transfer functions.
