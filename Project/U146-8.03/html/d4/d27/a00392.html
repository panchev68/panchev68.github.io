Detailed Description

Backup RAM Domain management.

Provides a singleton class for managing the Backup RAM Domain, including enabling, disabling, and resetting the backup domain. This file ensures safe access and control of backup SRAM and related RTC registers, facilitating robust management of persistent memory and RTC functionality in STM32-based embedded systems.
