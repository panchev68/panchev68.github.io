Detailed Description

FreeRTOS queue C++ wrapper.

Encapsulates FreeRTOS queue creation, management, and operations in a type-safe, resource-managed C++ interface. Supports dynamic and static allocation, provides methods for sending, receiving, peeking, overwriting, and registry management. Ensures safe usage in embedded systems with input validation and invariant enforcement. Suitable for deterministic, real-time inter-task communication.
