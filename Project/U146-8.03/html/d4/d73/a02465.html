Detailed Description

MCU last reset state detector.

This file defines the LastResetState class, which provides an interface to determine and store the cause of the last MCU reset using RCC reset flags. The class enables applications to query the reset reason after startup and take actions based on the detected cause, supporting robust system diagnostics and fault handling in STM32 embedded environments.
