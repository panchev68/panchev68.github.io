Lib::HAL::SPI::Master::Port< IT > Class Reference

SPI Master IT port specialization. More...

Description

Specialization of the Port class for interrupt-driven (IT) SPI master operations using STM32 HAL. Provides non-blocking transmit, receive, and transceive operations with event callbacks for SPI transactions. Supports initialization, deinitialization, abort, and completion event handling for SPI master in IT mode.
