Detailed Description

template<typename T = uint8_t *>
class DataHandle< T >

DataHandle template class for non-owning data pointer and size management.

This class provides a simple, type-safe wrapper for a pointer to a data block and its size, supporting construction, move semantics, and utility methods for pointer and size access. It is designed for efficient use in embedded and performance-critical systems where dynamic memory allocation is undesirable, enabling safe and flexible handling of external data buffers without ownership or lifetime management.

Template Parameters
TPointer type for the data block (default: uint8_t *).
