Lib::Xbee::Core::Frame0x00 Class Reference

Frame0x00 TX Request frame (64-bit address). More...

Description

Represents a TX Request frame for XBee modules using a 64-bit destination address. Provides members for setting the frame ID, destination address, transmission options, and RF data payload.
