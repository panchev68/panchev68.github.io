Transparent I2C master base class for device communication and event handling. More...

Description

This file defines the TransperentBase class, which serves as a foundation for I2C master device drivers. It handles synchronization, event emission, callback registration, and provides abstract methods for derived classes to implement device-specific logic. The class integrates with FreeRTOS for thread safety and uses a DeviceManager for centralized device management and callback dispatching.
