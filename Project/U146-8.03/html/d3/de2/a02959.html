Detailed Description

template<size_t BUFFER_SIZE>
class StaticBuffer< BUFFER_SIZE >

StaticBuffer provides a fixed-size buffer using stack allocation.

This template class is intended for use cases where a fixed-size buffer is required on the stack, avoiding dynamic memory allocation. It inherits from Buffer for interface compatibility.

Template Parameters
BUFFER_SIZESize of the static buffer.
