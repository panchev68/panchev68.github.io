enum class Lib::HAL::I2C::Master::MemAddrSize : uint32_t
strong

Memory address size enumeration for I2C master memory operations.

Specifies the memory address size (8-bit or 16-bit) for I2C memory access, ensuring type-safe configuration during memory read and write operations. This enumeration is used to select the address size for I2C memory transactions with external devices such as EEPROMs, sensors, or other peripherals that require memory addressing. Using a strongly typed enum class prevents accidental misuse and improves code clarity and safety.

Note
Used to select address size for I2C memory read/write operations.
