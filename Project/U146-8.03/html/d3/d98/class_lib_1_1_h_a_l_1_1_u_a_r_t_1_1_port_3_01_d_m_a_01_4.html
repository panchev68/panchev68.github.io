Lib::HAL::UART::Port< DMA > Class Reference

Description

UART port implementation using DMA.

Implements UART communication using DMA for STM32 HAL. Inherits from TransperentBase and overrides send/receive operations to use DMA-based APIs. All operations are performed via HAL DMA functions.
