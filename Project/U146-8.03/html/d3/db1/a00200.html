Detailed Description

ModelPeripheral abstraction for TouchGFX model layer.

This file defines the ModelPeripheral class, which provides access to peripheral input/output states, node activity, and test output client management for the TouchGFX model. It encapsulates hardware state and test output logic, enabling UI synchronization and event handling. All objects are statically allocated for deterministic embedded operation and real-time safety.
