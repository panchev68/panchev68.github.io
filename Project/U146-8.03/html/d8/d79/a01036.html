Detailed Description

Implements a buzzer driver using STM32 timer output compare.

The Beep class provides an interface to control a buzzer by configuring and toggling a timer channel, supporting multiple beeps with customizable duration and pause intervals. It manages the hardware timer and state transitions to ensure correct beep sequences and integrates with FreeRTOS for timing operations.
