Detailed Description

Provides access to the last system reset reason.

Inherits from Lib::HAL::LastResetState, exposing the reset state functionality in the System namespace for higher-level code. This class allows querying the last reset cause using the State enumeration, facilitating diagnostics and system monitoring.
