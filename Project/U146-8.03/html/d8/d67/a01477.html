Detailed Description

Abstract base class for ADC port management.

Provides a unified interface for ADC port operations, including opening/closing the peripheral, starting/stopping conversions, and reading ADC values. Designed for STM32-based systems to ensure consistent ADC hardware access and control across implementations.
