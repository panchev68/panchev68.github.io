Lib::FreeRTOS::MemoryManager Class Reference

FreeRTOS heap memory manager with diagnostics. More...

Description

Encapsulates static methods for allocation, deallocation, and heap usage tracking. Maintains diagnostic counters for allocations, deallocations, and active allocations. All operations validate inputs and update diagnostics for runtime monitoring.
