Detailed Description

template<class T>
class Lib::Helper::JaggedArray< T >

Compact jagged array container for non-owning pointers.

Stores non-owning pointers to objects, prevents duplicates, and provides fast add/remove.

Template Parameters
TType of the elements stored in the jagged array.
