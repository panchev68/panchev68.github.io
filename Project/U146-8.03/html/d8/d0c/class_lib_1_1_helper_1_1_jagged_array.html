Lib::Helper::JaggedArray< T > Class Template Reference

Description

template<class T>
class Lib::Helper::JaggedArray< T >

Compact jagged array container for non-owning pointers.

Provides efficient storage and management of non-owning pointers to objects, preventing duplicates and supporting fast add/remove operations. Designed for embedded and performance-critical applications where memory usage and ownership are externally managed.

Template Parameters
TType of the elements stored in the jagged array.
