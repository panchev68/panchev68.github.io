Detailed Description

CAN transmit header wrapper providing configuration capabilities.

Inherits from CAN_TxHeaderTypeDef and extends functionality with constructors and methods for setting up CAN transmit parameters, including identifier, data length, identifier format, remote transmission request, and timestamp options, supporting robust CAN communication in embedded systems.
