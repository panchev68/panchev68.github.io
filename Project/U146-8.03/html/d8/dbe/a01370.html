Detailed Description

CAN bus service base class.

Abstract base class for CAN bus service implementations, providing core communication, diagnostics, and FreeRTOS task management. Inherits from HAL::CAN::Port and encapsulates message transmission, reception, error handling, and diagnostics for STM32-based systems.
