Detailed Description

Helper utilities for embedded and real-time systems.

Byte swap utilities for fixed-width integer types.

This namespace contains utility classes and functions for efficient data handling and memory management in embedded and real-time applications.

Provides functions for endianness conversion of 16/32/64-bit integers and raw buffers.
