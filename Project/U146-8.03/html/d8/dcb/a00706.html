Detailed Description

Board Support Package namespace containing singleton hardware abstraction layer instances.

The BSP namespace exposes globally accessible inline objects for interacting with hardware components, abstracting device-specific details and providing a unified interface for application and middleware code. All hardware access should go through these singleton instances to ensure proper resource management.

◆ OutputVersion

enum class BSP::OutputVersion
strong

Enumerates supported digital output hardware versions for the Outputs class template.

This strongly typed enumeration allows compile-time selection of the appropriate driver logic for different hardware revisions, such as DRV8908 and DRV8908B, ensuring type safety and maintainability in embedded digital output control implementations.
