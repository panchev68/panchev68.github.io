Lib::BSP::Ethercat::PDI::LAN9252::Interface Class Reference

Description

LAN9252 SPI Interface.

Provides thread-safe, mutex-protected SPI communication and chip select control for the LAN9252 EtherCAT slave device using FreeRTOS and DMA SPI driver.
