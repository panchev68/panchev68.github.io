Detailed Description

Provides logic for managing system information and state in the TouchGFX model layer.

The ModelSystem class enables access to system version, reset state, production and sale dates, and startup mode. It encapsulates mechanisms for querying and updating system state, supporting UI display, persistent configuration, and event-driven updates for STM32 embedded UI applications.
