Lib::Helper::Integrator< T, SIZE > Class Template Reference

Description

template<typename T, size_t SIZE>
class Lib::Helper::Integrator< T, SIZE >

Moving average integrator for fixed-size window.

Maintains a circular buffer of the most recent SIZE values and computes their average efficiently. Suitable for smoothing sensor readings or streaming data in embedded systems.

Template Parameters
TValue type supporting arithmetic operations.
SIZEWindow size (number of samples).
