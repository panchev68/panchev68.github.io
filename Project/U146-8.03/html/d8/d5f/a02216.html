BSP::Beep

Example usage of BSP::Beep

Demonstrates how to use the BSP::Beep class to initiate a beep sequence with custom parameters.

Example usage:
#include "BSP/Beep.hpp"
int main() {
BSP::Beep beep;
beep(200, 100, 3); // 3 beeps, each 200ms, with 100ms pause between
return 0;
}
