Detailed Description

Enumeration for STM32 TIM peripheral operating modes.

Defines the supported operating modes for STM32 timer peripherals in the HAL layer. Used to specify the functional configuration of TIM devices, such as encoder mode, PWM mode, or other supported timer modes. This enumeration enables type-safe selection of timer operating modes for device initialization and configuration in embedded applications.
