Detailed Description

CAN bus server service base class.

This file defines the base class for CAN bus server services, providing node scanning, message sending, and event handling. The ServerBase class implements core CAN bus server functionality, including node scanning, message transmission, diagnostics retrieval, and event callbacks. It is designed for extension by concrete CAN bus server implementations and provides a foundation for robust, scalable CAN communication in embedded systems. The class supports scanning for active nodes, sending messages with various payloads, and handling received messages and diagnostics. Derived classes must implement the post-scan and task functions for application-specific logic.
