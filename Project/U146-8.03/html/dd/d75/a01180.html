Detailed Description

LAN9252 EtherCAT PDI base driver.

Implements SPI-based communication, register access, chip reset, and burst mode operations for the LAN9252 EtherCAT peripheral device interface. Provides initialization, state management, and low-level register read/write logic for robust EtherCAT device operation on STM32 platforms.
