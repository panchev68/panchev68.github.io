Detailed Description

Utility class for managing FreeRTOS software timers using modern C++ idioms and callbacks.

This class enables creation, control, and manipulation of FreeRTOS software timers, supporting both task and interrupt contexts, and providing a flexible interface for embedded real-time applications.
