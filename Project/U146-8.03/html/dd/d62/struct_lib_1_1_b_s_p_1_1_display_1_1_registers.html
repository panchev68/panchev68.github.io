Detailed Description

ILI9341 register addresses.

This struct provides register address constants for the ILI9341 display controller, allowing software to configure, control, and query the display hardware. The constants are used for sending commands and reading status or data from the controller, supporting initialization, power management, display modes, and diagnostics.

Note
These constants are used to communicate with the ILI9341 controller via SPI or parallel interface.
