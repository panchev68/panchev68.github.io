Detailed Description

ILI9341 register addresses.

Provides register address constants for the ILI9341 display controller. Use these constants to send commands and read status or data from the controller for initialization, power management, display modes, and diagnostics.

Note
Communicate with the ILI9341 controller via SPI or parallel interface using these constants.
