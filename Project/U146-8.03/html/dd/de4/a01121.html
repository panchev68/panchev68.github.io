Detailed Description

template<size_t T_PIN_NUM>
class Lib::BSP::DigitalPeripheral::OutputsBase< T_PIN_NUM >

Generic base for SPI digital output port abstraction.

Manages SPI communication, output mapping, grouping, error detection, and change events for T_PIN_NUM digital outputs. Intended for extension by concrete output port classes.

Template Parameters
T_PIN_NUMNumber of output pins managed.
