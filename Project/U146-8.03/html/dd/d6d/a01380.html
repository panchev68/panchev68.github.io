Detailed Description

Abstract base class for FreeRTOS semaphore wrappers.

This class provides resource management and a unified interface for derived semaphore types, encapsulating creation, deletion, and basic operations for FreeRTOS semaphores, ensuring safe and consistent usage across different semaphore implementations in embedded systems.
