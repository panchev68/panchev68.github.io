Detailed Description

Device driver base class.

Provides a common interface for BSP device driver initialization and periodic tick processing. All BSP drivers must inherit from this class and implement the tick method to handle periodic processing required by the hardware or device.

Performs periodic processing for the device driver.

This pure virtual method must be implemented by all derived classes. It is typically called from a system tick or scheduler to handle time-dependent operations such as polling hardware, updating state, or servicing timeouts.
