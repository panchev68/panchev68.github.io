Detailed Description

Service for managing DIGI XBEE module parameters.

This class provides functionality to read and write static and dynamic parameters of the DIGI XBEE module, supporting both local and remote parameter management. It handles parameter updates, state transitions, and integrates with the Xbee AT command interface for robust embedded wireless configuration.
