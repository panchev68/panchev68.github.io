Detailed Description

template<size_t T_SIZE>
class BufferStatic< T_SIZE >

Fixed-size stack-allocated byte buffer.

This template class provides a buffer of compile-time fixed size, allocated on the stack, for efficient and deterministic memory usage in embedded and real-time applications.

Template Parameters
T_SIZESize of the buffer in bytes.
