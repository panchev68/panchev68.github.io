Detailed Description

ModelSystem class for system state and metadata access.

The ModelSystem class provides access to system startup mode, reset state, build/hardware version, serial number, and production dates. It encapsulates logic for retrieving and updating system information, including color coding and string conversion, and is used for UI and diagnostics in embedded STM32 projects.
