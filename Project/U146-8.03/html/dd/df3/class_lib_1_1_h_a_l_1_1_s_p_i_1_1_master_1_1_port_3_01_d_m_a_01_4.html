Detailed Description

SPI Master DMA port specialization.

This class implements SPI master communication using DMA for STM32 HAL. It provides methods for opening, closing, initializing, deinitializing, sending, receiving, and handling transfer completion events. The class uses function objects for event callbacks and overrides low-level transfer functions to use DMA-based HAL SPI routines. All operations are performed on the SPI handle provided at construction.
