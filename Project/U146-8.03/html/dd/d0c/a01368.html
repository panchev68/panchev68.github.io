Detailed Description

Encapsulates FreeRTOS mutex creation, management, and operations.

This class provides a C++ interface for creating, acquiring, releasing, and managing FreeRTOS mutexes, ensuring robust resource management and type safety for embedded applications requiring mutual exclusion.
