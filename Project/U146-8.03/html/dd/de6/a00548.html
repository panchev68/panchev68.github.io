Detailed Description

Base class for button menu widgets in TouchGFX.

Provides common logic for managing a scrollable menu of buttons with selection and navigation. Designed for embedded systems, it minimizes memory usage and ensures efficient button state management, supporting smooth navigation and selection in resource-constrained environments. The class leverages TouchGFX containers and button styles, and integrates with FreeRTOS for timing control, providing a flexible foundation for custom menu widgets.
