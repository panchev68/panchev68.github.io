Detailed Description

I2C Master Interrupt Mode class.

This file defines the Port<IT> class, implementing the I2C master port for interrupt-driven (IT) mode operations in STM32 embedded systems. The class supports asynchronous data transfers, event callback registration, and integrates with the transparent I2C communication base. It provides methods for sending and receiving data using interrupts, both for direct device access and memory-mapped devices, and manages event emission for transfer completion, errors, and aborts.
