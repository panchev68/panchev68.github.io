Detailed Description

Forward declaration for OutputsListener class.

Abstract base class for output device management. Provides a polymorphic interface for output device initialization, deinitialization, error state checking, and error handler binding. Intended for use as a base class for concrete output device implementations in embedded systems, supporting flexible hardware abstraction and error notification mechanisms.
