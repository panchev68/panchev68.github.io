Detailed Description

Spectrum Analyzer Service for Xbee API.

This class implements a spectrum analyzer for DIGI XBEE modules, periodically sending scan requests, processing energy data responses, and maintaining arrays for current, average, and maximum channel energy levels. It is designed for embedded systems requiring real-time spectrum monitoring and channel quality assessment.
