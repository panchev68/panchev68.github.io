Detailed Description

Transparent UART base class.

Implements UART operations with FreeRTOS message buffer and semaphore for RX/TX, and manages device registration. Provides synchronized send/receive operations, device registration, and HAL callback dispatching.

Note
Intended for STM32 HAL UART peripherals with FreeRTOS integration.
