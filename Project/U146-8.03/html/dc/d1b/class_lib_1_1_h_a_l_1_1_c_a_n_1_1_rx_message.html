Detailed Description

CAN receive message wrapper class inheriting from CAN_RxHeaderTypeDef.

The RxMessage class encapsulates the CAN message header and payload, providing methods for accessing the message identifier, data length, and payload data. This design enables efficient and type-safe handling of received CAN messages in STM32 embedded systems, supporting robust CAN communication and simplifying message processing in real-time applications.
