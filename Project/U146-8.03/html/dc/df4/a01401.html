Detailed Description

Type-safe FreeRTOS event group for bitwise signaling between tasks and ISRs.

Manages the underlying FreeRTOS event group resource and provides methods for event bit manipulation, synchronization, and resource management. Ensures safe creation, manipulation, and destruction of event groups, supporting both task and ISR contexts. Validates resource usage, prevents leaks, and provides atomic synchronization primitives for inter-task communication.
