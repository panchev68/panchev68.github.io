CAN bus operation mode enumeration.

Represents the available operation modes for CAN bus communication, distinguishing between client and server roles. This enumeration is used to configure CAN bus services and select the appropriate behavior for client or server implementations. It enables clear specification of the intended communication role within the CAN bus subsystem, supporting type safety and explicit mode selection in service templates and runtime logic.
