Detailed Description

Detects hardware revision and converts it to a string.

Reads hardware revision pins, encodes the detected version, and provides a string representation for use in system diagnostics or display.

std::string System::HardwareVersion::getString ( ) const

Returns the hardware version as a string based on the revision pins.

Converts the detected hardware version ID to a human-readable string such as "4.03".

Returns
String representing the hardware version
Note
Returns "4.03", "4.04", or "4.05" for known versions, or "UNKNOWN" if not recognized
Precondition
Hardware revision pins must be connected and readable

Reads and encodes the hardware version ID from GPIO pins.

Samples and inverts four hardware revision pins, then encodes them as a 4-bit value.

Returns
Encoded hardware version ID
Note
Pin logic is inverted; high means not asserted
Precondition
