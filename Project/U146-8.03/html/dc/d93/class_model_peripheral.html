Detailed Description

Provides logic for managing peripheral node states and test outputs in the TouchGFX model layer.

The ModelPeripheral class enables monitoring and control of peripheral nodes, including input/output state changes, node activity, and test output sequences. It supports event-driven updates via listener interfaces and encapsulates implementation details using the PIMPL idiom for maintainability and modularity.
