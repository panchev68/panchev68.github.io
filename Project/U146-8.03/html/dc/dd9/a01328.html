Detailed Description

Base class for CAN bus service abstraction.

This class provides core functionality for CAN bus communication, including message transmission, reception, diagnostics, and task management. It is designed for embedded systems using STM32 HAL and FreeRTOS, allowing derived classes to implement custom CAN service logic while handling low-level operations and error tracking.
