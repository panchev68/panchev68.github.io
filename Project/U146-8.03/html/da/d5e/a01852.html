Detailed Description

Service for discovering devices in the Xbee network.

The NetworkDiscovery class implements device discovery for Digi Xbee wireless networks, parsing node information and managing the results. It interacts with the Xbee AT command interface to enumerate all joined devices and provides access to discovered node data for further processing or display.
