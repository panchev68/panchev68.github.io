Detailed Description

Encapsulates FreeRTOS co-routine creation and provides static methods for scheduling and queue operations from ISR context.

This class enables creation of co-routines, scheduling, and queue operations in a type-safe and assertive manner, ensuring correct usage and error detection at runtime.
