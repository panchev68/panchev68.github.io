Detailed Description

OTP Flash Memory class.

Provides access to 512 OTP (one-time programmable) bytes for user data. The OTP area is 0x1FFF7800 - 0x1FFF7A0F (528 bytes), with 16 additional bytes used to lock the corresponding OTP data block. This class offers type-safe read and write operations, address validation, and singleton access to ensure safe usage in embedded applications.
