Detailed Description

RxMessage CAN receive message wrapper class.

The RxMessage class encapsulates a CAN message header (CAN_RxHeaderTypeDef) and its payload data array. It provides efficient, type-safe access to the CAN message identifier, data length, and payload for robust, real-time embedded CAN communication. The class is designed for use with STM32 HAL drivers and supports up to 8 bytes of payload data per CAN frame.
