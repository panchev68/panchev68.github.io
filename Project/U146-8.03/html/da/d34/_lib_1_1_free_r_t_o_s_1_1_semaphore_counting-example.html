Lib::FreeRTOS::SemaphoreCounting

Example usage of SemaphoreCounting in FreeRTOS.

Demonstrates creation, usage, and destruction of a counting semaphore using the SemaphoreCounting class.

Example usage:

#include "SemaphoreCounting.hpp"
void exampleSemaphoreCounting() {
Lib::FreeRTOS::SemaphoreCounting semaphore;
constexpr uint32_t MAX_COUNT = 5;
constexpr uint32_t INIT_COUNT = 2;
if (semaphore.create(MAX_COUNT, INIT_COUNT)) {
// Take the semaphore (decrement count)
if (semaphore.take(10)) {
// Critical section
semaphore.give(); // Release the semaphore (increment count)
}
// Query current count
uint32_t count = semaphore.getCount();
(void)count;
semaphore.release(); // Clean up
}
}
