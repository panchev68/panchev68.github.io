Detailed Description

Base class for button menu widgets in TouchGFX.

Provides logic for managing a scrollable menu of buttons with selection and navigation. Designed for embedded systems, it ensures efficient memory usage and smooth user interaction, supporting customizable appearance and behavior for menu buttons. The class integrates with TouchGFX containers and supports real-time requirements.
