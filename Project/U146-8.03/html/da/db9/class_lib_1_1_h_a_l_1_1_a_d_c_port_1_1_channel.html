Detailed Description

ADC channel configuration wrapper for STM32 ADC peripherals.

Encapsulates ADC channel configuration and setup, including zero offset management, to simplify analog input handling and calibration in embedded STM32 applications, supporting flexible channel initialization and runtime configuration.
