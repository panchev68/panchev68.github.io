Detailed Description

Timeout utility for FreeRTOS tick-based timing.

The Timeout class provides methods to set, check, reset, disable, and manipulate timeouts using FreeRTOS tick counters. It is designed for deterministic embedded systems and ensures type safety, clear ownership, and predictable behavior.
