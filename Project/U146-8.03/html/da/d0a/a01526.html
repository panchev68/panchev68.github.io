Detailed Description

CAN transmit message wrapper.

The TxMessage class combines a CAN transmit header and payload, enabling complete CAN transmit operations with header and data encapsulated in a single object. It inherits from TxHeader and provides storage for the message payload. The payload buffer is fixed at 8 bytes, matching the CAN standard maximum data length. The class provides constructors for default, copy, and parameterized initialization, as well as accessors for the message identifier, payload data, and payload length.
