Detailed Description

CAN filter configuration class.

The Filter class extends CAN_FilterTypeDef to provide a type-safe, convenient interface for configuring CAN hardware filters on STM32 microcontrollers, supporting initialization, parameter validation, and filter activation management for robust embedded CAN communication.
