Lib::BSP::Ethercat::PDI::LAN9252::Base Class Reference

Description

Base class for LAN9252 EtherCAT PDI driver.

Provides register access, chip reset, and SPI communication primitives for the EtherCAT slave controller. Encapsulates low-level operations, register access, and chip state management.
