Detailed Description

Data Abstraction Layer namespace containing global singleton device objects for hardware interface management.

The DAL namespace provides centralized access to hardware peripherals and CAN bus communication nodes through global inline singleton instances, ensuring thread-safe operations and unified device abstraction across all system components and translation units.
