Enumerator
SR_WIP 

Bit 0 - Write in Progress

  • 1 = Device Busy, a Write Registers (WRR), program, erase or other operation is in progress
  • 0 = Ready Device is in standby mode and can accept commands
SR_WEL 

Bit 1 - Write Enable Latch

  • 1 = Device accepts Write Registers (WRR), program or erase commands
  • 0 = Device ignores Write Registers (WRR), program or erase commands
    Note
    This bit is not affected by WRR, only WREN and WRDI commands affect this bit.
SR_BP0 

Bit 2 - Block Protection Protects selected range of sectors (Block) from Program or Erase.

SR_BP1 

Bit 3 - Block Protection Protects selected range of sectors (Block) from Program or Erase.

SR_BP2 

Bit 4 - Block Protection Protects selected range of sectors (Block) from Program or Erase.

SR_E_ERR 

Bit 5 - Erase Error Occurred

  • 1 = Error occurred
  • 0 = No Error
SR_P_ERR 

Bit 6 - Programming Error Occurred

  • 1 = Error occurred
  • 0 = No Error
SR_SRP0 

Bit 7 - Status Register Write Disable

  • 1 = Locks state of SRWD, BP, and configuration register bits when WP# is LOW by ignoring WRR command
  • 0 = No protection, even when WP# is LOW
