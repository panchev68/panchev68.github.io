Introduction

Specifies the include paths for the EthercatSlave project.

Commands

This is a collection of IAR project connectors. The connectors are used to connect the IAR project to the source files and documentation. The connectors are written in XML format and are used by the IAR Embedded Workbench.

Connection

The iarProjectConnection attribute is used to connect the IAR project to the source files and documentation. The connectors are written in XML format and are used by the IAR Embedded Workbench.

Example
<iarProjectConnection name="Project/Library 2.00" oneShot="false">

Set IAR Compiler options

The <extraOptions> tag is used to define the compiler options that should be included. Each <arg> should be a valid compiler option.

Example
<extraOptions>
<args>
<arg>-I$PROJ_DIR$</arg>
<arg>--preinclude=X:/Lib/FW/EWARM/Common.h</arg>
</arg>
</extraOptions>

Include Path

The <includePath> tag is used to define the directories that should be included. Each <path> should point to a directory that contains header files required by the project.

Parameters
<path>Specifies a directory path to be included. Each <path> should be a valid directory containing necessary header files.
Example
<includePath>
<path>$PROJ_DIR$/../</path>
<path>X:/Lib/FW/IAR/Inc</path>
<path>X:/Lib/FW/IAR/Inc/Lib</path>
</includePath>

Add, CopyTo or Reference a file in the project

Add

The add attribute is used to add a file to the project.

Example
<files>
<group name="Documentation" build="false">
<path>$SIMPLELINK_CC13X2_26X2_SDK_INSTALL_DIR$/examples/rtos/CC26X2R1_LAUNCHXL/drivers/spimaster/README.md</path>
</group>
</files>

the file README.md is added to the project directory.

CopyTo

The copyTo attribute is used to copy a file to a specified location in the project.

Example
<files>
<group name="Documentation" build="false">
<path copyTo="$PROJ_DIR$/README.md">$SIMPLELINK_CC13X2_26X2_SDK_INSTALL_DIR$/examples/rtos/CC26X2R1_LAUNCHXL/drivers/spimaster/README.md</path>
</group>
</files>

the file README.md is copied to the project directory.

Reference

The reference attribute is used to reference a file in the project.

Example
<files>
<group name="Documentation" build="false">
<path reference="$PROJ_DIR$/Board.html">$SIMPLELINK_CC13X2_26X2_SDK_INSTALL_DIR$/source/ti/boards/CC26X2R1_LAUNCHXL/Board.html</path>
</group>
</files>

the file Board.html is referenced in the project directory.
