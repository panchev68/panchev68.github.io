Creating a new group

The group tag is used to create a new group in the IAR project. The group tag has two attributes: name and build. The name attribute is used to specify the name of the group. The build attribute is used to specify if the group should be built or not. The build attribute can have two values: true or false. The default value is true.

<iarProjectConnection name="Project/Library 2.00" oneShot="false">
