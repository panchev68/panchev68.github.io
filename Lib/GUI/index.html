GUI Project Readme

Overview

This project is part of the Firmware Library and is developed using IAR Embedded Workbench for ARM (EWARM). It provides a graphical user interface (GUI) for embedded applications.

  • IAR Embedded Workbench for ARM (EWARM)
  • ARM Cortex-M microcontroller
  • Necessary hardware peripherals

Installation

  1. Clone the repository to your local machine.
  2. Open the project in IAR Embedded Workbench.
  3. Build the project using the provided workspace.

Usage

  1. Connect the hardware to your development environment.
  2. Load the compiled firmware onto the microcontroller.
  3. Run the application to interact with the GUI.
