Lib::GUI::Char Class Reference

Detailed Description

A class representing a character with a specific face and dimensions.

The Char class inherits from the Dimension class and represents a character with a specific face (bitmap) and dimensions (width and height).

Note
The face, width, and height must be valid (non-zero) values.
