Lib::GUI::Line Class Reference

Detailed Description

A class to represent and draw a line on a display.

The Line class provides functionality to draw a line on a display using specified coordinates and color.

Parameters
colorThe color of the line.
x0The x-coordinate of the starting point of the line.
y0The y-coordinate of the starting point of the line.
x1The x-coordinate of the ending point of the line.
y1The y-coordinate of the ending point of the line.
p0The starting position of the line.
p1The ending position of the line.
Note
The class assumes the existence of a display object with methods fill() and putPixel().
