Lib::GUI::ProgressBar Class Reference

Detailed Description

A class to represent and manage a graphical progress bar.

The ProgressBar class provides functionality to display and manage a graphical progress bar with customizable value ranges and colors. It inherits from the Canvas class and includes methods to set the progress bar value, value margins, and colors.
