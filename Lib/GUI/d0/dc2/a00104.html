Lib::GUI::ProgressBar::ValueMargins Struct Reference

Detailed Description

ValueMargins struct.

The ValueMargins struct defines the minimum and maximum values for the progress bar. It includes two uint32_t values representing the minimum and maximum values of the progress bar.
