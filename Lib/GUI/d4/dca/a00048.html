◆ defaultFont

Font * Lib::GUI::defaultFont = nullptr
inline

Pointer to default font.

This pointer is used to store the default font object. The default font is used for rendering text on the display. The default font object must be set before any text rendering operations can be performed. If no default font is set, text rendering operations will fail.

◆ display

DisplayDriverBase * Lib::GUI::display = nullptr
inline

Display driver object.

Pointer to display driver object.

This pointer is used to interact with the display driver object. The display driver object is responsible for rendering graphics on the display. The display driver object must be set before any graphics operations can be performed.
