Example of Align class usage

This example demonstrates the usage of the Align class to align objects within a specified dimension.

// Example usage of the Align class
int main()
{
using namespace Lib::GUI;
Dimension box(100, 100);
Dimension object(50, 50);
Position initialPosition(0, 0);
// Align object to the center of the box
Align alignCenter(box, object, initialPosition, Align::HCENTER | Align::VCENTER);
Position centeredPosition = alignCenter;
// Print the centered position
std::cout << "Centered Position: (" << centeredPosition.x << ", " << centeredPosition.y << ")" << std::endl;
return 0;
}

The Align class provides functionality to align objects within a specified dimension based on various alignment modes. The example demonstrates how to use the Align class to align an object to the center of a box. The Align object is constructed with the box dimension, object dimension, initial position, and alignment mode. The aligned position is then retrieved using the Align object, and the result is printed to the console.

The output of the example will be:

Centered Position: (25, 25)

This output indicates that the object has been successfully aligned to the center of the box.
