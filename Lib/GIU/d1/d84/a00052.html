Lib::GUI::Align Class Reference

Detailed Description

A class for aligning objects within a specified dimension.

The Align class provides functionality to align objects within a given dimension using various alignment modes such as left, right, horizontal center, top, bottom, and vertical center. It supports constructing alignment objects with specific dimensions, positions, and modes, and provides operators for aligning objects and setting alignment modes.
