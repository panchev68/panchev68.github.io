Lib::GUI::Text Class Reference

Detailed Description

A class to represent and render text on a canvas.

The Text class provides functionalities to set various properties such as position, colors, canvas, font, and alignment for rendering text. It also includes a method to print the text on the canvas.

◆ print()

void Lib::GUI::Text::print ( const std::string & string)
inline

Prints the given string onto the canvas using the specified font and alignment.

This function renders each character of the provided string onto the canvas. It calculates the dimensions of the string using the specified font and adjusts the position based on the alignment settings. It handles special characters like newline ('
') and carriage return ('\r') appropriately.

Parameters
stringThe string to be printed on the canvas.
