Lib::GUI::Color Class Reference

Detailed Description

Represents a color in RGB format.

The Color class provides a set of predefined colors and allows for the creation of custom colors using RGB values. It supports conversion between RGB values and 32-bit integer representations.

Note
The RGB values are converted to a 16-bit format using a lambda function.

Example usage:

Color red = Color::RED;
uint32_t colorValue = static_cast<uint32_t>(red);

inlinestaticconstexpr

Converts RGB color values to a 16-bit color format.

This function takes red, green, and blue color values (each ranging from 0 to 255) and converts them into a 16-bit color format. The conversion is done by truncating the least significant bits of each color component and combining them into a single 16-bit value.

Parameters
rThe red color component (0-255).
gThe green color component (0-255).
bThe blue color component (0-255).
Returns
A 16-bit color value representing the RGB color.
