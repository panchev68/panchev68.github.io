Lib::GUI::Canvas Class Reference

Detailed Description

The Canvas class provides methods to fill areas with color and render images onto the canvas.

The Canvas class inherits from the Box class and provides methods to fill areas with color and render images onto the canvas. It interacts with a display driver to perform these operations.
