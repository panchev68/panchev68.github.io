Lib::GUI::DisplayDriverBase Class Referenceabstract

Detailed Description

Abstract base class for display drivers.

The DisplayDriverBase class provides an interface for display drivers, defining essential methods that must be implemented by any derived class. These methods include:

  • putPixel: Draw a single pixel at a specified location with a given color.
  • putBlock: Draw a block of pixels at a specified location with given dimensions.
  • fill: Fill a rectangular area with a specified color.
  • cls: Clear the screen.
  • getHeight: Retrieve the height of the display.
  • getWidth: Retrieve the width of the display.

Derived classes must implement these methods to provide specific functionality for different types of displays.
