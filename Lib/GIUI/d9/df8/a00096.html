Lib::GUI::Position Class Reference

Detailed Description

A class to represent a 2D position with x and y coordinates.

The Position class provides a simple representation of a point in 2D space with integer coordinates. It includes default and parameterized constructors.

Note
The coordinates are stored as 16-bit signed integers.
