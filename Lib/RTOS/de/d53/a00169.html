RTOS::SemaphoreCounting Class Reference

Description

Counting semaphore for FreeRTOS.

Provides a counting semaphore with configurable maximum and initial count. Supports take/give operations, ISR-safe methods, and count queries for resource management.

Examples
SemaphoreCounting.

#include <SemaphoreCounting.hpp>
