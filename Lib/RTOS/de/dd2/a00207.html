RTOS::StreamBuffer Class Reference

Description

Stream buffer wrapper for FreeRTOS.

Provides type-safe stream buffer operations with RAII resource management. Supports efficient byte-stream communication between tasks and ISRs.

#include <StreamBuffer.hpp>
