RTOS::SemaphoreCounting Class Reference

Description

Counting semaphore wrapper for FreeRTOS.

Provides type-safe counting semaphore operations with configurable limits. Inherits RAII resource management and move semantics from SemaphoreBase.

#include <SemaphoreCounting.hpp>
