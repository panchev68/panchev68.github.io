RTOS::Timer Class Reference

Description

Comprehensive FreeRTOS software timer C++ wrapper with RAII management.

Encapsulates FreeRTOS timer creation, control, and callback handling using modern C++ design patterns. Provides factory methods for various time units, automatic resource management through RAII, type-safe configuration, and comprehensive timer state management. Supports both one-shot and auto-reload modes with ISR-safe operations for real-time embedded applications.
