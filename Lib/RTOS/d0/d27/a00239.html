RTOS::Timer::Config Struct Reference

Description

Timer configuration parameter structure with validation and factory methods.

Holds complete timer configuration including name, callback function, period, and operating mode. Provides validation of parameters and factory methods for convenient creation of timer configurations with different time units and modes. Ensures minimum period requirements and proper callback assignment for reliable timer operation.
