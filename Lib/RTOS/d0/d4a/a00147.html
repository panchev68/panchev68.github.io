RTOS::EventGroup Class Reference

Description

C++ interface for FreeRTOS event groups with enhanced functionality.

Encapsulates FreeRTOS event group operations for bitwise signaling, waiting, and synchronization between tasks and ISRs with factory methods, RAII support, and advanced timeout management.
