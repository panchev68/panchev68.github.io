RTOS::TickCounter Class Reference

Description

Tick counter utility class for FreeRTOS tick/time conversions and elapsed time calculations.

Provides static methods for converting between milliseconds, seconds, and FreeRTOS ticks, as well as elapsed time and timeout checks. Not instantiable.
