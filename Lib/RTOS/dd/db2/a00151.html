RTOS::Kernel Class Reference

Description

Static interface for FreeRTOS kernel operations with enhanced C++ functionality.

Provides static methods for querying and controlling the FreeRTOS kernel, including scheduler state, tick count, task management, interrupt handling, RAII critical sections, memory statistics, and time utilities. Designed for safe, efficient use in embedded C++ systems with type safety.
