RTOS::Kernel::CriticalSectionISR Class Reference

Description

RAII critical section guard for ISR context with interrupt mask preservation.

Automatically enters critical section on construction and exits on destruction. Preserves and restores interrupt mask state for safe ISR-level critical sections.
