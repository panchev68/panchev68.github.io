RTOS::Timeout Class Reference

Description

Timeout management class for real-time embedded applications.

Provides tick-based timeout functionality with expiration checking, progress tracking, and various utility methods. Supports both absolute and relative timing operations with millisecond and second conversion utilities for convenience.
