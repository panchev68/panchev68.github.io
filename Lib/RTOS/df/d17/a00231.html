RTOS::Timeout Class Reference

Description

Timeout management utility for precise timing control in RTOS operations.

The Timeout class provides comprehensive timeout functionality including setting durations, checking expiration status, tracking progress, and managing timeout extensions. It supports both tick-based and millisecond-based operations with overflow protection and state management.
