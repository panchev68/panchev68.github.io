RTOS::SemaphoreBase Class Referenceabstract

Description

Base class for managing FreeRTOS semaphores.

Provides core interface for FreeRTOS semaphore operations, including creation, destruction, take/give/overwrite, and ISR-safe methods. Designed for inheritance by higher-level synchronization classes.

#include <SemaphoreBase.hpp>
