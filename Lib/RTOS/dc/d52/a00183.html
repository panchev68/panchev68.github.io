RTOS::Mutex Class Reference

Description

FreeRTOS mutex wrapper with C++ safety features and RAII support.

Provides a comprehensive C++ interface for FreeRTOS mutual exclusion primitives with enhanced safety features and automatic resource management. Inherits from SemaphoreBase to leverage common semaphore functionality while adding mutex-specific semantics. Supports both task and ISR contexts with appropriate blocking behavior. Includes integrated Guard support for RAII-based automatic lock management and exception-safe critical sections.
