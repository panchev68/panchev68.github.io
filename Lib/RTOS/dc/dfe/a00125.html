Timer.hpp File Reference

Description

Comprehensive C++ wrapper for FreeRTOS software timers with enhanced functionality.

Provides object-oriented creation and management of FreeRTOS software timers with enhanced features including factory methods, time unit conversion, RAII resource management, and comprehensive timer control. Supports both one-shot and auto-reload modes with ISR-safe operations and flexible callback handling for embedded real-time applications.
