RTOS::StreamBuffer Class Reference

Description

C++ wrapper for FreeRTOS stream buffers.

Encapsulates FreeRTOS stream buffer operations, providing methods for creation, destruction, sending, receiving, and querying buffer state. Supports both task and ISR contexts for flexible data streaming.
