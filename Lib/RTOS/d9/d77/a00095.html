Description

C++ interface for managing FreeRTOS Queue Sets.

Provides the RTOS::QueueSet class, a type-safe, RAII-style wrapper for FreeRTOS Queue Sets. Enables creation, management, and selection of queue set members in both task and ISR contexts for efficient event multiplexing.
