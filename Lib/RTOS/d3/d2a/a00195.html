RTOS::Semaphore Class Reference

Description

Encapsulates a FreeRTOS binary semaphore with enhanced functionality.

Provides methods for creation, destruction, take/give/overwrite operations, ISR-safe methods, factory functions, RAII support, and advanced synchronization patterns.
