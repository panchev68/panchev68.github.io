RTOS::MemoryManager Class Reference

Description

Static memory management utility for FreeRTOS heap operations.

Provides thread-safe memory allocation and deallocation methods with comprehensive error handling and optional diagnostic capabilities. Wraps FreeRTOS heap functions with C++ safety features including size validation, null pointer checks, and allocation failure handling. Supports compile-time diagnostic tracking for memory usage analysis and debugging in embedded applications.

Memory allocation and deallocation operation status codes.

Enumeration defining possible outcomes of memory management operations. Used to provide detailed error reporting for allocation and deallocation failures, enabling robust error handling in embedded applications where memory management failures must be handled gracefully.

Enumerator
SUCCESS 

Memory operation completed successfully.

INVALID_SIZE 

Requested size is zero or invalid.

OUT_OF_MEMORY 

Insufficient heap memory available.

NULL_POINTER 

Null pointer passed to deallocation function.

SIZE_TOO_LARGE 

Requested size exceeds total heap capacity.
