RTOS::Timeout Class Reference

Description

Timeout wrapper for RTOS tick values.

Provides a type-safe interface for managing timeouts in RTOS tasks, supporting construction from ticks, milliseconds, or seconds. Includes expiration checks, remaining/elapsed time queries, arithmetic operations, and helper methods for disabled/immediate timeouts.
