RTOS::TickCounter Class Reference

Description

Static utility class for FreeRTOS tick count operations and timing calculations.

Provides comprehensive timing utilities for embedded real-time systems using FreeRTOS tick counts. Offers type-safe tick operations, context-aware tick retrieval for both task and ISR contexts, and overflow-aware elapsed time calculations. Includes convenient conversion utilities between different time units (ticks, milliseconds, seconds) and timeout management functions. All methods are static for zero-overhead access without instantiation, making it highly efficient for embedded applications where timing precision and performance are critical. The class is non-instantiable by design.
