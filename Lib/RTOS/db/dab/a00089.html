Description

Mutex class providing mutual exclusion for RTOS tasks and its RAII guard for scoped locking.

This file defines the Mutex class, which extends SemaphoreBase to provide mutual exclusion mechanisms, and its nested Guard class for RAII-style scoped locking. It also provides type aliases for convenience.
