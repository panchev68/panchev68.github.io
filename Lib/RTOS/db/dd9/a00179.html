RTOS::Guard< LOCKABLE_TYPE > Class Template Reference

Description

template<typename LOCKABLE_TYPE>
class RTOS::Guard< LOCKABLE_TYPE >

Template-based RAII guard for automatic resource management of lockable objects.

Provides comprehensive automatic resource management for any type that implements lock/unlock semantics. Uses RAII principles to ensure automatic unlocking when the guard goes out of scope, preventing deadlocks and resource leaks. Supports multiple locking strategies including immediate locking, deferred locking, try-lock operations, and timeout-based locking. Thread-safe and exception-safe for robust embedded applications. Move-only semantics prevent accidental copying while allowing efficient transfer of ownership.

Template Parameters
LOCKABLE_TYPEType that provides lock/unlock interface compatible with Mutex semantics.

Locking strategy enumeration for controlling guard construction behavior.

Defines different approaches for acquiring locks during guard construction, enabling flexible synchronization patterns based on application requirements. Each strategy provides different blocking and timing characteristics suitable for various embedded scenarios.

Enumerator

Acquire lock immediately during construction, blocking if necessary.

DEFERRED 

Defer lock acquisition until manual lock() call is made.

TRY_LOCK
