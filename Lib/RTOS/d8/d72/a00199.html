RTOS::SemaphoreBase Class Referenceabstract

Description

Abstract base for FreeRTOS semaphore wrappers.

Implements core logic for semaphore management and ISR-safe operations. Provides RAII resource management and type-safe operations.

#include <SemaphoreBase.hpp>
