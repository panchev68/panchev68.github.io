RTOS::Task Class Reference

Description

Description of Task type.

This class encapsulates creation, control and lifetime management of a FreeRTOS task. It stores task attributes such as name, stack size and priority and exposes start/stop, notification and suspend/resume operations with error codes suitable for embedded error handling layers.

Constructor of Task.

Construct a Task wrapper with the provided entry callable, task name, stack size and priority. The constructor performs lightweight validation and stores parameters. The actual FreeRTOS task is not created until start() is called.

Convenience helper to create a Task and start it immediately.

Allocates a Task instance on the heap and starts it. Returns the native handle to the created task. This convenience is intended for quick task creation patterns and will leak the wrapper pointer if not tracked.

Parameters
entryFunctionCallable to run in the task.
taskNameName for the created task.
stackSizeStack size in bytes for the created task.
priorityPriority for the created task.
Returns
Native handle of the created task or nullptr on failure.

Execute the stored entry function.
