RTOS::Queue< T, QueueLength > Class Template Reference

Description

template<typename T, size_t QueueLength = 1>
class RTOS::Queue< T, QueueLength >

Queue class template providing RTOS queue abstraction for typed message passing.

This class template wraps FreeRTOS queue APIs for type-safe message passing, supporting dynamic and static allocation, ISR-safe operations, and registry management.
