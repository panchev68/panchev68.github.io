Detailed Description

XCOM communication protocol implementation.

This file defines the Protocol class for the XCOM communication protocol, providing methods for building and parsing protocol frames, handling escape coding, and CRC16 checksum calculation. The Protocol class encapsulates frame structure, encoding/decoding logic, and error checking mechanisms for reliable data transmission in embedded systems. The frame format includes a start delimiter, length, command, payload, and CRC16 checksum, with support for escape substitution to avoid delimiter collisions. The implementation ensures type safety, input validation, and efficient memory usage using modern C++17 features and standard containers.
