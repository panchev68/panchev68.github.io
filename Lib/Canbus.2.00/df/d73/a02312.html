Detailed Description

Time change rule for daylight saving or similar adjustments.

The TimeChangeRule struct defines a rule for time changes such as daylight saving time. It includes the rule name, week of the month, day of the week, month, hour, and offset in minutes. This allows for flexible configuration of time zone transitions.
