Detailed Description

Driver mode enumeration for HAL drivers.

Defines available modes of operation: blocking, interrupt, or DMA.

enum class DriverMode : uint8_t

Enumeration of driver operation modes for HAL drivers.

BLOCK: Blocking mode; operations wait until completion. IT: Interrupt mode; operations use interrupts for completion. DMA: DMA mode; operations use direct memory access.
