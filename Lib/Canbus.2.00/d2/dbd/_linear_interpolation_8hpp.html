Detailed Description

Linear interpolation implementation for STM32 embedded systems.

This file provides a LinearInterpolation class for performing linear interpolation between data points. The implementation is optimized for embedded systems with fixed-point arithmetic support and constexpr evaluation capabilities for compile-time interpolation when possible. The class supports efficient runtime and compile-time interpolation, type safety, overflow protection, and bounded access. It is suitable for embedded and real-time applications requiring fast, reliable interpolation between variable data points, including lookup tables and direct value interpolation.
