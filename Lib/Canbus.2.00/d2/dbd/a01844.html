Detailed Description

XBee server base class.

The ServerBase class provides a foundation for XBee server implementations, managing message handling, connection management, and transaction state transitions. It supports efficient message transmission, reception, and connection state updates, enabling reliable Digi XBee API communication. The class is designed for extensibility, allowing derived classes to implement custom protocol logic and event handling. It emphasizes type safety, performance, and robust resource management for embedded and general-purpose applications.
