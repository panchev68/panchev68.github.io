Detailed Description

template<typename T_VALUE, size_t T_WINDOW_SIZE>
class Lib::Helper::Integrator< T_VALUE, T_WINDOW_SIZE >

Moving average integrator for fixed-size window.

The Integrator class template computes a moving average over a fixed-size window of arithmetic values. It uses a circular buffer to store the most recent values and provides methods to add values, compute the average, reset the window, and check if the window is full. The class is type-safe and suitable for embedded systems and real-time signal processing.

Template Parameters
T_VALUEArithmetic type for values (e.g., int, float, double).
T_WINDOW_SIZESize of the moving average window (must be > 0).
