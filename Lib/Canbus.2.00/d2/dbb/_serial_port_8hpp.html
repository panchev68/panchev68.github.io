Detailed Description

USB SerialPort class and configuration.

Provides a comprehensive implementation of the SerialPort class for USB serial communication. Includes configuration, open/close, send, reset, baud rate management, and event callback support. The class encapsulates UART port management, USB reset pin control, RTOS task handling, and message buffering for efficient and reliable USB serial data transfer. Designed for STM32 embedded systems, it ensures robust resource management, type safety, and extensibility for custom event handling and integration with RTOS environments.
