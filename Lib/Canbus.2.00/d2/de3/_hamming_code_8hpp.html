Detailed Description

Hamming code encoder/decoder for 16-bit values.

This file provides a class for encoding and decoding 16-bit values using the Hamming(21,16) code. The implementation supports single-bit error detection and correction, making it suitable for embedded systems and communication protocols requiring robust error handling. The class provides methods for encoding data, decoding and correcting single-bit errors, and querying the encoded word. The design uses precomputed arrays for parity and data bit positions to optimize performance and minimize memory usage.
