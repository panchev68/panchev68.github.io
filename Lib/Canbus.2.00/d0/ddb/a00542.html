Detailed Description

Queue abstraction for FreeRTOS queue management. This file provides a C++17 template class for FreeRTOS queue creation, control, and resource management. The Queue class encapsulates queue lifecycle, ownership, and basic operations for sending, receiving, peeking, and overwriting queue elements in both task and ISR contexts. It supports move semantics, safe destruction, and provides APIs for registry management. The class ensures proper resource management and error handling using RAII principles.
