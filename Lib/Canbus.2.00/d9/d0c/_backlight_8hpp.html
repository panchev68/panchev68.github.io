Detailed Description

Backlight control and dimmer management for STM32 PWM-based display backlight.

This file provides the Backlight class for controlling display backlight brightness using PWM timers, including smooth fading and automatic dimming via the nested Dimmer class. The design supports configurable fade steps, sleep timeouts, and minimum/maximum brightness levels. The Backlight class integrates with RTOS tick services for periodic updates and uses HAL timer handles for hardware PWM. The Dimmer class manages gradual brightness transitions and sleep-based dimming, ensuring smooth user experience and power efficiency. All timer channel mappings and error handling are encapsulated for robust embedded operation.
