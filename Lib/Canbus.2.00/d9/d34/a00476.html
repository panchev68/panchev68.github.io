Detailed Description

Moving average integrator for fixed-size window.

This file defines the Integrator template class, which computes a moving average over a fixed-size window of arithmetic values. The class supports adding new values, computing the average, resetting the window, and checking if the window is full. It is suitable for embedded and real-time applications where efficient averaging of sensor or signal data is required. The implementation uses a circular buffer for storage and ensures type safety and bounded arithmetic. The code is optimized for clarity, safety, and performance.
