HammingCode

Example demonstrating the use of the HammingCode class for encoding and decoding 16-bit values. This example shows how to encode a 16-bit value using the Hamming(21,16) code, introduce a single-bit error, and then decode and correct the error using the HammingCode class. It prints the original value, the encoded word, the corrupted word, and the decoded value after correction.

Example Usage
#include "Lib/Math/HammingCode.hpp"
#include <iostream>
int main() {
Lib::Math::HammingCode hamming;
uint16_t original = 0xA5B3;
uint32_t encoded = hamming.encode(original);
std::cout << "Original: 0x" << std::hex << original << std::endl;
std::cout << "Encoded: 0x" << std::hex << encoded << std::endl;
// Introduce a single-bit error at bit position 5
uint32_t corrupted = encoded ^ (1U << 4);
std::cout << "Corrupted: 0x" << std::hex << corrupted << std::endl;
uint16_t decoded = hamming.decode(corrupted);
std::cout << "Decoded: 0x" << std::hex << decoded << std::endl;
return 0;
}
