Detailed Description

template<typename T = uint8_t>
class Lib::Helper::DataHandle< T >

DataHandle class template for managing raw data buffers.

The DataHandle class provides a generic, type-safe handle for memory buffers, encapsulating a pointer to the data and its size. It supports construction from typed references or raw pointers, enabling flexible access and manipulation of memory regions. Designed for embedded and general-purpose use, it ensures correctness, type safety, and efficient resource usage.
