Detailed Description

template<typename T>
class Lib::Helper::Container< T >

Container manages pointers to objects of type T.

The Container class provides efficient management of object pointers, preventing duplicates and supporting batch addition, removal, clearing, and callback-based processing. It uses std::vector for dynamic storage and offers direct access to the internal vector for advanced operations. The class is designed for type safety, performance, and robust resource management in embedded and general-purpose applications.
