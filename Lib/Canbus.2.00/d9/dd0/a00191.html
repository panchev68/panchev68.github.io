Detailed Description

Digi API Mode enumeration for XBEE modules.

Defines the ApiMode enum class representing Digi XBee API operating modes. This file provides a type-safe enumeration for selecting the operating mode of Digi XBee modules, supporting transparent, API1, and API2 modes. The type ensures clarity and reliability in embedded communication code, using scoped enumerations for safety and maintainability.
