Detailed Description

Namespace for BSP library classes and functions.

The Lib::BSP namespace contains all abstractions, helpers, and types related to the Board Support Package (BSP) library. It provides APIs for hardware initialization, peripheral configuration, and board-specific features required in embedded systems. All BSP-related code should reside within this namespace to ensure modularity and maintainability.
