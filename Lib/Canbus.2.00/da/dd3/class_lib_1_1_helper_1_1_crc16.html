Detailed Description

CRC16 calculation helper class.

This class provides methods to calculate CRC16 checksums for raw buffers and byte arrays. It uses the CRC-16-CCITT polynomial (0x1021) and an initial value of 0xFFFF. The class is designed for type safety, performance, and robust resource management in embedded and general-purpose applications.
