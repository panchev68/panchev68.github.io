Detailed Description

RTC abstraction layer for STM32 HAL.

Provides the RTCBase class for managing RTC operations, including reading, writing, and managing RTC (Real-Time Clock) data. This file ensures robust integration with STM32 HAL, supporting flexible RTC operations, type safety, and maintainability for various hardware requirements. The design promotes extensibility and reliability in embedded applications, facilitating seamless adaptation to evolving system needs.
