Detailed Description

Frame0x17 class for Remote AT Command Request.

Represents a Digi XBee API frame for querying or setting AT command parameters on a remote device. For parameter changes to take effect, set the Apply Changes option or send an AC command to the remote. The response frame for this request is Remote AT Command Response (Frame 0x97).
