ServiceBase Class Referenceabstract

Detailed Description

A base class for CAN bus services.

The ServiceBase class provides a base class for CAN bus services. It inherits from Lib::HAL::CAN::Port and includes functionality for starting, stopping, sending, and receiving CAN messages. It also provides diagnostic information and event handling for received messages and errors.

Note
This class uses FreeRTOS for task and queue management.
