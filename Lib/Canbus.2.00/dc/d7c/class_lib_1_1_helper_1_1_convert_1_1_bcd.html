Detailed Description

Utility class for Binary-Coded Decimal (BCD) conversions.

The Bcd class provides static methods for converting BCD values to ASCII strings and characters. It supports conversion of 16-bit BCD values to their ASCII string representations and conversion of individual BCD digits to ASCII characters. The class is designed for use in embedded systems and applications requiring efficient and type-safe BCD to ASCII conversions.
