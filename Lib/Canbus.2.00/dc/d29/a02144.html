Detailed Description

template<class T_HANDLE>
class Lib::HAL::Handle< T_HANDLE >

Type-safe wrapper for device handle pointers. Encapsulates a device handle pointer, enforcing non-null initialization, type safety, and convenient access. Supports comparison and retrieval.
