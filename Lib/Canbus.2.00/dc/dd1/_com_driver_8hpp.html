Detailed Description

ComDriver class for Digi XBee modules.

Provides low-level UART communication, frame handling, and hardware reset for Digi XBee modules. The ComDriver class encapsulates UART communication, frame parsing, hardware reset, and event hooks for message transmission and reception. The implementation emphasizes type safety, efficient resource management, and extensibility, allowing seamless integration with Digi XBee API infrastructure and RTOS environments.
