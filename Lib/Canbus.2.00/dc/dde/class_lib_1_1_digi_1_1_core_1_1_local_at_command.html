Detailed Description

LocalAtCommand class for Digi XBee modules.

Manages sending and receiving local AT commands, handling response events, and maintaining open/close state for command transactions. The class provides queue-based message handling, event-driven response processing, and robust lifecycle management for Digi XBee AT command operations.
