Detailed Description

Data class for Digi Core Frame internal data container.

Provides a container for frame type and associated data, supporting efficient assignment and append operations for Digi Core Frame handling. The Data class enables flexible manipulation of frame data for Digi XBee API frames, including type-safe appending of various integer types in big-endian order. This design promotes extensibility, type safety, and efficient integration with Digi XBee API infrastructure and RTOS environments.
