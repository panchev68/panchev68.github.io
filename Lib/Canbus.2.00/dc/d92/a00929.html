ServiceBase::Diagnostic Struct Reference

Detailed Description

Structure to hold diagnostic information for CAN bus communication.

This structure contains various counters and error codes related to CAN bus communication, including the number of received and sent messages, the number of times the transmit mailbox was aborted, the number of times the receive FIFO was full, and error-related information.
