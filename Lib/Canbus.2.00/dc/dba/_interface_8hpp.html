Detailed Description

LAN9252 EtherCAT Slave Controller SPI interface.

Provides thread-safe SPI communication methods for LAN9252 EtherCAT Slave Controller. This file defines the Interface class, which encapsulates SPI and NSS pin operations with mutex protection. The class provides methods for full-duplex SPI transfers, single/multi-byte read/write operations, and ensures thread safety using RTOS mutexes. It is designed for robust and efficient communication with the LAN9252 EtherCAT slave controller in embedded systems.
