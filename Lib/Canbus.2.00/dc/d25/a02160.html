Detailed Description

template<Mode T_MODE>
class Lib::HAL::Timer< T_MODE >

Timer class template for TIM peripheral management.

Defines an interface for TIM peripheral management, parameterized by Mode (e.g., ENCODER). Provides RAII ownership of TIM handle and channel, supporting future extension for callback/event integration. The class is designed for embedded applications, ensuring type safety, maintainability, and compatibility with STM32 HAL and project documentation guidelines.
