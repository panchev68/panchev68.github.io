Detailed Description

Provides utilities for Binary-Coded Decimal (BCD) conversions.

This file defines the Bcd class, which offers static methods for converting BCD values to ASCII strings and characters. The class supports conversion of 16-bit BCD values to their ASCII string representations and conversion of individual BCD digits to ASCII characters. It is suitable for embedded systems and applications requiring efficient and type-safe BCD to ASCII conversions.
