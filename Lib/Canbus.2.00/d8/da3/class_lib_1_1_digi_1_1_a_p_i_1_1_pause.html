Detailed Description

Pause service base class.

The Pause class provides an abstract interface for pause and resume operations in Digi XBee modules. Derived classes can implement custom logic for controlling pause and resume behavior, enabling flexible management of Digi XBee API components. The class is designed for extensibility and maintainability, supporting robust resource management and type safety.
