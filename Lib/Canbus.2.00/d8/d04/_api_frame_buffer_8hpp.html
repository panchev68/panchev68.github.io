Detailed Description

XBee API1 Frame Buffer for serialized data.

Provides the ApiFrameBuffer class for storing and managing serialized XBee API1 frames. This type supports assignment, appending, and clearing operations, ensuring efficient and type-safe handling of frame data for embedded communication. The buffer uses fixed-width integer types and standard containers for reliability and maintainability. It is designed for robust embedded use, with bounded operations and input validation.
