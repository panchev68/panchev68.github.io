Detailed Description

Counting semaphore wrapper for FreeRTOS synchronization primitives.

Provides a counting semaphore abstraction for real-time STM32 embedded applications using FreeRTOS. This class enables creation and management of counting semaphores, supporting configurable maximum and initial counts for resource pooling and multi-token synchronization. It offers methods to query the current count, maximum capacity, initial value, and state of the semaphore, facilitating robust resource management and signaling between tasks. The class enforces input validation, safe handle management, and exposes utility methods for checking semaphore fullness and emptiness. It inherits from SemaphoreBase, leveraging RAII and move semantics for safe resource cleanup and transfer.
