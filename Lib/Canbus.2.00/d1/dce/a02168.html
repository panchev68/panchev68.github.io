Detailed Description

Manages a dynamic memory buffer.

The Buffer class encapsulates a dynamically resizable memory buffer, supporting construction from raw pointers, constant data, and copy semantics. It provides type-safe access, input validation, and robust resource management for efficient buffer operations.
