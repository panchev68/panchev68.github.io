Detailed Description

Hamming code encoder/decoder for 16-bit values.

This class implements the Hamming(21,16) code for encoding and decoding 16-bit values. It supports single-bit error detection and correction, and provides methods for encoding, decoding, and querying the encoded word. The implementation uses precomputed arrays for parity and data bit positions to optimize performance and minimize memory usage.
