Detailed Description

Defines the Group class for logical error grouping and management.

Provides functionality to group errors for improved management and control, enabling isolation and handling of errors between independent nodes or subsystems. The Group class allows associating errors with a named group and optional user data, facilitating granular error handling, reporting, and system partitioning in embedded applications. It supports user-defined tags for custom metadata and ensures consistent group identification via captions. This class is designed for robust error management in complex systems where error isolation and grouping are critical for reliability and maintainability.
