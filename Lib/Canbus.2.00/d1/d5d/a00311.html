Detailed Description

ADC port alias template header.

Provides an alias template for ADC ports, mapping driver modes to corresponding ADC port types for simplified and type-safe usage in embedded applications. Enables convenient selection of ADC port types based on driver mode for STM32 embedded systems. This file ensures robust integration with STM32 ADC drivers, supporting flexible configuration and efficient resource management in embedded applications. The design promotes type safety, maintainability, and extensibility for various ADC driver modes, facilitating seamless adaptation to evolving hardware requirements.
