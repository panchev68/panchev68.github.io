Detailed Description

Incremental encoder timer interface.

Provides timer-based incremental encoder and button event handling for STM32 MCUs. This file defines the EncoderTimer class, which interfaces with a hardware timer and GPIO pin to track incremental encoder position and button events. It uses RTOS queues for event synchronization and supports detection of button push, pull, hold, and encoder rotation events. The implementation ensures type safety, bounded control flow, and modern C++17 practices for embedded systems.
