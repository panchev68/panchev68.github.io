Detailed Description

Utility unions for LAN9252 EtherCAT slave controller. Provides type-safe conversions between uint16_t, uint32_t, uint64_t and their byte/word layouts. This file defines the Helper class, which encapsulates unions for efficient conversion between integer types and their corresponding byte, word, and dword representations. These utilities are essential for low-level register access and communication with the LAN9252 EtherCAT slave controller, enabling precise manipulation of data formats.
