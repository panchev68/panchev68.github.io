template<typename T>
class Lib::Helper::Convert::Hex< T >

Hex template specialization for conversions.

The Hex class provides type-safe conversion methods for unsigned integer types, supporting conversion to hexadecimal strings, BCD formats, and BCD strings with optional leading zero suppression. Template specializations are provided for 8-bit, 16-bit, and 64-bit unsigned integers.
