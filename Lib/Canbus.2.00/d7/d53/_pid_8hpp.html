Detailed Description

PID controller implementation for control systems.

This file provides a robust and flexible PID controller class for discrete-time control systems. The implementation supports advanced features such as output clamping, integral windup protection, derivative kick prevention, and bumpless transfer between automatic and manual modes. The controller can be configured for direct or reverse action and allows runtime adjustment of gains, output limits, and sample time. The design is suitable for embedded and real-time applications, ensuring safe parameter validation and bounded control outputs. Type aliases for single and double precision are provided for convenience.
