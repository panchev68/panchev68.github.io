Detailed Description

Spectrum Analyzer service for Digi API.

Provides spectrum analysis functionality using Digi's LocalAtCommand interface. The SpectrumAnalyzer class performs periodic energy scans, processes spectrum data, and provides current, average, and maximum spectrum arrays. Designed for embedded Digi modules, it ensures robust spectrum analysis, error handling, and maintainable, type-safe service development. The class supports event-driven updates and reliable access to spectrum data for diagnostics and monitoring.
