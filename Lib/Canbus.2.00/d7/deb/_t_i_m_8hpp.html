Detailed Description

STM32 HAL Timer/Encoder interface header.

This file provides the interface for STM32 timer and encoder peripherals in the HAL layer. It includes type definitions and forward declarations required for timer and encoder operations. The file is designed for embedded applications using STM32 microcontrollers, ensuring type safety, modularity, and compatibility with C++17 standards. It supports timer modes, handles, and template specializations for flexible hardware abstraction. The header is intended for use in both real-time and non-real-time contexts, with careful separation of implementation details and interface contracts.
