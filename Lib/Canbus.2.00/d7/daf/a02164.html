Detailed Description

Timer class template specialization for ENCODER mode.

This specialization manages STM32 TIM peripherals in ENCODER mode, supporting efficient value change hooks and safe resource handling. It enables type-safe usage, event-driven updates, and integration with STM32 HAL. The class is designed for embedded applications, ensuring compatibility with C++17 standards and project documentation guidelines.
