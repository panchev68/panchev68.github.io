Detailed Description

template<typename T_VALUE, typename T_INPUT = T_VALUE>
class Lib::Math::LinearInterpolation< T_VALUE, T_INPUT >

Linear interpolation class for embedded applications.

Provides efficient linear interpolation between two points or within a lookup table. Supports both runtime and compile-time interpolation with type safety and overflow protection. Suitable for embedded and real-time systems requiring fast, reliable interpolation between variable data points, including lookup tables and direct value interpolation.

Template Parameters
T_VALUENumeric type for values (typically float, double, or fixed-point types).
