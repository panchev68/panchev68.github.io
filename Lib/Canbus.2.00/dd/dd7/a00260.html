Detailed Description

Frame0x80.hpp: 64-bit Receive Packet Response for Digi Core.

Provides the Frame0x80 class for parsing and representing 64-bit receive packet responses from Digi XBee modules. This class encapsulates Digi XBee API frame construction for 64-bit receive packet responses, supporting type safety, extensibility, and efficient integration with Digi XBee API infrastructure and RTOS environments. It enables reliable communication and host correlation for embedded and general-purpose applications.
