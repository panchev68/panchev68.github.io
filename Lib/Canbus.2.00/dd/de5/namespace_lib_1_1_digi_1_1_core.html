Detailed Description

Digi XBee API functionality namespace.

This namespace contains API components and utilities for Digi XBee interaction. All Digi XBee related API logic should be placed within this namespace.

enum class ApiMode
strong

Digi API Mode enumeration.

Represents the available Digi XBee API operating modes, including transparent, API1, and API2.

Enumerator
TRANSPARENT 

API modes are disabled; the module operates in transparent mode.

API1 

API mode without escaped characters.

API2 

API escaped operating mode (API 2).
