Detailed Description

TickService provides periodic tick event dispatching to registered clients.

This file defines the TickService class, which manages periodic tick events and notifies registered clients in a thread-safe manner. Clients derive from ClientBase and are automatically registered/unregistered with the TickService singleton. The service runs in its own task, dispatching tick events at a configurable interval. Thread safety is ensured via a mutex. The service supports dynamic client registration, configurable tick interval, and priority management. Designed for embedded RTOS systems.
