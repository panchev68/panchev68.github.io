Detailed Description

QueueSet abstraction for FreeRTOS queue set management.

This file provides a C++17 class for FreeRTOS queue set creation, control, and resource management. The QueueSet class encapsulates queue set lifecycle, ownership, and basic operations for adding, removing, and selecting queue set members. It supports move semantics, safe destruction, and provides APIs for selecting members in both task and ISR contexts. The class ensures proper resource management and error handling using RAII principles.
