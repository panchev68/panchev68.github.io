Detailed Description

RTOS timer abstraction for FreeRTOS software timers.

Provides a high-level interface for creating, managing, and controlling FreeRTOS timers. Supports one-shot and auto-reload modes, ISR-safe operations, and conversion between ticks and milliseconds. Ensures type safety, resource management, and clear error handling for embedded applications.
