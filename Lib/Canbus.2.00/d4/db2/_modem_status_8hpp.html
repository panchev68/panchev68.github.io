Detailed Description

Modem Status Service class for Digi XBee modules.

Handles modem status responses and provides event hooks for various modem status changes. This class manages modem status frames and emits events for status changes, supporting robust embedded communication. It uses type-safe enumerations and standard function objects for event handling, ensuring reliability and maintainability in Digi XBee module integration.
