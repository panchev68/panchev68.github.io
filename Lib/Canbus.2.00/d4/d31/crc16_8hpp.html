Detailed Description

CRC16 calculation helper for buffers and byte arrays.

This file provides a utility class for calculating CRC16 checksums over raw buffers and byte arrays. The implementation uses the standard CRC-16-CCITT polynomial (0x1021) and an initial value of 0xFFFF. It supports calculation from both raw pointers and std::vector<uint8_t> containers, making it suitable for embedded and general-purpose applications where data integrity verification is required. The class avoids dynamic allocation, uses fixed-width types, and ensures type safety. It is designed for fast, deterministic CRC computation in resource-constrained environments.
