Detailed Description

I2C port abstraction for STM32 HAL.

Provides the I2CPort class for STM32 I2C communication, supporting master/memory transfers, event-driven callbacks, and error handling for embedded bus control. Uses modern C++17 features and standard libraries for safety and maintainability. Integrates with RTOS event group and device manager for multi-instance support.
