Detailed Description

DeviceManager.hpp file for device management in Lib::BSP.

This file defines the DeviceManager class, which provides centralized management of device objects in the BSP layer. It supports automatic device registration, setup, and tick event handling using an RTOS timer. The DeviceManager ensures that all registered devices are properly initialized and receive periodic tick events for real-time operations. Devices are managed via a singleton pattern, and registration is handled automatically through the DeviceBase constructor and destructor. This design simplifies device lifecycle management and enables scalable extension of device types.
