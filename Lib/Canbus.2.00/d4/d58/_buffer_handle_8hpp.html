Detailed Description

Provides BufferHandle for safe and efficient byte buffer management.

This file defines the BufferHandle class, which encapsulates dynamic memory management for byte buffers. BufferHandle supports construction from raw pointers, copy semantics, and assignment, ensuring safe and efficient handling of buffer data. The implementation validates inputs, manages resource ownership, and provides type-safe access to buffer contents. Suitable for embedded and general-purpose applications requiring robust buffer management.
