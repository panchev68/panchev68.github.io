Detailed Description

template<typename T>
class Lib::Helper::Singleton< T >

Template for creating singleton classes.

The Singleton class template ensures only one instance of a class exists and provides global access. It uses a static local variable for thread-safe initialization and enforces non-copyable semantics. Suitable for embedded and general-purpose applications requiring controlled global access to a single instance of a class.

TType to be made a singleton.
