Detailed Description

USB SerialPortRepeaterBase class for USB serial repeater functionality.

Defines the SerialPortRepeaterBase class, extending SerialPort to support USB serial port repeater features, including open/close, USB enable/disable, connection detection, and callback support. The class manages USB connection detection, port enable/disable, event callbacks, and integrates with STM32 USB CDC device stack. It provides robust resource management, type safety, and extensibility for custom event handling and integration with RTOS environments. Designed for STM32 embedded systems, it ensures reliable USB serial data transfer and flexible repeater functionality for advanced USB communication scenarios.
