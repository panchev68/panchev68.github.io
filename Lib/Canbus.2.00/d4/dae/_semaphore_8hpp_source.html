2 * This file provides a C++17 class for FreeRTOS semaphore creation, control, and resource management.
3 * The Semaphore class encapsulates semaphore lifecycle, ownership, and basic operations for binary semaphores.
4 * It supports move semantics, safe destruction, and provides APIs for taking, giving, and overwriting semaphores
5 * in both task and ISR contexts. The class ensures proper resource management and error handling using RAII principles.

21 /** Semaphore abstraction for FreeRTOS binary semaphore management.

32 /** State of the semaphore.

34 */
35 enum class State : uint8_t {
36 AVAILABLE = 0, /*!< Semaphore is available. */
37 TAKEN = 1, /*!< Semaphore is taken. */

202 /** Get the state of the semaphore.
203 * @return State of the semaphore.
204 */
205 State getState() const {

207 return uxSemaphoreGetCount(handle) > 0 ? State::AVAILABLE : State::TAKEN;
