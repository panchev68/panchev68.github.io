Detailed Description

Semaphore abstraction for FreeRTOS semaphore management.

This file provides a C++17 class for FreeRTOS semaphore creation, control, and resource management. The Semaphore class encapsulates semaphore lifecycle, ownership, and basic operations for binary semaphores. It supports move semantics, safe destruction, and provides APIs for taking, giving, and overwriting semaphores in both task and ISR contexts. The class ensures proper resource management and error handling using RAII principles.
