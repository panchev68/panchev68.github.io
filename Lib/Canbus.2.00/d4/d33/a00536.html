Detailed Description

C++ RAII wrapper for FreeRTOS mutex functionality. Provides type-safe, resource-managed access to FreeRTOS mutexes for STM32/FreeRTOS systems. Supports both normal and recursive mutex types, safe creation, destruction, locking, unlocking, and status queries. Offers LockGuard for scoped locking, preventing deadlocks and resource leaks. Ensures correct resource management and error handling, improving reliability and maintainability.
