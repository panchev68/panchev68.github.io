Detailed Description

Connection class for Digi XBee API.

This file provides the Connection class for Digi XBee modules, managing connection state, quality monitoring, and connection statistics. The Connection class encapsulates connection state transitions, quality assessment, and runtime statistics, supporting robust monitoring and diagnostics for Digi XBee modules in embedded and general-purpose applications. The implementation emphasizes type safety, efficient resource management, and extensibility, allowing seamless integration with Digi XBee API infrastructure and RTOS environments.
