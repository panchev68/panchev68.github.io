Detailed Description

Optimization configuration and performance tuning settings for embedded RTOS C++ library.

This header provides centralized optimization configuration and compile-time performance tuning for the embedded RTOS C++ library. Contains optimization macros, inline configurations, and performance-related constants that improve runtime efficiency in STM32 embedded applications. The configuration enables aggressive inlining, branch prediction, memory alignment, and buffer sizing strategies tailored for ARM Cortex-M4 and similar microcontrollers. It supports compile-time selection of optimization levels, zero-overhead abstractions, and resource-efficient allocation for queues, message buffers, and tasks. The file is designed to be included in performance-critical modules to ensure consistent optimization across the RTOS library, facilitating maintainable and robust embedded software.
