Detailed Description

Utility functions for string and value conversions.

This header defines functions for converting strings to values and vice versa, as well as constants for numerical operations. It provides lightweight implementations suitable for embedded and real-time systems, focusing on type safety, bounded access, and minimal dependencies. The functions include string-to-long, string-to-double, and integer-to-string conversions, supporting various bases and error handling.
