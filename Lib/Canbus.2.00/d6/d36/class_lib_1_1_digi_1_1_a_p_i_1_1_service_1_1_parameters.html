Detailed Description

Parameters service class.

Provides an interface to read, write, and update both static and dynamic parameters of a module via local and remote AT commands. Supports operations such as reading node identifiers, serial numbers, PAN ID, channel, firmware/hardware versions, and dynamic metrics. Manages state transitions for parameter updates, supports both local and remote configuration, and ensures synchronization with the module's current state. Designed for RTOS environments and integrates with ServiceBase.
