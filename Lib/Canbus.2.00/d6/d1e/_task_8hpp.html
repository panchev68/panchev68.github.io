Detailed Description

Task abstraction for FreeRTOS thread management.

This file provides a C++17 wrapper for FreeRTOS task creation, control, and notification. It encapsulates task lifecycle, priority, stack management, and notification mechanisms. The Task class enables safe and convenient management of FreeRTOS tasks, including creation, suspension, resumption, priority adjustment, and notification handling. It supports both static and dynamic task creation, provides type-safe notification APIs, and ensures proper resource management using RAII principles.
