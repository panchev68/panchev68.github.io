Detailed Description

DIGI API Frame Type enumeration for Digi Core Frame.

Defines frame type constants for Digi XBee API frames, enabling type-safe identification and handling of various Digi XBee API frame types in embedded and general-purpose applications. This file provides a centralized enumeration for all supported Digi XBee API frame types, facilitating extensibility, maintainability, and integration with Digi XBee API infrastructure and RTOS environments.
