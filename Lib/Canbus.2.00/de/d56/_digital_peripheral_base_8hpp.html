Detailed Description

Base class for digital peripheral management using SPI.

Framework for digital input/output handling, update mechanisms, and SPI-based expansion. Designed for STM32 and MC33879-like peripherals with flexible pin mapping and RTOS task scheduling. Provides robust abstraction for SPI-driven digital I/O, supporting flexible pin mapping, update scheduling, and error handling for MC33879-like expanders. Integrates with STM32 HAL and RTOS for efficient task management.
