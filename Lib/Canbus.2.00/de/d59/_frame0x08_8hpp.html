Detailed Description

Frame0x08: Local AT Command Request for Digi Core.

Provides the Frame0x08 class for querying or setting AT command parameters on the local device. This class encapsulates Digi XBee API frame construction for local AT command requests, enabling immediate application of parameter changes. For queued changes, use Frame 0x09. Querying a parameter returns its current value in the response frame (0x88). The design ensures type safety, extensibility, and efficient integration with Digi XBee API infrastructure and RTOS environments. It supports flexible payloads and transmission options for embedded and general-purpose applications, facilitating reliable communication and host correlation.
