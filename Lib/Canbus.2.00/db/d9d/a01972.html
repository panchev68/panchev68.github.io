Detailed Description

Modem Status handler for Digi XBee modules.

Handles modem status responses and emits events for status changes. Provides event hooks for hardware reset, network join/leave, coordinator actions, security updates, BLE events, firmware updates, and error conditions. Ensures type safety and efficient event dispatch for embedded applications.
