Detailed Description

GPIO abstraction layer for handling GPIO pins and ports.

Provides classes and methods for managing GPIO pins, ports, and external interrupts in the STM32 HAL. Includes convenient aliases and macros for pin access and configuration. This file ensures robust integration with STM32 GPIO hardware, supporting flexible configuration and efficient resource management in embedded applications. The design promotes type safety, maintainability, and extensibility for various STM32 hardware requirements, facilitating seamless adaptation to evolving system needs.
