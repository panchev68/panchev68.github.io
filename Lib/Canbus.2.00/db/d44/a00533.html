class  MessageBuffer

Detailed Description

C++ wrapper for FreeRTOS message buffer functionality. Provides a type-safe, RAII-enabled interface for variable-length message passing between tasks and ISRs. Simplifies message buffer usage, improves reliability, and prevents resource leaks in STM32/FreeRTOS systems. Offers safe creation, destruction, sending, receiving, and status queries for FreeRTOS message buffers. Ensures correct resource management and error handling, reducing complexity and improving maintainability.
