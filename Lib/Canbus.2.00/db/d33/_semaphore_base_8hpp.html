Detailed Description

SemaphoreBase abstraction for FreeRTOS semaphore management.

This file provides a C++17 base class for FreeRTOS semaphore creation, control, and resource management. The SemaphoreBase class encapsulates the lifecycle, ownership, and basic operations for semaphores. It supports move semantics, safe destruction, and provides APIs for taking, giving, and overwriting semaphores in both task and ISR contexts. The class ensures proper resource management and error handling using RAII principles.
