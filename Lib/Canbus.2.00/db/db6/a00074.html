Detailed Description

Beep control for STM32 PWM-based buzzer with timed beep and pause sequences. This file provides the Beep class for controlling a buzzer using STM32 timers and PWM output. The class supports configurable beep duration, pause duration, and repeat count, allowing for flexible beep patterns in embedded applications. The implementation integrates with RTOS tick services for periodic updates and uses HAL timer handles for hardware PWM control. Error handling is performed via assertions to ensure robust operation. All timer channel mappings and state transitions are encapsulated for reliable embedded operation.
