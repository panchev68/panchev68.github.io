Detailed Description

NetworkDiscovery service class.

Provides device discovery for Digi XBee networks, enabling scanning and identification of all joined devices. Manages AT command communication, parses node information, and maintains a list of discovered nodes. Designed for embedded Digi modules, supporting robust network scanning, error handling, and extensibility for custom node processing and diagnostics. Ensures reliable device discovery and event-driven updates for maintainable and type-safe service development.
