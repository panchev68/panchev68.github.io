Detailed Description

template<typename T, size_t QUEUE_LENGTH = 1>
class RTOS::Queue< T, QUEUE_LENGTH >

Queue abstraction for FreeRTOS queue management. The Queue class provides RAII management for FreeRTOS queues, supporting creation, destruction, element send/receive/peek/overwrite, and registry operations. It ensures safe resource handling and move semantics.

Template Parameters
TElement type (must be trivially copyable).
QUEUE_LENGTHNumber of elements in the queue.
