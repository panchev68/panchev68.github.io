Detailed Description

EventGroup abstraction for FreeRTOS event group management.

This file provides a C++17 class for FreeRTOS event group creation, control, and resource management. The EventGroup class encapsulates event group lifecycle, ownership, and basic operations for setting, clearing, waiting, and synchronizing event bits in both task and ISR contexts. It supports move semantics, safe destruction, and provides APIs for bit manipulation and state queries. The class ensures proper resource management and error handling using RAII principles. The implementation is designed for embedded systems and ensures safe usage, bounded operations, and robust error handling.
