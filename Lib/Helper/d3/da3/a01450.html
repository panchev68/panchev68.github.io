Singletone< T > Class Template Reference

Detailed Description

A template class that implements the Singleton design pattern.

The Singletone class ensures that only one instance of the class exists throughout the application. It provides a global point of access to that instance.

Usage:

class MyClass : public Lib::Helper::Singletone<MyClass> {
friend class Lib::Helper::Singletone<MyClass>;
private:
MyClass() {} // Private constructor
public:
void doSomething() {
// Implementation
}
};
// Access the single instance
MyClass::instance().doSomething();
