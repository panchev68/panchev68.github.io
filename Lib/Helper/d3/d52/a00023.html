CityHash.hpp File Reference

Detailed Description

Provides a hash function for 32-bit values.

This file provides the CityHash class, which is used to compute a hash value for a given buffer. The CityHash class uses the CityHash algorithm to compute the hash value of the buffer. The CityHash algorithm is a non-cryptographic hash function that is optimized for speed and produces a 32-bit hash value.

The CityHash class takes a pointer to a source buffer and the length of the buffer as input, and computes the hash value of the buffer using the CityHash algorithm. The hash value is stored in a private member variable, which can be accessed using the conversion operator to uint32_t.
