Processes each element in the container using the provided function.

This method iterates over each element in the container, casts it to the specified type, and applies the given function to it.

Template Parameters
TThe type of elements in the container.
Parameters
funcA function pointer that takes a pointer to an element of type T and a void pointer as arguments.
argAn optional argument that will be passed to the function. Defaults to nullptr.
Note
The function pointer func must not be null. The elements in the container must be castable to type T.
