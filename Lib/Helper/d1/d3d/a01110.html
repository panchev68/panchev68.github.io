Example usage of the TimePeriod class.

This example demonstrates how to use the TimePeriod class to measure the time taken to perform an operation. The class provides methods to start and stop the measurement and retrieve the elapsed time. The example below shows how to measure the time taken to perform a loop.

#include <iostream>
#include "TimePeriod.hpp"
int main() {
Lib::Helper::TimePeriod period;
period.start();
// Perform some operation
for (int i = 0; i < 1000000; i++) {
// Do something
}
uint32_t elapsed;
period.stop(elapsed);
std::cout << "Elapsed time: " << elapsed << " ticks" << std::endl;
return 0;
}
