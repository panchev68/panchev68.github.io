SwapBytes.hpp File Reference

Detailed Description

Provides functions and templates for swapping the byte order of various integral types.

This header file contains a set of functions and template specializations for swapping the byte order of integral types such as int16_t, uint16_t, int32_t, uint32_t, int64_t, and uint64_t. It also provides a function for swapping the bytes of a given source buffer and storing the result in a destination buffer.

The functions are defined within the Lib::Helper namespace.

The following functionalities are provided:

  • Template function swapBytes for swapping the byte order of an integral value.
  • Template specializations for swapping the byte order of specific integral types.
  • Functions for swapping the byte order of values pointed to by pointers.
  • Function for swapping the bytes of a given source buffer and storing the result in a destination buffer.

example:

int16_t value = 0x1234;
int16_t swappedValue = Lib::Helper::swapBytes(value); // swappedValue will be 0x3412
