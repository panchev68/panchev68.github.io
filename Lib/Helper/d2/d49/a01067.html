static int32_t Lib::Helper::StringHelper::findStringInArray ( const char * str,
const char ** array )
inlinestatic

Finds the index of a string in an array of strings.

This function searches for a given string within an array of strings and returns the index of the first occurrence. If the string or the array is null, it returns -2. If the string is not found, it returns -1.

Parameters
strThe string to search for.
arrayThe array of strings to search within. The array must be null-terminated.
Returns
int32_t The index of the string in the array, -2 if the input is invalid, or -1 if the string is not found.
Note
The function uses strncmp to compare the strings, so it is case-sensitive.
The function uses strlen to determine the length of the first string in the array.
const char * array[] = { "one", "two", "three", nullptr };
int32_t index = StringHelper::findStringInArray ("two", array);
// index = 1
