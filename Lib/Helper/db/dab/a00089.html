StringHelper.hpp File Reference

Detailed Description

This file contains the declaration of the StringHelper class, which provides utility functions for string operations.

The StringHelper class includes deprecated functions for checking ASCII characters and digits, and a function to find a string in an array of strings.

Deprecated
The functions isAscii, isAsciiDigit, and isAsciiHexDigit are deprecated. Use the corresponding functions in Lib::Helper::Convert::Ascii instead.
#include "Convert/Ascii.hpp"
#include <algorithm>
#include <string>
#include <vector>
