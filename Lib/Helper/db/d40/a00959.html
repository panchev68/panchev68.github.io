Lib::Helper::Convert::Bcd Class Reference

Detailed Description

Provides utility functions for working with Binary-Coded Decimal (BCD) values.

The Bcd class provides utility functions for working with Binary-Coded Decimal (BCD) values. It provides functions to convert BCD values to hexadecimal values and ASCII characters.
