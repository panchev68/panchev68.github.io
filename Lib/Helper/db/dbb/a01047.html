Lib::Helper::Singleton< T > Class Template Reference

Detailed Description

template<typename T>
class Lib::Helper::Singleton< T >

Provides a template for creating singleton classes.

The Singleton class provides a template for creating singleton classes. It ensures that only one instance of a class is created.

Template Parameters
TThe type of the class to be made a singleton.
