Detailed Description

Defines the MoonPhase class for calculating and representing the phase of the moon based on a given date.

This file contains the MoonPhase class which provides functionality to calculate the moon phase index and name based on a given date. The class uses an internal algorithm to determine the moon phase and provides methods to retrieve the phase name and index.
