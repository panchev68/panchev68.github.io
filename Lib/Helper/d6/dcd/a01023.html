Lib::Helper::DateTime::Date Class Reference

Detailed Description

A class to represent a specific date.

The Date class encapsulates the day, month, and year components of a date value. It provides methods for validation, comparison, and conversion to string representations. It also provides functionality to determine the day of the week and the number of days in a month.
