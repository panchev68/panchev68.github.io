Lib::Helper::DateTime Class Reference

Detailed Description

A class to represent date and time.

The DateTime class encapsulates date and time information and provides various utility functions to manipulate and format date and time.

◆ operator tm()

Lib::Helper::DateTime::operator tm ( ) const
inline

Converts the DateTime object to a tm structure.

This operator allows the DateTime object to be implicitly converted to a standard C library tm structure. The tm structure is populated with the corresponding values from the DateTime object.

Returns
A tm structure representing the same date and time as the DateTime object.
Note
The tm_year field in the tm structure is set to the year minus 100, as the tm_year field represents the number of years since 1900.

static DateTime Lib::Helper::DateTime::unpack ( uint32_t value)
inlinestatic

Unpacks a 32-bit value into a DateTime object.

This function takes a 32-bit packed value and extracts the date and time components, storing them in a DateTime object. The packed value is interpreted as a Packet structure, which contains the individual fields for year, month, day, hours, minutes, and seconds.

Parameters
valueThe 32-bit packed value containing the date and time information.
Returns
A DateTime object with the unpacked date and time components.
