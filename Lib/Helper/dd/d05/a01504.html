BufferStatic< T_SIZE > Class Template Reference

Detailed Description

template<size_t T_SIZE>
class BufferStatic< T_SIZE >

A class that provides a static array of bytes.

The BufferStatic class provides a static array of bytes with a fixed size. It is a template class that allows the size of the buffer to be specified at compile time. The BufferStatic class provides methods for accessing and modifying the data in the buffer.
