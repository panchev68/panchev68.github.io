Lib::Helper::Buffer Class Reference

Detailed Description

A class that provides a dynamic array of bytes.

The Buffer class provides a dynamic array of bytes with various constructors and methods for managing the buffer. It can be used to store binary data and provides methods for accessing and modifying the data. The Buffer class can be used to create a buffer of a specific size, copy data from an existing buffer, or create a buffer from a pointer to data. The Buffer class also provides methods for clearing the buffer and accessing the data.

Note
The Buffer class is a reference-counted object, which means that it can be shared among multiple objects.
