Hex.hpp File Reference

Detailed Description

Provides template classes for converting numeric values to hexadecimal and BCD (Binary-Coded Decimal) representations.

This file contains template specializations for uint8_t, uint16_t, and uint64_t types, offering methods to convert these types to hexadecimal strings and BCD strings. The conversion utilities are provided by the HexHelper class.

Note
The uint64_t specialization supports both local and external storage of the value.
