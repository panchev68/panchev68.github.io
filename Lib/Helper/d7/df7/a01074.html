Example usage of the CityHash class.

This example demonstrates how to use the CityHash class to compute a hash value for a given buffer. The example below shows how to create a CityHash object and retrieve the hash value.

#include <iostream>
#include "CityHash.hpp"
int main() {
const char *data = "example data";
size_t len = strlen(data);
Lib::Helper::CityHash hash(data, len);
uint32_t result = hash;
std::cout << "Hash result: " << result << std::endl;
return 0;
}
