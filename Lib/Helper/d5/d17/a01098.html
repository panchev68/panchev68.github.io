Example usage of the MoonPhase class.

This example demonstrates how to use the MoonPhase class to calculate the phase of the moon based on a given date. The example below shows how to create a MoonPhase object and retrieve the phase name and index.

#include <iostream>
#include "MoonPhase.hpp"
int main() {
Lib::Helper::DateTime::Date date = {2022, 1, 1};
Lib::Helper::MoonPhase moonPhase (date);
std::cout << "Moon phase: " << moonPhase.name () << " (" << moonPhase.index () << ")" << std::endl;
return 0;
}
