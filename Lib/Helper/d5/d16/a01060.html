◆ getOffset()

int TimeZone::getOffset ( ) const
inline

Gets the offset from UTC in minutes.

This function returns the offset from UTC in minutes. The offset is the difference between the local time and UTC time, expressed in minutes. Positive values indicate time zones east of UTC, and negative values indicate time zones west of UTC.

Returns
The offset from UTC in minutes.
