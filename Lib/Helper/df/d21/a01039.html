Lib::Helper::MoonPhase Class Reference

Detailed Description

A class to calculate and represent the phase of the moon based on a given date.

The MoonPhase class provides functionality to calculate the phase of the moon based on a given date. It uses an internal algorithm to determine the moon phase index and name and provides methods to retrieve the phase name and index.
