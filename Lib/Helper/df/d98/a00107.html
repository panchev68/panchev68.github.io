template<>
int32_t Lib::Helper::swapBytes< int32_t > ( int32_t value)
inline

Specialization of the swapBytes template function for int32_t type.

This function swaps the bytes of a 32-bit integer (int32_t).

Parameters
valueThe 32-bit integer value whose bytes are to be swapped.
Returns
The 32-bit integer with its bytes swapped.

The function performs the following operations:

  • Shifts the input value left by 8 bits and masks it with 0xFF00FF00.
  • Shifts the input value right by 8 bits and masks it with 0xFF00FF.
  • Combines the results of the above two operations using bitwise OR.
  • Shifts the combined result left by 16 bits and combines it with the result of shifting the combined result right by 16 bits using bitwise OR.

Specialization of the swapBytes template function for int64_t type.

This function swaps the bytes of a 64-bit integer (int64_t).

Parameters
valueThe 64-bit integer value whose bytes are to be swapped.
Returns
The 64-bit integer with its bytes swapped.

The function performs the following operations:

  • Shifts the input value left by 8 bits and masks it with 0xFF00FF00FF00FF00.
  • Shifts the input value right by 8 bits and masks it with 0x00FF00FF00FF00FF.
  • Combines the results of the above two operations using bitwise OR.
  • Shifts the combined result left by 16 bits and combines it with the result of shifting the combined result right by 16 bits using bitwise OR.
  • Finally, it shifts the combined result left by 32 bits and combines it with the result of shifting the combined result right by 32 bits using bitwise OR.
