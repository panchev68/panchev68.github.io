Lib::Helper::TimePeriod Class Reference

Detailed Description

A class to measure time periods.

The TimePeriod class provides functionality to measure time periods using the FreeRTOS TickCounter. It includes methods to start and stop the measurement and retrieve the elapsed time.

Example usage:

TimePeriod period;
period.start();
// Perform some operation
uint32_t elapsed;
period.stop(elapsed);
