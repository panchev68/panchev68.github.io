Lib::Helper::Convert::Ascii Class Reference

Detailed Description

Provides utility functions for working with ASCII characters.

The Ascii class provides utility functions for working with ASCII characters. It provides functions to determine if a character is an ASCII character, a digit, or a hexadecimal digit. It also provides a function to convert a hexadecimal character to a numeric value.

Determines if a character is an ASCII character.

This function determines if the specified character is an ASCII character. An ASCII character is defined as a character with a value greater than or equal to 0x20 (space) and less than 0x7A (z).
