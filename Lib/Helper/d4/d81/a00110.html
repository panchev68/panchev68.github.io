Lib::Helper Namespace Reference

Detailed Description

The namespace encapsulates the Base class and its type aliases to avoid naming conflicts.

Template Parameters
TThe type of elements managed by the Base class.

The namespace containing helper utilities for the library.

The namespace containing the TimePeriod class.

Note
The TimePeriod class uses the FreeRTOS TickCounter to measure time periods.
