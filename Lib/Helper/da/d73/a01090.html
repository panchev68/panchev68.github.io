Example usage of the FindPeaks class.

This example demonstrates how to use the FindPeaks class to detect rising and falling edges in a sequence of data. The example below shows how to create a FindPeaks object, enter data values, and retrieve the positions of the rising and falling edges.

#include <iostream>
#include "FindPeaks.hpp"
int main() {
Lib::Helper::FindPeaks findPeaks;
findPeaks.start();
int32_t data[] = {1, 3, 7, 8, 7, 5, 4, 2, 1};
for (int32_t value : data) {
findPeaks.enterData(value);
}
std::cout << "Rising edge position: " << findPeaks.getRisingEdgePosition() << std::endl;
std::cout << "Falling edge position: " << findPeaks.getFallingEdgePosition() << std::endl;
return 0;
}
