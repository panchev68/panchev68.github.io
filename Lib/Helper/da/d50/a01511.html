BufferHandle Class Reference

Detailed Description

A class that provides a handle to a buffer of bytes.

The BufferHandle class provides a handle to a buffer of bytes. It allows access to the data buffer and its size. The BufferHandle class can be used to pass buffers between functions without copying the data.
