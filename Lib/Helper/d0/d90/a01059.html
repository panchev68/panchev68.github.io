Lib::Helper::RingBuffer< T, T_SIZE > Class Template Reference

Detailed Description

template<typename T, size_t T_SIZE>
class Lib::Helper::RingBuffer< T, T_SIZE >

A class to represent a ring buffer.

The RingBuffer class provides a ring buffer implementation with push and pop operations. It includes methods to push and pop elements from the buffer, and to clear the buffer.

Template Parameters
TThe type of elements stored in the buffer.
T_SIZEThe size of the buffer.
