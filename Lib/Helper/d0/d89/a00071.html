JaggedArray.hpp File Reference

Detailed Description

This file contains the definition of a templated Base class for managing a jagged array structure.

The Base class provides functionalities to add, remove, and access elements in a vector of pointers. It also includes utility functions to clear the vector and retrieve its size.

The Row and Array templates are type aliases for the Base class, representing a row and an array of rows respectively.
