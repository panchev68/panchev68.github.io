Example usage of the crc16 class

This example demonstrates how to use the crc16 class to calculate the CRC-16 checksum for a given data. The data can be provided as a pointer to raw data or as a std::vector<uint8_t>. The calculated checksum can be obtained by casting an instance of the class to uint16_t. The example below shows how to calculate the CRC-16 checksum for a vector of bytes.

#include <iostream>
#include "crc16.hpp"
int main() {
// Example data
std::vector<uint8_t> data = {0x01, 0x02, 0x03};
// Calculate CRC-16 checksum
crc16 checksum(data);
uint16_t result = checksum;
// Print the result
std::cout << "CRC-16 checksum: " << std::hex << result << std::endl;
return 0;
}
