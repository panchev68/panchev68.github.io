Bootstrap.hpp File Reference

Detailed Description

The Bootstrap class is responsible for managing the bootstrapping process of EtherCAT devices. It provides functionalities to handle file operations, buffer management, and state transitions during the boot process.

The class includes nested types such as BootState and Buffer, and provides various member functions to initialize, update, and manage the boot process. It also allows binding of custom functions for file writing, validation, and other operations.

The file also includes necessary includes for FreeRTOS and EtherCAT FOE (File Over EtherCAT) functionalities.

Note
This file is part of the EtherCAT library and is intended to be used with the IAR Embedded Workbench for ARM (EWARM).
