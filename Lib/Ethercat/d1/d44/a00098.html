SlaveBase.hpp File Reference

Detailed Description

The SlaveBase class provides an interface for managing EtherCAT slave devices. It includes methods for starting and stopping the device, handling object updates, and managing input and output callbacks. The class also supports binding various functions for file operations and state changes.
