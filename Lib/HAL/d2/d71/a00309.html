Lib::HAL::I2Cx Class Reference

Detailed Description

I2C Base port class.

I2C port base class

This is a base class that is used by all I2C master ports.
By default, the I2C driver works in interrupt mode. If it is necessary to work in DMA mode, then only those functions that use DMA must be redefined. However, if it is used in block mode then all waitForCompleted functions should be redefined and they should return true. For this class to work USE_HAL_I2C_REGISTER_CALLBACKS = 1. It is located in the stm32f4xx_hal_conf.h file.
