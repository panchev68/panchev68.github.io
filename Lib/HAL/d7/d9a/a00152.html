RTC.hpp File Reference

Detailed Description

This file contains the definition of the RTCBase class, which provides an interface for interacting with the Real-Time Clock (RTC) hardware.

The RTCBase class allows for reading and writing date and time information, as well as managing backup registers. It uses FreeRTOS for mutex handling to ensure thread safety.

Note
This file is part of the HAL (Hardware Abstraction Layer) and requires the HAL_RTC_MODULE_ENABLED macro to be defined.
