DeviceManager.hpp File Reference

Detailed Description

This file contains the definition of the DeviceManager class for CAN devices.

The DeviceManager class is responsible for managing CAN devices and their callbacks. It inherits from DeviceManagerBase and provides methods to add and remove devices, as well as register and unregister their callbacks.

Note
This file requires the HAL CAN module to be enabled and the USE_HAL_CAN_REGISTER_CALLBACKS macro to be set to 1.
