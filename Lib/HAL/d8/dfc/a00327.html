Lib::HAL::OTP Class Reference
OTP - One-Time-Programmable Memory

Detailed Description

OTP Flash Memory class.

512 OTP (one-time programmable) bytes for user data OTP area 0x1FFF'7800 - 0x1FFF'7A0F 528 bytes The OTP area contains 16 additional bytes used to lock the corresponding OTP data block.
