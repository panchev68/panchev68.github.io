Lib::HAL::RTCBase Class Reference

Detailed Description

A class representing the Real-Time Clock (RTC) hardware.

The RTCBase class provides an interface for reading and writing date and time information, as well as managing backup registers. It uses FreeRTOS for mutex handling to ensure thread safety.
