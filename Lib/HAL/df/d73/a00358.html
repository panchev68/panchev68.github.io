Handle.hpp File Reference

Detailed Description

This file contains the definition of the Handle class for handling TIM peripherals.

The Handle class is part of the Lib::HAL::Timer namespace and is used to manage TIM (Timer) peripherals in the HAL (Hardware Abstraction Layer). It requires USE_HAL_TIM_REGISTER_CALLBACKS to be set to 1.

Note
Ensure that USE_HAL_TIM_REGISTER_CALLBACKS is set to 1 before including this file.
