Lib::HAL::SPIx Class Reference
Communication interface » SPI - Serial Peripheral Interface

Detailed Description

A class for handling SPI communication.

This class provides an interface for initializing, deinitializing, sending, receiving, and aborting SPI communication. It also handles SPI events and errors.

Template Parameters
SPI_HandleTypeDefThe type of the SPI handle.
Note
For this class to work USE_HAL_SPI_REGISTER_CALLBACKS = 1. It is located in the stm32f4xx_hal_conf.h file.
