PortBase.hpp File Reference

Detailed Description

This file contains the definition of the PortBase class for CAN communication.

The PortBase class provides an interface for CAN communication, including methods for opening and closing the port, handling events, setting filters, and sending messages. It utilizes FreeRTOS for task and semaphore management.

Note
This file should only be included if HAL_CAN_MODULE_ENABLED is defined.
