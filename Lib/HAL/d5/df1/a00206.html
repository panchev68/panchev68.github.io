◆ DriverDataDirection

enum Lib::HAL::DriverDataDirection

Enumeration of driver data directions.

This enumeration defines the data directions for drivers in the HAL library. The data direction can be set to receive (RX), transmit (TX), or both (RX_TX).

Enumerator
RX 

Receive data direction.

TX 

Transmit data direction.

RX_TX 

Receive and transmit data direction.

Enumeration of driver modes.

This enumeration defines the driver modes for HAL drivers. The driver mode can be set to BLOCK, IT, or DMA.
