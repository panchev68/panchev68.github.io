Detailed Description

SPI port base class.

This file contains the declaration of the SPIx class, which is a base class for all SPI ports. The SPI driver works in interrupt mode by default. For DMA mode, only functions using DMA need to be redefined. In block mode, all waitForCompleted functions should be redefined to return true.
