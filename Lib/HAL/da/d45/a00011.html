GPIO.hpp File Reference

Detailed Description

This file contains the declaration of the GPIO class and its nested classes for handling GPIO operations.

The GPIO class provides an interface for configuring and controlling General Purpose Input/Output (GPIO) pins. It includes nested classes for handling pin modes, input/output operations, and external interrupts (EXTI).

Note
This file requires the HAL_GPIO_MODULE_ENABLED macro to be defined.
