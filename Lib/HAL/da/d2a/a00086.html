PortTransparentBase.hpp File Reference

Detailed Description

This file contains the definition of the PortTransparentBase class, which is a base class for ADC port handling with transparent operations.

The PortTransparentBase class provides a base implementation for handling ADC ports with transparent operations. It includes methods for opening, closing, starting, stopping, and reading from the ADC port. The class also manages the registration and unregistration of ADC callbacks using the HAL library.

Note
This file requires the HAL_ADC_MODULE_ENABLED macro to be defined and USE_HAL_ADC_REGISTER_CALLBACKS to be set to 1.
