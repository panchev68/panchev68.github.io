Lib::HAL::GPIO::EXTIx Class Reference

Detailed Description

A class representing an external interrupt on a GPIO pin.

The EXTIx class provides methods for configuring and handling external interrupts on GPIO pins.

#include <GPIO.hpp>

bool Lib::HAL::GPIO::EXTIx::bind ( const std::function< void(const EXTIx *)> & callback)
inline

Binds a callback function to the GPIO object.

This function registers the GPIO object with the DeviceManager and assigns the provided callback function to be called when an external interrupt occurs.

Parameters
callbackA std::function object representing the callback function to be bound. The callback function should take a pointer to an EXTIx object as its parameter.
Returns
true if the GPIO object was successfully registered with the DeviceManager and the callback function was assigned; false otherwise.
