Lib::HAL::SPIBase Class Reference

Detailed Description

SPI port base class

This is a base class that is used by all SPI ports.
By default, the SPI driver works in interrupt mode. If it is necessary to work in DMA mode, then only those functions that use DMA must be redefined. However, if it is used in block mode then all waitForCompleted functions should be redefined and they should return true. For this class to work USE_HAL_SPI_REGISTER_CALLBACKS = 1. It is located in the stm32f4xx_hal_conf.h file.
