Lib::HAL::CAN::TxHeader Class Reference

Detailed Description

A class for CAN transmission headers.

The TxHeader class is a wrapper around the CAN_TxHeaderTypeDef structure, providing constructors and methods to initialize and manipulate CAN transmission headers.
