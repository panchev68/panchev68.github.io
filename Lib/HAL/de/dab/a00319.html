Lib::HAL::TIM::Timer< Mode::ENCODER > Class Reference

Detailed Description

Timer class template specialization for the ENCODER mode.

This class provides methods to open, close, start, and stop the timer in ENCODER mode.

Note
This class is a template specialization of the Timer class for the ENCODER mode.
This class is derived from the Base class.
