Lib::HAL::Adc::Interface Class Reference

Detailed Description

A class representing the ADC interface.

The Interface class provides an abstraction for handling ADC components using the ADC_HandleTypeDef handle. It inherits from the Handle class and provides a protected constructor and a virtual destructor.
