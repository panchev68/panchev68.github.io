Lib::HAL::UARTBase Class Reference

Detailed Description

A class representing UART communication using the HAL library.

The UARTBase class provides an interface for initializing, deinitializing, sending, and receiving data over UART. It also handles various UART events and errors using FreeRTOS stream buffers and event groups.
