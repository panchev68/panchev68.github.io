Lib::HAL::CAN::Filter Class Reference
CAN - Controller Area Network

Detailed Description

A class representing a CAN filter configuration.

The Filter class is a wrapper around the CAN_FilterTypeDef structure, providing additional functionality for setting up CAN filters.

Note
This class is part of the CAN module in the HAL library.
