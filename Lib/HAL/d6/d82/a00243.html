Detailed Description

template<class T_DEVICE, typename T_HANDLE>
class Lib::HAL::DeviceManagerBase< T_DEVICE, T_HANDLE >

Base class for managing devices.

This class provides basic functionalities to add, remove, and find devices. It uses an unordered map to store devices and their handles.

Template Parameters
T_DEVICEThe type of the device.
T_HANDLEThe type of the handle.
