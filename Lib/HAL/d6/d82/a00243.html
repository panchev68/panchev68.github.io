Detailed Description

template<class T_DEVICE, typename T_HANDLE>
class Lib::HAL::DeviceManagerBase< T_DEVICE, T_HANDLE >

Device Manager Base class template.

Template Parameters
T_DEVICEThe type of the device objects managed by this class.
T_HANDLEThe type of the handle used to identify the devices.

This class template provides a base class for managing devices. It inherits from the Lib::Helper::Container class template and provides methods for adding, removing, and retrieving devices.
