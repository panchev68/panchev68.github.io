Lib::HAL::GPIO Class Reference

Detailed Description

A class representing General Purpose Input/Output (GPIO) functionality.

The GPIO class provides an interface for configuring and controlling GPIO pins. It contains nested classes for handling specific pin configurations and operations.

Nested Classes:

  • PinHandle: A class for handling individual GPIO pins.
  • PinInput: A class for configuring GPIO pins as inputs.
  • PinOutput: A class for configuring GPIO pins as outputs.
  • EXTIx: A class for handling external interrupts on GPIO pins.
  • Port: A class for handling GPIO ports.

Private Nested Class:

  • PinMode: A class for configuring the mode of GPIO pins.
