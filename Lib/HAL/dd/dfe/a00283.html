Lib::HAL::I2Cx Class Reference

Detailed Description

A class representing I2C communication using the HAL library.

The I2Cx class provides an interface for initializing, deinitializing, sending, and receiving data over I2C. It also handles various I2C events and errors using FreeRTOS event groups.

#include <I2C.hpp>

Receives data from an I2C device.

This function reads data from a specified I2C device memory address into a destination buffer. It uses an interrupt-based method for reading data.

Parameters

addressSizeThe size of the memory address (in bytes).
addressThe memory address to read from.
dstPointer to the destination buffer where the read data will be stored.
lenThe length of data to read (in bytes).
timeoutThe timeout duration (in milliseconds). Default is 0.
Returns
true if the read operation was successful, false otherwise.
Note
This function asserts that the destination buffer is not null and the length is greater than 0. It also asserts that the HAL_I2C_Mem_Read_IT function returns HAL_OK.
