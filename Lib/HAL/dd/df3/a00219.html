Lib::HAL::Adc::Port< IT > Class Reference

Detailed Description

ADC Port class template specialization for DriverMode::IT mode.

Template Parameters
DriverMode::IT

This class provides methods to open, close, start, stop, and read from the ADC port.

Note
This class is a template specialization of the Port class for the DriverMode::IT mode.
This class is derived from the PortTransparentBase class.

#include <Port-IT.hpp>
