Lib::HAL::TIM::Timer< Mode::ENCODER > Class Reference

Detailed Description

Timer class template specialization for ENCODER mode.

This class provides methods to open, close, start, and stop the timer in ENCODER mode. It also allows handling timer events through a callback mechanism.
