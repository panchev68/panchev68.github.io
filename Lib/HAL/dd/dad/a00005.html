BKPSRAM.hpp File Reference

Detailed Description

Header file for the BKPSRAM class, which provides an interface to the Backup SRAM (BKPSRAM) functionality.

This file contains the declaration of the BKPSRAM class and its nested Domain class. The BKPSRAM class provides methods to enable, disable, and manage the Backup SRAM. The Domain class is a singleton that manages the power and clock settings for the BKPSRAM.

Note
This file should only be included if the HAL_PWR_MODULE_ENABLED macro is defined.
