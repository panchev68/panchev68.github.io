Lib::HAL::UARTx Class Reference

Detailed Description

A class representing UART communication using the HAL library.

The UARTx class provides an interface for initializing, deinitializing, sending, and receiving data over UART. It also handles various UART events and errors using FreeRTOS stream buffers and event groups.
