Detailed Description

template<DriverMode>
class Lib::HAL::Adc::Port< DriverMode >

A template class representing the ADC port.

The Port class is a template class representing the ADC port. It is parameterized by the DriverMode, which can be set to BLOCK, IT, or DMA. The class provides methods to open, close, start, stop, and read from the ADC port.

Template Parameters
DriverModeThe driver mode for the ADC port.
