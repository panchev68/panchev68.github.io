Lib::HAL::GPIO::EXTIx::DeviceManager Class Reference

Detailed Description

Manages devices of type EXTIx.

This class is a singleton that inherits from DeviceManagerBase and provides functionality to add, remove, and retrieve devices. It also handles events for the devices.

#include <GPIO.hpp>
