Lib::HAL::Adc::PortBase Class Referenceabstract

Detailed Description

A base class for handling ADC ports.

The PortBase class provides a common interface for opening, starting, stopping, and closing ADC ports, as well as retrieving raw ADC values. It inherits from the Interface class and is protectedly derived from the Channel class.

#include <PortBase.hpp>
