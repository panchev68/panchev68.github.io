Lib::HAL::CAN::TxMessage Class Reference

Detailed Description

Represents a CAN transmission message with a header and data payload.

The TxMessage class extends the TxHeader class to include functionality for managing the data payload of a CAN transmission message.

#include <TxMessage.hpp>
