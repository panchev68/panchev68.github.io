Lib::HAL::ADCPort::Port< DMA > Class Reference

Detailed Description

ADC Port class template specialization for DriverMode::DMA mode.

Template Parameters
DriverMode::DMA

This class provides methods to open, close, start, stop, and read from the ADC port.

Note
This class is a template specialization of the Port class for the DriverMode::DMA mode.
This class is derived from the TransparentPortBase class.

#include <DMAMode.hpp>
