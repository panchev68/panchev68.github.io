IWDG.hpp File Reference

Detailed Description

This file contains the declaration of the Watchdog class for handling the Independent Watchdog (IWDG) peripheral.

The Watchdog class provides methods to setup and refresh the IWDG peripheral. It ensures that the system can recover from unexpected software malfunctions by resetting the system if the watchdog is not refreshed within a certain time period.

Note
This file should only be included if HAL_IWDG_MODULE_ENABLED is defined.
