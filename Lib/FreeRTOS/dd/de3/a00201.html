Lib::FreeRTOS::TaskHelper Class Reference

Detailed Description

A class providing helper functions for FreeRTOS tasks.

This class provides a set of static functions that allow access to FreeRTOS tasks. The functions include resuming and suspending tasks, setting and getting task priorities, and handling task notifications.

Examples
TaskHelper.

#include <TaskHelper.hpp>

◆ beginCriticalSection()

static void Lib::FreeRTOS::TaskHelper::beginCriticalSection ( )
inlinestatic

Enters a critical section by disabling interrupts.

This function calls the FreeRTOS macro taskENTER_CRITICAL() to disable interrupts and enter a critical section. Critical sections are used to protect code that must not be interrupted, ensuring atomic access to shared resources.

Note
It is important to pair this function with a corresponding call to endCriticalSection() to re-enable interrupts and exit the critical section.

Ends a critical section.

This function exits a critical section that was previously entered. It should be used to mark the end of a critical section where interrupts were disabled to protect shared resources.

Note
