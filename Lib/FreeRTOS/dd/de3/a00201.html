RTOS::SemaphoreCounting Class Reference

Detailed Description

A counting semaphore implementation for FreeRTOS.

This class provides a counting semaphore mechanism that can be used to manage access to a finite number of resources in a multitasking system.

#include <SemaphoreCounting.hpp>
