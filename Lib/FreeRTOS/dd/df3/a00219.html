Lib::FreeRTOS::TickService Class Reference

Detailed Description

A class that manages TickService events in the FreeRTOS environment.

The TickService class is responsible for managing TickService events in the FreeRTOS environment. It provides a singleton interface for registering clients that handle TickService events. The TickService class uses a mutex to ensure thread-safe access to the list of registered clients. The clients are notified of TickService events through the handleTickServiceEvent method defined in the ClientBase abstract class.

Note
This class is designed to work with FreeRTOS.

#include <TickService.hpp>

void Lib::FreeRTOS::TickService::start ( )
inline

Starts the TickService timer.

This function initializes and starts the TickService timer by calling the Timer::run method. It creates a task named "TickService" that runs the provided lambda function. The lambda function takes a mutex, iterates over all registered clients, and calls their handleTickServiceEvent method. After processing all clients, the mutex is destroyed.

The timer is set to run with a priority of 1 and is configured to auto-reload.
