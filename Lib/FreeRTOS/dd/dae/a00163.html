Lib::FreeRTOS::Karnel Class Reference

Detailed Description

A class providing access to FreeRTOS kernel functions.

This class provides a set of static functions that allow access to the FreeRTOS kernel. The functions include starting and stopping the scheduler, suspending and resuming tasks, and managing the scheduler state.

Suspends all tasks in the FreeRTOS scheduler.

This function calls the FreeRTOS API function vTaskSuspendAll() to suspend all tasks. Suspending all tasks can be useful when you need to perform critical operations that should not be interrupted by the scheduler.
