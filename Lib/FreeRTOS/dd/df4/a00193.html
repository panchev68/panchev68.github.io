Lib::FreeRTOS::Task Class Referencefinal

Detailed Description

A C++ wrapper for FreeRTOS tasks.

This class provides a C++ interface for creating and managing FreeRTOS tasks. It includes methods for creating tasks, suspending and resuming tasks, setting task priorities, and handling task notifications.

Examples
Task, TaskHelper, and TickCounter.

#include <Task.hpp>

enum class Lib::FreeRTOS::Task::NotifyAction
strong

Enumeration of possible notification actions.

This enumeration defines the possible actions that can be taken when sending a notification to a task. The actions include setting bits, incrementing a value, and setting a value with or without overwriting.
