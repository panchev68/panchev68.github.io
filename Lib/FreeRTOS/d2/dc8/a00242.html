RTOS::Timer Class Reference

Detailed Description

Wrapper class for FreeRTOS timers.

This class provides an object-oriented interface for managing FreeRTOS timers. It supports creating, starting, stopping, and configuring timers, as well as handling timer callbacks using std::function.

#include <Timer.hpp>
