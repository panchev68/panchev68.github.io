Detailed Description

template<size_t StackSize = configMINIMAL_STACK_SIZE, int Priority = DEFAULT_TASK_PRIORITY>
struct Lib::FreeRTOS::Task< StackSize, Priority >::Config

Configuration struct for task creation.

This struct defines the configuration parameters for creating a task. It includes the task name, function to run, stack depth, and priority.
