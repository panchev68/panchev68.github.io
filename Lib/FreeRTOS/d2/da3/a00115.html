const std::initializer_list< Member > & list )

Queue sets provide a mechanism to allow an RTOS task to block (pend) on a read operation from multiple RTOS queues or semaphores simultaneously.

Parameters
len- Queue sets store events that occur on the queues and semaphores contained in the set.
list- list with members
Returns
true if the QueueSet was created successfully
