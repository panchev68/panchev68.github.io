Task.hpp File Reference

Detailed Description

This file contains the definition of the Task class for FreeRTOS.

The Task class provides a C++ wrapper around FreeRTOS tasks, allowing for easier task creation, management, and notification handling using modern C++ features such as std::function and std::bitset.

Note
This file is part of the FreeRTOS library and depends on FreeRTOS headers.

The Task class inherits from TaskHelper and provides functionalities to:

  • Create and run tasks with specified configurations.
  • Suspend, resume, and delete tasks.
  • Handle task notifications with various actions.

The class also includes a nested Config struct for task configuration and an enum class NotifyAction for specifying notification actions.
