Detailed Description

Header file for the MemoryManager class.

This file contains the declaration of the MemoryManager class, which provides memory management functions for FreeRTOS. It includes functions for allocating and freeing memory, as well as retrieving the amount of free heap memory available. The class also includes diagnostic features for tracking memory allocations and deallocations when enabled.
