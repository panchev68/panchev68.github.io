Detailed Description

Header file for the Mutex class in FreeRTOS.

This file contains the declaration of the Mutex class, which provides a C++ interface for creating and managing mutexes in FreeRTOS. Mutexes are used to protect shared resources from concurrent access by multiple tasks.
