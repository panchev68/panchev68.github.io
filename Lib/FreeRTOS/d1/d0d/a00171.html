Lib::FreeRTOS::Mutex::Guard Class Reference

Detailed Description

A RAII-style guard for managing Mutex locking and unlocking.

The Guard class ensures that a Mutex is locked upon creation and unlocked upon destruction, providing exception-safe locking and unlocking.

Note
This class is non-copyable and non-movable.

Destructor for the Guard class.

This destructor unlocks the associated mutex when the Guard object goes out of scope. It ensures that the mutex is properly released, preventing deadlocks and ensuring that other threads can acquire the mutex.

Note
This destructor is marked noexcept, indicating that it does not throw any exceptions.
