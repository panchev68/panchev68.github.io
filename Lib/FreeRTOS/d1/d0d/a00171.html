Lib::FreeRTOS::TickCounter Class Referencefinal

Detailed Description

A class providing a C++ interface for FreeRTOS tick counter.

This class provides a C++ interface for accessing the FreeRTOS tick counter. The tick counter is a 32-bit value that increments every tick of the FreeRTOS scheduler.

Gets the current tick count from an ISR context.

This function is used to get the current tick count when called from an Interrupt Service Routine (ISR). It is a wrapper around the FreeRTOS function xTaskGetTickCountFromISR().
