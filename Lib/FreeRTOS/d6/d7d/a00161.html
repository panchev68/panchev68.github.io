Lib::FreeRTOS::MessageBuffer Class Reference

Detailed Description

A class providing a C++ wrapper for FreeRTOS message buffers.

This class provides a C++ interface for creating and managing FreeRTOS message buffers. It includes methods for sending and receiving messages, as well as checking the status of the message buffer.

Examples
MemoryManager, and MessageBuffer.

#include <MessageBuffer.hpp>

Receives a message from the message buffer.

This function attempts to receive a message from the message buffer. It will block for the specified number of ticks if no message is immediately available.

Parameters
dstPointer to the buffer where the received message will be stored.
lenThe length of the buffer pointed to by dst.
ticksToWaitThe maximum amount of time (in ticks) to wait for a message to become available.
Returns
The number of bytes received, or 0 if no message was received within the specified time.
Note
This function asserts that the message buffer handle is not null, the destination buffer is not null, and the length is non-zero.

size_t Lib::FreeRTOS::MessageBuffer::send ( const void * src,
size_t len,
uint32_t ticksToWait )
inline

Sends a message to the message buffer.

This function sends a message to the message buffer. The message is copied from the memory pointed to by src and has a length of len bytes. The function will block for up to ticksToWait ticks if there is not enough space in the buffer to hold the message.

Parameters
srcPointer to the message to be sent.
lenLength of the message in bytes.
ticksToWaitMaximum number of ticks to wait for space to become available in the buffer.
Returns
The number of bytes actually sent to the buffer, which will be equal to len if the message was sent successfully, or 0 if the message could not be sent within the specified wait time.
Note
This function asserts that the message buffer handle is not null, that the source pointer is valid, and that the message length is greater than 0.
Examples
MemoryManager, and MessageBuffer.
