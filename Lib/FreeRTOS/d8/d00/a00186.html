RTOS::QueueSet Class Reference

Detailed Description

A class to manage and interact with queue sets in FreeRTOS.

The QueueSet class provides an interface to create, manage, and interact with queue sets in FreeRTOS. It allows for the creation of queue sets, adding and removing members, and selecting members from the queue set.

Note
This class uses assertions to ensure that operations are performed on valid queue set handles.
