Lib::FreeRTOS::Mutex Class Reference

Detailed Description

A class that represents a mutex, inheriting from SemaphoreBase.

The Mutex class provides functionality for creating, taking, giving, and releasing a mutex. It includes methods for handling mutex operations both in normal and Interrupt Service Routine (ISR) contexts.

Note
This class is designed to work with FreeRTOS.
See also
SemaphoreBase
