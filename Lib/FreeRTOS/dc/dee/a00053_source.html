144 * @brief Enters a critical section by disabling interrupts.
145 *
146 * This function calls the FreeRTOS API function `taskENTER_CRITICAL()`
147 * to enter a critical section, ensuring that the code within the critical
148 * section is not interrupted by any other tasks or interrupts.
149 *
150 * Usage:
