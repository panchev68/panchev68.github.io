Detailed Description

A base class for semaphore implementations.

Note
This class is intended to be inherited by specific semaphore types.

#include <SemaphoreBase.hpp>

Destructor for the SemaphoreBase class.

This destructor ensures that the semaphore handle is destroyed before the object is destroyed. It asserts that the handle is nullptr, indicating that the semaphore has been properly destroyed.

Note
If the handle is not nullptr, an assertion failure will occur with the message "Must be destroyed before destroy the object".

inlineprotectedvirtual

Gives the semaphore from an ISR context.

This function attempts to release the semaphore from within an Interrupt Service Routine (ISR). If the semaphore is successfully given, it will also check if a higher priority task was woken by the operation and yield to it if necessary.

Returns
true if the semaphore was successfully given, false otherwise.
Note
This function should only be called from an ISR context.
The semaphore must be created before calling this function.

inlineprotectedvirtual

Overwrites the semaphore from an ISR (Interrupt Service Routine).

This function attempts to take and then give the semaphore from within an ISR context. If the semaphore is successfully given, the function returns true. Otherwise, it returns false. The function also ensures that a context switch is requested if a higher priority task was woken.

Returns
true if the semaphore was successfully given from the ISR, false otherwise.
Note
This function asserts if the semaphore is not created.
