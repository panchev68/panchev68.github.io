SemaphoreCounting.hpp File Reference

Description

Counting semaphore implementation for FreeRTOS.

Defines the SemaphoreCounting class, a C++ wrapper for FreeRTOS counting semaphores. Supports creation, destruction, take/give operations, ISR support, and querying the current count. Enables resource management where multiple tokens are required.
