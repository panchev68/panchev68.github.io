using Lib::FreeRTOS::SemaphoreBase::Handle = SemaphoreHandle_t
protected

Handle to a semaphore.

This handle is used to manage and interact with a semaphore in FreeRTOS. It is initialized to nullptr and should be assigned a valid semaphore handle before use.

◆ ~SemaphoreBase()

virtual Lib::FreeRTOS::SemaphoreBase::~SemaphoreBase ( )
inlineprotectedvirtual

Destructor for the SemaphoreBase class.

This destructor ensures that the semaphore handle is released before the object is destroyed. It asserts that the handle is nullptr, indicating that the semaphore has been properly released.

Note
If the handle is not nullptr, an assertion failure will occur with the message "Must be released before destroy the object".

inlineprotectedvirtual

Overwrites the semaphore from an ISR (Interrupt Service Routine).

This function attempts to take and then give the semaphore from within an ISR context. If the semaphore is successfully given, the function returns true. Otherwise, it returns false. The function also ensures that a context switch is requested if a higher priority task was woken.

Returns
true if the semaphore was successfully given from the ISR, false otherwise.
Note
This function asserts if the semaphore is not created.
