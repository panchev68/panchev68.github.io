Lib::FreeRTOS::Task< StackSize, Priority > Class Template Reference

Detailed Description

template<size_t StackSize = configMINIMAL_STACK_SIZE, int Priority = DEFAULT_TASK_PRIORITY>
class Lib::FreeRTOS::Task< StackSize, Priority >

A C++ wrapper for FreeRTOS threads.

A C++ wrapper for FreeRTOS tasks.

This class provides a C++ interface for creating and managing FreeRTOS threads.

Template Parameters
StackSizeThe stack size of the thread. Default is configMINIMAL_STACK_SIZE.
PriorityThe priority of the thread. Default is 2.

This class provides a C++ interface for creating and managing FreeRTOS tasks. It includes methods for creating tasks, suspending and resuming tasks, setting task priorities, and handling task notifications.

Examples
Task, and TaskHelper.

#include <~Task.hpp>

Delays the execution of the current task for a specified number of milliseconds.

This function uses the FreeRTOS vTaskDelay function to suspend the execution of the calling task for the specified duration. The delay period is specified in milliseconds and is converted to ticks based on the portTICK_PERIOD_MS constant.
