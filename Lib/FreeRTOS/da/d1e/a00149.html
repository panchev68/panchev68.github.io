Lib::FreeRTOS::EventGroup Class Reference

Detailed Description

A class representing an event group in FreeRTOS.

This class provides an interface to create and manage event groups in FreeRTOS. Event groups are used to manage sets of events, where each event is represented by a bit in a 32-bit value.

Note
This class initializes the event group handle to nullptr.

inline

Waits for one or more bits to be set within an event group.

This function waits for the specified bits to be set within the event group. It can be configured to clear the bits on exit, wait for all bits to be set, and specify a timeout period.

Parameters
bitsToWaitForThe bits within the event group to wait for. Default is 0.
clearOnExitIf true, the bits will be cleared before the function returns. Default is false.
waitForAllBitsIf true, the function will wait for all bits to be set. If false, it will wait for any bit to be set. Default is false.
ticksToWaitThe maximum amount of time to wait for the bits to be set, specified in tick periods. Default is portMAX_DELAY.
Returns
The value of the event group bits at the time the call returns.
Note
This function asserts that the event group has been created before waiting for bits.
