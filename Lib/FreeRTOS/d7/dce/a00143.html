Lib::FreeRTOS::Semaphore Class Reference

Detailed Description

A class representing a binary semaphore in FreeRTOS.

This class provides a C++ interface for creating and managing binary semaphores in FreeRTOS. Binary semaphores are used to control access to shared resources by multiple tasks.
