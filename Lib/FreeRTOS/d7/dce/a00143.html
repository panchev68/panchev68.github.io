Lib::FreeRTOS::QueueSet Class Reference

Detailed Description

A class representing a queue set in FreeRTOS.

This class provides a C++ interface for creating and managing queue sets in FreeRTOS. Queue sets are used to manage multiple queues and synchronize access to shared resources.
