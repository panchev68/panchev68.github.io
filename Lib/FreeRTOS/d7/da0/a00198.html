RTOS::SemaphoreCounting Class Reference

Detailed Description

A class representing a counting semaphore in FreeRTOS.

This class provides a C++ interface for creating and managing counting semaphores in FreeRTOS. Counting semaphores are used to control access to a shared resource by multiple tasks.
