A RAII-style guard that locks a mutex upon creation and unlocks it upon destruction.

This class template is used to manage the locking and unlocking of a mutex object. It ensures that the mutex is locked when the lock_guard object is created and automatically unlocks the mutex when the lock_guard object goes out of scope.

Template Parameters

Destructor for the lock_guard class.

This destructor unlocks the associated mutex when the lock_guard object goes out of scope. It ensures that the mutex is properly released, preventing deadlocks and ensuring that other threads can acquire the mutex.
