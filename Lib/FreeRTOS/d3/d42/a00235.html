RTOS::Timer Class Reference

Detailed Description

A class that encapsulates a FreeRTOS timer.

The Timer class provides an interface for creating, starting, stopping, and managing FreeRTOS timers. It supports both one-shot and auto-reload timers, and can be used in both task and ISR contexts.

Note
This class uses assertions to ensure that operations are performed on valid timer handles.

The Timer class provides the following functionalities:

  • Creating a timer with a specified name, callback function, period, and auto-reload flag.
  • Starting and stopping the timer, both from tasks and ISRs.
  • Setting the period of the timer, both from tasks and ISRs.
  • Resetting the timer.
  • Checking if the timer is active.
  • Retrieving the name, period, and expiry time of the timer.
  • Releasing the timer and deleting the object.
