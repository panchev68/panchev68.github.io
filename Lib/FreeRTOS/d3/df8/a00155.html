Lib::FreeRTOS::CoRoutine Class Reference

Detailed Description

A class to encapsulate FreeRTOS co-routine functionality.

This class provides a C++ interface for creating and managing FreeRTOS co-routines. It includes methods for creating co-routines, scheduling them, and sending/receiving data from queues within an ISR context.

CoRoutine.

CoRoutine constructor that creates a new co-routine.

This constructor initializes a co-routine by calling the FreeRTOS function xCoRoutineCreate. If the creation fails, it triggers an assertion.

Schedules the next co-routine to run.

This static function calls the FreeRTOS function vCoRoutineSchedule() to perform the scheduling of co-routines. It should be called periodically to ensure that co-routines are executed.
