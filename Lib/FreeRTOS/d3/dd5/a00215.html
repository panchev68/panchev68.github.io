Detailed Description

Abstract base class for handling TickService events.

The ClientBase class serves as an interface for handling TickService events. It provides a virtual destructor and a pure virtual function that must be implemented by derived classes.

#include <TickService.hpp>

Constructor for the ClientBase class.

This constructor attempts to register the client with the FreeRTOS TickService instance. If the registration fails, an assertion is triggered.

Note
The registration process is handled by the Lib::FreeRTOS::TickService::getInstance().registerClient() method.
Warning
If the client registration fails, the program will assert with a failure message.
