Timer

Example usage of the Timer class to create and manage FreeRTOS timers.

Example usage of the Timer class to create and manage FreeRTOS timers.This example demonstrates how to create, start, and stop a FreeRTOS timer using the Timer class. It shows how to set up a timer with a callback function and manage its lifecycle.

#include "Timer.hpp"
#include <iostream>
void timerCallback() {
std::cout << "Timer expired!" << std::endl;
}
int main() {
// Create a timer with a period of 1000 ticks (1 second) and auto-reload enabled
Lib::FreeRTOS::Timer timer("ExampleTimer", timerCallback, 1000, true, true);
// Wait for some time to let the timer expire a few times
vTaskDelay(pdMS_TO_TICKS(5000));
// Stop the timer
timer.stop();
// Release the timer
timer.release();
return 0;
}
