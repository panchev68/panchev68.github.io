TickService

This example demonstrates how to use the TickService class to manage TickService events in the FreeRTOS environment. It shows how to create a client that handles TickService events and register it with the TickService class. The example also demonstrates how to start the TickService event handling and receive TickService events.

/**
* @file TickService.cpp
* @brief Demonstrates the usage of the TickService class in FreeRTOS.
*
* This file contains an example implementation of a tick service client using the FreeRTOS TickService class.
* The example increments a counter every tick (1 ms) using the handleTickEvent method.
*
* @author Bobby
* @date 05.02.2025
*/
#include <iostream>
#include <RTOS.hpp>
uint32_t counter = 0;
/**
* @brief Client class to handle tick events.
*
* This class inherits from the RTOS::TickService::ClientBase class and implements the handleTickEvent method.
* The handleTickEvent method increments a counter every tick (1 ms).
*/
class Client : public RTOS::TickService::ClientBase {
void handleTickEvent () override {
// Calling every 1 ms (1 tick) from the TickService
counter++;
}
};
/**
* @brief Main function to demonstrate the Tick class.
*
* This function demonstrates how to use the Tick class to manage tick events in the FreeRTOS environment.
* It creates an instance of the Tick class, registers a client to handle tick events, and starts the tick event handling.
*
* @return int The exit status of the application.
*/
int main () {
auto & tickService = RTOS::TickService::getInstance ();
tickService.start ();
return 0;
}
