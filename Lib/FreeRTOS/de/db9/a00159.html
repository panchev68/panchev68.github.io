RTOS::MessageBuffer Class Reference

Description

C++ wrapper for FreeRTOS message buffers.

Encapsulates FreeRTOS message buffer operations, providing methods for creation, destruction, sending, receiving, and status queries. Supports both task and ISR contexts for flexible inter-task messaging.
