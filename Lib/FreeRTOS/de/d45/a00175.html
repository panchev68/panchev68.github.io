Detailed Description

template<typename T, size_t T_QUEUE_LEN = 1>
class Lib::FreeRTOS::Queue< T, T_QUEUE_LEN >

A wrapper class for FreeRTOS queue operations.

This class provides a C++ interface for creating, managing, and interacting with FreeRTOS queues.

Template Parameters
TThe type of elements stored in the queue.

This function is used to send an item to the back of the queue from within an Interrupt Service Routine (ISR). It ensures that the queue is created before attempting to send the item. If the item is successfully sent to the queue, it will yield from the ISR if a higher priority task was woken.
