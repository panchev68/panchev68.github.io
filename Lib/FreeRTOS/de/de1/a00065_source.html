226 /**
227 * @brief Sends a message to the queue from an ISR context.
228 *
229 * This function sends a message to the queue from an interrupt
230 * service routine (ISR). The message is copied from the memory pointed to
231 * by `src` and has a length of `len` bytes. The function will not block
232 * and will return immediately if there is not enough space in the buffer
233 * to hold the message.
234 *
235 * @param src Pointer to the message to be sent.
236 * @param len Length of the message in bytes.
237 * @return The number of bytes actually sent to the buffer, which will be
238 * equal to `len` if the message was sent successfully, or 0 if the message
239 * could not be sent.
240 *
241 * @note This function asserts that the message buffer handle is not null,
242 * that the source pointer is valid, and that the message length is greater
243 * than 0.
244 */
245 virtual bool sendToBackFromISR (const T & src)

253 return true;
