Queue.hpp File Reference

Detailed Description

This file contains the definition of the Queue class template for FreeRTOS.

The Queue class provides a C++ wrapper around the FreeRTOS queue functionality. It allows for creating, managing, and interacting with FreeRTOS queues in a type-safe manner.

Note
This implementation assumes that FreeRTOS is properly configured and included in the project.
