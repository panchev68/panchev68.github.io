Lib::FreeRTOS::Mutex Class Reference

Detailed Description

A class representing a mutex in FreeRTOS.

This class provides a C++ interface for creating and managing mutexes in FreeRTOS. Mutexes are used to protect shared resources from concurrent access by multiple tasks.
