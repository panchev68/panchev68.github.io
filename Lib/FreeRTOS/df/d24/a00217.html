RTOS::TaskHelper Class Reference

Detailed Description

A class providing helper functions for FreeRTOS tasks.

This class provides a set of static functions that allow access to FreeRTOS tasks. The functions include resuming and suspending tasks, setting and getting task priorities, and handling task notifications.

#include <TaskHelper.hpp>
