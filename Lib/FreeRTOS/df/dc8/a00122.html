TickService.hpp File Reference

Detailed Description

Header file for the TickService class in the FreeRTOS library.

This file contains the declaration of the TickService class, which is responsible for managing TickService events in the FreeRTOS environment. It includes the definition of the ClientBase abstract class, which serves as an interface for handling TickService events. The TickService class is implemented as a singleton to ensure that only one instance exists. It provides methods for starting the TickService event handling, registering clients, and unregistering clients.

The TickService class uses a mutex to ensure thread-safe access to the list of registered clients. The clients are notified of TickService events through the handleTickServiceEvent method defined in the ClientBase class.

Dependencies:

  • Mutex.hpp: Provides the Mutex class for thread synchronization.
  • Timer.hpp: Provides the Timer class for scheduling TickService events.
  • <vector>: Used for storing the list of registered clients.
Note
This file is part of the FreeRTOS library and is intended to be used with the IAR Embedded Workbench for ARM (EWARM) toolchain.
