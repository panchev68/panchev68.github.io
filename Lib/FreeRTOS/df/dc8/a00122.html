TickCounter.hpp File Reference

Detailed Description

Provides an interface to retrieve the current tick count from the FreeRTOS scheduler.

This header file defines the TickCounter class, which provides static methods to get the current tick count from the FreeRTOS scheduler, both from a normal task context and from an ISR context.
