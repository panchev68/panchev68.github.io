Tick.hpp File Reference

Detailed Description

Header file for the Tick class in the FreeRTOS library.

This file contains the declaration of the Tick class, which is responsible for managing tick events in the FreeRTOS environment. It includes the definition of the ClientBase abstract class, which serves as an interface for handling tick events. The Tick class is implemented as a singleton to ensure that only one instance exists. It provides methods for starting the tick event handling, registering clients, and unregistering clients.

The Tick class uses a mutex to ensure thread-safe access to the list of registered clients. The clients are notified of tick events through the handleTickEvent method defined in the ClientBase class.

Dependencies:

  • Mutex.hpp: Provides the Mutex class for thread synchronization.
  • Timer.hpp: Provides the Timer class for scheduling tick events.
  • <vector>: Used for storing the list of registered clients.
Note
This file is part of the FreeRTOS library and is intended to be used with the IAR Embedded Workbench for ARM (EWARM) toolchain.
