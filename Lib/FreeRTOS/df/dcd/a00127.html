Lib::FreeRTOS::MemoryManager Class Reference

Detailed Description

A class providing memory management functions for FreeRTOS.

This class provides static functions for allocating and freeing memory using the FreeRTOS memory management functions. It also includes functions for retrieving the amount of free heap memory available.

inlinestatic

Frees memory using the FreeRTOS memory management functions.

This function frees memory using the FreeRTOS function vPortFree. It asserts that the memory block is not null before freeing it.

Parameters

inlinestatic

Allocates memory using the FreeRTOS memory management functions.

This function allocates memory using the FreeRTOS function pvPortMalloc. If the allocation fails, the function asserts and throws a std::bad_alloc exception if exceptions are enabled.
