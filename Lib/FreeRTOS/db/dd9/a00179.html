Lib::FreeRTOS::Timeout Class Reference

Detailed Description

A class providing a C++ interface for FreeRTOS timeouts.

This class provides a C++ interface for creating and managing timeouts in FreeRTOS. Timeouts are used to trigger an event after a specified duration has elapsed.

Examples
Timeout.
