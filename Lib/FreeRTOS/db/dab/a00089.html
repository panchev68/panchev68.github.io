Detailed Description

A C++ wrapper for FreeRTOS message buffers.

This file contains the declaration of the MessageBuffer class, which provides a C++ interface for creating and managing FreeRTOS message buffers. The class includes methods for sending and receiving messages, as well as checking the status of the message buffer.
