Lib::System::Configuration Class Reference

Detailed Description

Manages configuration data stored in OTP (One-Time Programmable) memory.

This class provides methods to read and write configuration data to a specific address in OTP memory. It ensures that the data written is masked with a valid mask and provides methods to check if the configuration is empty or valid.

Lib::System::Configuration::Configuration ( uint32_t offset)
inline

Constructs a Configuration object with a specified offset.

This constructor initializes the Configuration object by setting the flash address to the beginning address of the OTP (One-Time Programmable) memory plus the given offset. It also asserts that the offset is within the valid range of the OTP memory.

Parameters
offsetThe offset to be added to the beginning address of the OTP memory.
Note
The offset must be less than the difference between the end address and the beginning address of the OTP memory.

Checks if the value at the specified flash address is valid.

This function reads the value from the flash memory at the address specified by _flashAddress and checks if it matches the VALID_MASK. If the value matches the mask, the function returns true, indicating that the value is valid. Otherwise, it returns false.
