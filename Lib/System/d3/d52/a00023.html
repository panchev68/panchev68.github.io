Configuration.hpp File Reference

Detailed Description

This file contains the definition of the Configuration class, which provides an interface for reading and writing configuration data to OTP (One-Time Programmable) memory.

The Configuration class is part of the Lib::System namespace and utilizes the Lib::HAL::OTP class for interacting with OTP memory.

Note
The class ensures that the offset provided during construction is within the valid range of the OTP memory.
  • The class uses a bitset to represent the configuration data.
  • It provides methods to read and write configuration data.
  • It includes methods to check if the configuration data is empty or valid.
