This section provides an overview of the main classes in the System Library.

Lib::System::FirmwareVersion

"FirmwareVersion" class is responsible for handling the firmware version of the project. It provides methods to get the build date, version value, subversion, and to convert the version to a string.

Lib::System::Configuration

"Configuration" class is responsible for handling the system configuration. It provides methods to read the configuration from a file and store it in a map. The configuration can be accessed using the getConfiguration method.

Lib::System::ParametersData

"ParametersData" class is responsible for handling the system parameters. It provides methods to read the parameters from a file and store them in a map. The parameters can be accessed using the getParameter method.

Lib::System::ProduceDate

"ProduceDate" class is responsible for handling the produce date of the project. It provides methods to get the produce date and convert it to a string.

Lib::System::Version

"Version" class is responsible for handling the version of the project. It provides methods to get the firmware version, produce date, and convert the version to a string.
