Lib::System::Version::Firmware Class Reference

Detailed Description

A class representing firmware version information, derived from the Base class.

This class provides functionalities to retrieve firmware version details, including build date and version values.

Note
This class is protectedly inherited from the Base class.

◆ stringToU32()

static uint32_t Lib::System::Version::Base::stringToU32 ( const std::string & string)
inlinestaticprotectedinherited

Converts a version string in the format "X.Y" to a 32-bit unsigned integer.

This function takes a version string with two components separated by a dot (e.g., "1.2") and converts it to a 32-bit unsigned integer. The first component is placed in the higher byte and the second component is placed in the lower byte of the resulting integer.

Parameters
stringThe version string to convert. It should be in the format "X.Y".
Returns
A 32-bit unsigned integer representing the version.
