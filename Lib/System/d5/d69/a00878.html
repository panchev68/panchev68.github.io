Lib::System::ParametersBase< T >::Data Class Reference

Detailed Description

template<class T>
class Lib::System::ParametersBase< T >::Data

A packed class representing a data area with a CRC16 checksum.

The Data class is marked with the __packed attribute to ensure that its members are packed together without any padding bytes. This is useful for memory-constrained environments or when interfacing with hardware where precise data layout is required.

TThe type of the data contained in the data area.

Updates the validation by calculating the CRC16 checksum of the data.

This function computes the CRC16 checksum of the data and updates the crc member variable. The checksum is calculated over the memory region starting at the address of data and spanning the size of the type T.
