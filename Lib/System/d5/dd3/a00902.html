Lib::System::Version::Base Class Reference

Detailed Description

A base class for handling version information.

This class provides functionality to store, convert, and retrieve version information. The version is represented as a 32-bit unsigned integer, with the major and minor versions packed into the higher and lower bytes respectively. An optional subversion can also be specified.

This operator allows an object of the class to be implicitly converted to a uint32_t. The conversion is performed by shifting the value obtained from getValue() by 8 bits to the left and then performing a bitwise OR with the value obtained from getSubVersion().

inlinestaticprotected

Converts a version string in the format "X.Y" to a 32-bit unsigned integer.

This function takes a version string with two components separated by a dot (e.g., "1.2") and converts it to a 32-bit unsigned integer. The first component is placed in the higher byte and the second component is placed in the lower byte of the resulting integer.

Parameters
stringThe version string to convert. It should be in the format "X.Y".
Returns
A 32-bit unsigned integer representing the version.

Converts the version information to a string representation.

This function formats the version information stored in the object into a string in the format "major.minor.subversion". The major and minor versions are extracted from the _version member variable, while the subversion is obtained from the getSubVersion() method.
