Lib::System::SerialNumber Class Reference

Detailed Description

A class to represent and calculate a serial number.

This class provides functionality to calculate a serial number from a unique identifier (UID) and convert it to different formats such as uint32_t and std::string.

Note
The calculation is done using a hashing function from the Lib::Helper namespace.
