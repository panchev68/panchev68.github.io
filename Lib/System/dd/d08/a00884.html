Lib::System::ParametersBase Class Referenceabstract

Detailed Description

Manages system parameters with storage and validation.

The ParametersBase class is responsible for managing system parameters, including reading from and writing to storage, validating data integrity using CRC16, and restoring default values if necessary. It uses a StorageBase object for storage operations and a DataBase object for data management.

Note
This class is designed to be inherited by derived classes that implement specific parameter management functionality.

◆ setup()

bool Lib::System::ParametersBase::setup ( )
inline

Sets up the parameters by opening the storage, reading the data, validating it, and restoring default values if necessary.

Returns
true if the setup is successful, false otherwise.

This function performs the following steps:

  1. Opens the storage. If it fails, an assertion is triggered and the function returns false.
  2. Reads the data from the storage. If it fails, an assertion with a "Reading error" message is triggered and the function returns false.
  3. Checks if the data is valid. If it is not valid: a. Restores the parameters to their default values. b. Writes the default data to the storage. If it fails, an assertion with a "Writing error" message is triggered and the function returns false.
