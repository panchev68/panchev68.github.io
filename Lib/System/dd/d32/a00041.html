SerialNumber.hpp File Reference

Detailed Description

Defines the SerialNumber class for generating and representing a serial number.

This file contains the definition of the SerialNumber class, which is responsible for calculating a unique serial number based on the device's unique identifier (UID). The serial number can be represented as a 32-bit unsigned integer or as a formatted string.

Dependencies:

  • Helper/CityHash.hpp: Provides the CityHash function for hashing the UID.
  • <array>: Used for array operations.
  • <string>: Used for string operations.
Note
The UID_BASE macro should be defined to point to the base address of the device's UID.
#include "Helper/CityHash.hpp"
#include <array>
#include <string>
