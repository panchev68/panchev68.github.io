Lib::System::ProduceDate Class Reference

Detailed Description

A class to handle reading and writing production dates to a specific flash memory address.

This class provides methods to read, write, and check the validity of production dates stored in flash memory.

Note
The class assumes the use of a specific OTP (One-Time Programmable) memory interface provided by Lib::HAL::OTP.

virtual bool Lib::System::ProduceDate::read ( Date & result) const
inlinevirtual

Reads the date and time from a specified flash address.

This function reads a 32-bit value from the flash memory at the specified address, unpacks it into a DateTime object, and checks if the DateTime object is valid. If the DateTime object is valid, it assigns the DateTime object to the result parameter and returns true. If the flash memory is empty or the DateTime object is not valid, it returns false.

Parameters
[out]resultA reference to a DateTime object where the read value will be stored if it is valid.
Returns
true if the date and time were successfully read and are valid, false otherwise.

virtual bool Lib::System::ProduceDate::write ( const Date & value)
inlinevirtual

Writes a DateTime value to the flash memory.

This function writes a DateTime value to a specific flash memory address. It first checks if the memory is empty and if the DateTime value is valid. If both conditions are met, it unlocks the OTP (One-Time Programmable) memory, packs the DateTime value, and writes it to the flash memory address. If the first write attempt fails, it retries once. The function asserts if the write operation fails after the retry or if the OTP lock operation fails.

Parameters
valueThe DateTime value to be written to the flash memory.
Returns
true if the write operation is successful, false otherwise.
