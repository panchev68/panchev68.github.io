System Library

Detailed Description

System library.

Introduction

The system library provides the functionality to read the configuration from a file and store them in a map. The configuration can be accessed using the getConfiguration method. The system library also provides the functionality to read the parameters from a file and store them in a map. The parameters can be accessed using the getParameter method. The system library also provides the functionality to get the version of the project. The version is defined in the VersionDef.hpp file. The version is defined as a string. The version is used to identify the version of the project.
