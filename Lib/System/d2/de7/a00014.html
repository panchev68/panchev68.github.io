Detailed Description

This file contains the definition of the System class and its associated members.

The System class encapsulates various system-related information and functionalities, including reset state, production date, sale date, configuration, serial number, and firmware and hardware versions. It also provides a method to restart the system.

The class relies on several other components such as Configuration, ProduceDate, SerialNumber, Version, and Lib::HAL::LastResetState.

Note
The NVIC_SystemReset macro is redefined for IntelliSense purposes.
