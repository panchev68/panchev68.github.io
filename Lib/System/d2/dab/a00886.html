Lib::System::HardwareVersion Class Reference

Detailed Description

A class representing the hardware version, derived from VersionBase.

This class provides an interface to handle hardware versioning, including methods to retrieve the version value, sub-version, and string representation.

Note
This class inherits from VersionBase and overrides its virtual methods.

#include <HardwareVersion.hpp>

Converts a version string in the format "X.Y" to a 32-bit unsigned integer.

This function takes a version string with two components separated by a dot (e.g., "1.2") and converts it to a 32-bit unsigned integer. The first component is placed in the higher byte and the second component is placed in the lower byte of the resulting integer.

Parameters
stringThe version string to convert. It should be in the format "X.Y".
Returns
A 32-bit unsigned integer representing the version.
