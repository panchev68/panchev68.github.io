Configuration

Example of using the Configuration class. The Configuration class is used to read the configuration from a file and store them in a map. The configuration can be accessed using the getConfiguration method. The Configuration class is used to read the configuration from the file and store them in the ConfigurationData class. The Configuration class also provides a method to get the configuration from the ConfigurationData class.

Configuration.cpp
/**
* @file Configuration.cpp
* @brief Example code for configuring system settings.
*
* This file contains example code for reading and writing system configuration
* settings using the Lib::System::Configuration class. The configuration settings
* include support for a voltmeter and CAN bus.
*
* @date 08.01.2025
*/
#include "System/Configuration.hpp"
/**
* @brief Enumerates the configuration bits.
*/
enum ConfigBits {
VOLTMETER_SUPPORT = 0x01, ///< Bit indicating support for a voltmeter.
VOLTMETER_VPP_2V5 = 0x02, ///< Bit indicating a voltmeter with a Vpp of 2.5V.
CANBUS_SUPPORT = 0x04, ///< Bit indicating support for a CAN bus.
};
/**
* @brief The system configuration object.
*/
Lib::System::Configuration configuration (0x100);
/**
* @brief Reads the system configuration.
* @return true if the configuration is valid, false otherwise.
*/
bool read () {
if (configuration.isValid () == false) return false;
auto bits = configuration.read ();
if (bits.test (VOLTMETER_SUPPORT) == true) {
// The system supports a voltmeter
}
return true;
}
/**
* @brief Writes the system configuration.
* @return true if the configuration was written successfully, false otherwise.
*/
bool write () {
if (configuration.isValid () == true) return false;
return configuration.write ({VOLTMETER_SUPPORT | CANBUS_SUPPORT});
}
