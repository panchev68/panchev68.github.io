Lib::System::System Class Reference

Detailed Description

Represents the system configuration and state.

The System class encapsulates various system-related information such as reset state, production date, sale date, configuration, serial number, and firmware and hardware versions. It provides functionality to restart the system.

Examples
Example of using the System class.

#include <System.hpp>
