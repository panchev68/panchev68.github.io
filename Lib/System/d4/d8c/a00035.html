ProduceDate.hpp File Reference

Detailed Description

Defines the ProduceDate class for reading and writing production dates to OTP memory.

This file contains the definition of the ProduceDate class, which provides methods to read and write production dates stored in One-Time Programmable (OTP) memory. The class ensures that the dates are valid and handles the necessary unlocking and locking of the OTP memory during write operations.

Dependencies:

  • Lib/HAL/OTP.hpp: Provides access to the OTP memory.
  • Lib/Helper/DateTime.hpp: Provides the DateTime class for date and time manipulation.
