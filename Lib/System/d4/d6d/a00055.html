Reads the date from the flash memory.

This function attempts to read a date from the flash memory at the specified flash address. If the flash memory is empty, the function returns false. Otherwise, it reads the date and returns true if the date is valid.

◆ write()

virtual bool Lib::System::ProduceDate::write ( const Date & value)
inlinevirtual

Writes a Date value to the flash memory.

This function writes the provided Date value to the flash memory if the memory is empty and the Date value is valid. It uses the OTP (One-Time Programmable) memory interface to perform the write operation.

Parameters
valueThe Date value to be written to the flash memory.
Returns
true if the write operation was successful, false otherwise.
Note
The function performs the following steps:
  • Checks if the flash memory is empty.
  • Validates the provided Date value.
  • Unlocks the OTP memory.
  • Packs the Date value into a DateTime structure.
  • Writes the packed DateTime to the flash memory.
  • Locks the OTP memory.
  • Asserts if any of the critical operations fail.
