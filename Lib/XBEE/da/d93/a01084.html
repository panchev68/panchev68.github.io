Lib::Digi::API::Connection::Info::Period

Detailed Description

A class to measure and track time periods with minimum, maximum, and average values.

The Period class provides functionality to measure time intervals, track the minimum and maximum values, and calculate an average value. It uses FreeRTOS tick counts to measure elapsed time.

Note
The class assumes that FreeRTOS is being used and that the FreeRTOS::TickCounter::get() function is available to retrieve the current tick count.

The Period class provides the following functionalities:

  • Start and stop the measurement process.
  • Retrieve the current, raw, minimum, and maximum values.
  • Reset the measurement statistics.
  • Implicit conversion to uint32_t to get the current value.

Begins the measurement process.

◆ endMeasurement()

void Lib::Digi::API::Connection::Info::Period::endMeasurement ( )
inline

Ends the measurement process and updates the measurement statistics.

This function stops the measurement process if it is currently started. It calculates the elapsed time since the measurement started and updates the measurement value, raw value, minimum value, and maximum value accordingly.

The function performs the following steps:

  • Checks if the measurement process is started. If not, it returns immediately.
  • Stops the measurement process by setting _isStarted to false.
  • Calculates the elapsed time since the measurement started.
  • Updates _value based on the calculated elapsed time and the previous raw value.
  • Updates _rawValue with the calculated elapsed time.
  • Updates _minimum with the calculated elapsed time if it is less than the current minimum or if the minimum is zero.
  • Updates _maximum with the calculated elapsed time if it is greater than the current maximum or if the maximum is zero.
