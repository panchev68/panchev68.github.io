Lib::Xbee::Core::Message

Detailed Description

Manages and manipulates messages for the Xbee communication module.

The Message class provides functionality to manage and manipulate messages for the Xbee communication module. It includes constructors for initializing messages from vectors or raw data, and a conversion operator to convert the message back to a vector.

Note
The maximum size of the message data is defined by DATA_SIZE.
