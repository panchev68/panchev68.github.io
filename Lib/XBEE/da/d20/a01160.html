Lib::Digi::API::Service::Parameters::Data

Detailed Description

Represents the data structure for the Parameters class.

The Data struct represents the data structure for the Parameters class. It contains various parameters related to XBee devices, including the node identifier, serial number, PAN ID, channel, firmware version, hardware version, and other parameters.
