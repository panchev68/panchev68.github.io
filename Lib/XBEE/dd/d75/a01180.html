Detailed Description

SpectrumAnalyzer class.

The SpectrumAnalyzer class is responsible for analyzing the spectrum using local AT commands. It inherits from the ServiceBase class.

◆ updateCallback()

void Lib::Digi::API::Service::SpectrumAnalyzer::updateCallback ( )
inlineoverridevirtual

Updates the callback for the Spectrum Analyzer service.

This function handles the periodic update of the Spectrum Analyzer service. It sends a local AT command "ED" to the device and processes the response. The response is converted to digital percentage values and used to update the current, average, and maximum arrays.

The function uses a timeout mechanism to manage the sending and receiving of the AT command. If the timeout expires without receiving a response, the function resets the waiting state and adjusts the timeout duration.

Note
This function overrides a virtual function from the base class.
