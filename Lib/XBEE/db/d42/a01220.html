Lib::Digi::Core::Frame::Data

Detailed Description

Manages a vector of uint8_t elements.

The Data class provides functionalities to manage a vector of uint8_t elements, including assigning and appending data.

Specialization of the append function template for uint16_t type.

This function appends a 16-bit unsigned integer to the vector by splitting it into two 8-bit values. The higher 8 bits are pushed first, followed by the lower 8 bits.
