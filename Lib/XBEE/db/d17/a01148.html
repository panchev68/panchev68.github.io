Lib::Digi::API::Service::NetworkDiscovery::Node

Detailed Description

Represents a network node with address, MAC address, and identifier.

The Node class encapsulates the details of a network node, including its 16-bit address, MAC address, and a string identifier. It provides a constructor to initialize these details from raw data.

Note
This class assumes that the raw data provided to the constructor is correctly formatted and contains valid information for a node.
