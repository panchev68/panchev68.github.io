Detailed Description

Abstract base class for communication ports.

This class provides an interface for communication ports, defining the basic operations that any communication port must support. Derived classes must implement the pure virtual functions to handle the specific logic for opening, closing, sending, and receiving data through the communication port.

Note
This class cannot be instantiated directly. It must be subclassed, and the pure virtual functions must be implemented by the derived class.
