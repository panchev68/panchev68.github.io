Lib::Digi::API::Service::NetworkDiscovery

Detailed Description

Discovers nodes in a network.

The NetworkDiscovery class is responsible for discovering nodes in a network. It uses the Core::LocalAtCommand class to send and receive AT commands to/from the network.

bool Lib::Digi::API::Service::NetworkDiscovery::open ( )
inlineoverridevirtual

Opens the network discovery service.

This function initializes the AT command interface and sends the "NT" command to retrieve the network discovery timeout. The response is expected to be 2 bytes long. If the response size is not 2 bytes, the function returns false. The retrieved timeout value is converted to milliseconds and stored in _discoveryTimeout. Finally, the base service open function is called.

Returns
true if the service was successfully opened, false otherwise.

const std::vector< Node > & Lib::Digi::API::Service::NetworkDiscovery::start ( )
inline

Starts the network discovery process.

This function clears the existing data, sends the "ND" AT command to initiate network discovery, and waits for responses within a specified timeout period. Discovered nodes are stored in the _data vector.

Returns
const std::vector<Node>& A reference to the vector containing discovered nodes.
