Lib::Xbee::API::Service::NetworkDiscovery

Detailed Description

Discovers nodes in a network.

The NetworkDiscovery class is responsible for discovering nodes in a network. It uses the Core::LocalAtCommand class to send and receive AT commands to/from the network.

◆ start()

const std::vector< Node > & Lib::Xbee::API::Service::NetworkDiscovery::start ( )
inline

Starts the network discovery process.

This function initiates the network discovery process by sending the "ND" AT command to the XBee module. It waits for responses within a specified timeout period and collects discovered nodes into a vector.

Returns
const std::vector<Node>& A reference to the vector containing discovered nodes.
Note
The function clears the existing data before starting the discovery process. It uses FreeRTOS for timeout management.
Exceptions
assert(0)if the AT command fails to send.
