Lib::Xbee::Core::AtCommandString

Detailed Description

Represents an AT command string in the Xbee library.

The AtCommandString class is used to represent and manipulate AT command strings in the Xbee library. It provides constructors for initializing from a string or a uint16_t value and an operator for converting to uint16_t.
