Lib::Digi::Core::QueueMessage

Detailed Description

Manages a fixed-size array of data.

The QueueMessage class is used to store and manage a fixed-size array of data. It provides a conversion operator to convert the stored data into a std::vector<uint8_t>.

Lib::Digi::Core::QueueMessage::operator std::vector< uint8_t > ( ) const
inline

Conversion operator to convert the QueueMessage object to a std::vector<uint8_t>.

This operator allows the QueueMessage object to be implicitly converted to a std::vector<uint8_t>. It creates a new vector and assigns the contents of the internal array (data) to the vector, up to the size of the array.

Returns
A std::vector<uint8_t> containing the data from the internal array.
