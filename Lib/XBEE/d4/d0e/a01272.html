Lib::Xbee::Core::Frame0x08

Detailed Description

Represents a Local AT Command Request frame (Frame 0x08) for XBee communication.

This class is used to construct and manage the data for a Local AT Command Request frame, which is used to send a command to the local device. The frame type is identified as 0x08.
