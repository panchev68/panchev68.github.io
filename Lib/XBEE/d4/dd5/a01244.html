Lib::Digi::Core::LocalAtCommand

Detailed Description

Provides methods to send and receive AT commands to and from a local XBee module.

The LocalAtCommand class provides methods to open and close the communication service, send AT commands, and receive responses. It uses FreeRTOS queues to manage message passing and ensures thread safety.

void Lib::Digi::Core::LocalAtCommand::open ( )
inline

Opens the local AT command service.

This function initializes and opens the local AT command service. It ensures that the service is not already open, sets up the necessary callback for handling responses, and creates the message queue for the service.

Precondition
The service must not already be open.
The response event handler for local AT commands must be null.
Postcondition
The service will be marked as open and ready to handle local AT commands.
Exceptions
Assertionfailure if the service is already open or if the message queue creation fails.
