Detailed Description

This file contains the declaration and implementation of the Parameters class, which is part of the Lib::Xbee::API::Service namespace.

The Parameters class is responsible for managing and manipulating various parameters related to XBee devices, including reading and writing static and dynamic parameters.

Note
This file is part of the XBee library and is intended to be used with the IAR Embedded Workbench for ARM (EWARM) version 6.00.
