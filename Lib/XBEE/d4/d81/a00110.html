NetworkDiscovery.hpp File Reference

Detailed Description

Header file for the NetworkDiscovery service class.

This file contains the declaration of the NetworkDiscovery class, which is responsible for discovering nodes in a network. It uses the Core::LocalAtCommand class to send and receive AT commands to/from the network.
