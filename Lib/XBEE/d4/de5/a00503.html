Lib::Xbee::Core::Frame0x89

Detailed Description

Represents a Frame-0x89 response frame.

This class is used to parse and store the data from a Frame-0x89 response frame.

Frame-0x89 is used to identify the data frame for the host to correlate with a prior request.

Note
The frame vector must have a size of 2 for the constructor to properly initialize the frameId and deliveryStatus.
