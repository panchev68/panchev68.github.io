Lib::Xbee::Core::ResponseEvents

Detailed Description

Handles various XBee response events.

The ResponseEvents class provides a set of event handlers for different XBee response frames. It uses std::function to allow users to define custom behavior for each event. The class is designed to work with the Protocol class and processes response frames to trigger the appropriate events.
