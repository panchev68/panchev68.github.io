Lib::Digi::API::Connection::Quality

Detailed Description

Monitors the quality of an XBee connection.

The Quality class monitors the quality of an XBee connection based on various metrics such as retries, transmit time, and RSSI. It provides a percentage value representing the quality of the connection, as well as a state value indicating the overall quality level.

enum Lib::Digi::API::Connection::Quality::State

Enumeration for different quality states.

The State enumeration defines the following quality states:

  • DISCONNECTED: The connection is disconnected.
  • UNUSABLE: The connection is unusable.
  • NOT_GOOD: The connection quality is not good.
  • OKAY: The connection quality is okay.
  • VERY_GOOD: The connection quality is very good.
  • AMAZING: The connection quality is amazing.
