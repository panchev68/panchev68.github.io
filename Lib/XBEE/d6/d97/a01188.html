Lib::Digi::Core::AtCommandString

Detailed Description

A class to represent an AT command string.

The AtCommandString class provides a way to construct and manipulate AT command strings. It can be initialized from a string or a 16-bit unsigned integer value, and can be implicitly converted to a uint16_t.

Note
The string used to initialize the object must have a size of 2.

Constructs an AtCommandString object from a given string.

This constructor takes a string and converts it into an AtCommandString. The input string must have exactly 2 characters. The first character is shifted left by 8 bits and then combined with the second character using a bitwise OR operation to form the final value.

Parameters
stringThe input string which must be exactly 2 characters long.
Exceptions
std::assertif the input string does not have exactly 2 characters.
