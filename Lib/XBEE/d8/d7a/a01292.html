Detailed Description

An abstract base class for query objects.

The QueryBase class serves as an abstract base class for query objects that need to convert their data into a Frame::Data object. Derived classes must implement the pure virtual function to provide the specific behavior for this conversion.

Note
This class cannot be instantiated directly. It must be inherited by a derived class that implements the pure virtual function.
