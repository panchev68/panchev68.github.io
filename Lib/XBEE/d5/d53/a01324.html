Lib::Digi::Core::Frame0x8a

Detailed Description

Modem Status - 0x8A.

This frame type is emitted in response to specific conditions. The status field of this frame indicates the device behavior.

This class encapsulates the data and functionality related to a frame with ID 0x8a. It provides member variables to store frame-specific information and a constructor to initialize these variables from a given frame data.
