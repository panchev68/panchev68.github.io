Frame0x80.hpp File Reference

Detailed Description

Defines the Frame0x80 class for handling 64-bit Receive Packet frames (ID 0x80) from XBee devices.

This file contains the definition of the Frame0x80 class, which is used to parse and store data from frames with ID 0x80 received from XBee devices. The frame data includes the 64-bit source address, RSSI value, options, and RF data. The class provides member variables to store this information and a constructor to initialize these variables from a given frame data. The frame data is expected to be provided as a Frame::Data object.

Note
The frame data is expected to be provided as a Frame::Data object.
