Lib::Xbee::Core::Diagnostic

Detailed Description

Tracks various message statistics in the Xbee Core library.

The Diagnostic class is used to track various message statistics in the Xbee Core library, including the number of received and sent messages, ignored messages due to timeout, maximum message length errors, and invalid messages.
