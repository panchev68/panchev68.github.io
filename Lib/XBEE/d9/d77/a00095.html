Connection.hpp File Reference

Detailed Description

Defines the Connection class and its nested classes for managing and monitoring XBee connections.

This file contains the declaration of the Connection class, which is part of the Lib::Xbee::API namespace. The Connection class provides mechanisms to manage the state and quality of a connection, as well as information about the connection such as signal strength and transmission periods.

The Connection class contains the following nested classes:

  • State: Manages the state of the connection (INIT, READY, CONNECTED, DISCONNECTED).
  • Quality: Monitors the quality of the connection based on various metrics such as retries, transmit time, and RSSI.
  • Info: Holds information about the connection, including signal strength, transmit periods, and various counters.

The Quality class also defines an enumeration for different quality states (DISCONNECTED, UNUSABLE, NOT_GOOD, OKAY, VERY_GOOD, AMAZING) and a Mode enumeration for runtime and scanner modes.

The Info class contains nested classes for managing counters and periods, as well as a SignalStrength class for handling signal strength values.

Note
This file is part of the XBee library and is intended to be used with the FreeRTOS operating system.
