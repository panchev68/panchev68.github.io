Lib::Digi::API::Connection::Info::Counter

Detailed Description

A simple counter class that increments its value up to a maximum limit.

The Counter class provides a mechanism to increment a value up to a specified maximum value. It supports post-increment operation and can be cleared to reset the value to zero.

Note
The maximum value for the counter is defined by the MAX_VALUE constant.
