Lib::Xbee::Core::DigiApiFrameBuffer

Detailed Description

Provides a buffer for storing and manipulating data frames used in Digi API communication.

The DigiApiFrameBuffer class provides a buffer for storing and manipulating data frames used in Digi API communication. It supports operations such as assigning, appending, and clearing data, as well as retrieving the buffer size and data.


Example:
Consider the following sample data packet:
7E 00 0A 01 01 50 01 00 48 65 6C 6C 6F B8

Byte(s) Description
7E Start Delimiter
00 0A Length bytes
01 API identifier
01 API Frame ID
50 01 Destination Address Low
00 Option byte
48 65 6C 6C 6F Data Packet
B8 Checksum
