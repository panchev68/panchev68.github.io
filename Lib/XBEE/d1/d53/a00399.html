Lib::Xbee::Core::EscapedCode

Detailed Description

Determines if a given value is an escaped code in the Xbee communication protocol.

The EscapedCode class is used to determine if a given value is an escaped code in the Xbee communication protocol. It provides a method to precode a character by applying an XOR mask.
