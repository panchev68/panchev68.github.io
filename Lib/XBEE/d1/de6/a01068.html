Lib::Xbee::API::Connection::State

Detailed Description

Represents the state of a connection.

The State class encapsulates the state of a connection and provides mechanisms to handle state changes and notify listeners through a callback function.

The possible states are:

  • INIT: The initial state of the connection.
  • READY: The connection is ready to be established.
  • CONNECTED: The connection is established.
  • DISCONNECTED: The connection is disconnected.

The class provides methods to check the connection status, assign new states, and convert the state to a uint32_t value.
