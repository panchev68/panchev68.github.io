Lib::Xbee::Core::Frame0x80

Detailed Description

Represents a frame with ID 0x80 received from an XBee device.

64-bit Receive Packet-0x80

This frame type is emitted when a device configured with legacy API output—AO (API Output Options) = 2—receives an RF data packet from a device an RF data packet from a device configured to use 64-bit source addressing—MY = 0xFFFE.

This class is used to parse and store the data from a frame with ID 0x80, which includes the 64-bit source address, RSSI value, options, and RF data.

Note
The frame data is expected to be provided as a Frame::Data object.

The Frame0x80 class provides the following information:

  • 64-bit source address (srcAddr64)
  • RSSI value (rssi)
  • Options (option)
  • RF data (rfData)
Request frames:
  • Transmit Request
  • Explicit Addressing Command Request
  • 64-bit Transmit Request
  • 16-bit Transmit Request
Example
Frame::Data frameData = ...; // Obtain frame data from XBee device
Frame0x80 frame(frameData); // Parse the frame data
uint64_t address = frame.srcAddr64; // Get the source address
uint8_t signalStrength = frame.rssi; // Get the RSSI value
uint8_t options = frame.option; // Get the options
std::vector<uint8_t> data = frame.rfData; // Get the RF data

uint8_t Lib::Xbee::Core::Frame0x80::rssi

Received Signal Strength Indicator (RSSI) value.

This variable holds the RSSI value which indicates the strength of the received signal. The value is typically represented in dBm (decibel-milliwatts). The Hexadecimal equivalent of (-dBm) value.

example:
if RX signal strength is -40 dBm, then 0x28 (40 decimal) is returned.

◆ option

uint8_t Lib::Xbee::Core::Frame0x80::option

Represents the option byte in the frame.

This byte contains various flags and settings that determine the behavior and properties of the frame.

Bit field of options that apply to the received message:

  • Bit 0: Reserved
  • Bit 1: Packet was sent as a broadcast [0x02]
  • Bit 2: 802.15.4 only - Packet was broadcast across all PANs [0x04]
    Note
    Note option values may be combined.
