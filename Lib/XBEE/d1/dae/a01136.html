Lib::Digi::API::Service::ChannelScanner::Channel

Detailed Description

Represents a communication channel with a value and connection quality.

The Channel class encapsulates a communication channel with an associated value and connection quality. It provides methods to get and set the connection quality, as well as operators for assignment and conversion to uint8_t.
