Lib::Digi::Core::Checksum

Detailed Description

A class for calculating checksums for DigiApiFrameBuffer and raw data arrays.

The Checksum class provides functionality to calculate checksums for DigiApiFrameBuffer objects and raw data arrays. It supports two constructors for different input types and a conversion operator to retrieve the calculated checksum as a uint8_t.

Note
The checksum is calculated by subtracting each byte of the payload from an initial value of 0xff.

Lib::Digi::Core::Checksum::Checksum ( const DigiApiFrameBuffer & apiFrame)
inline

Constructs a Checksum object and calculates the checksum for the given DigiApiFrameBuffer.

This constructor takes a DigiApiFrameBuffer object, extracts the header, calculates the payload length, and computes the checksum by iterating over the payload data. The checksum is stored in the _result member.

Parameters
apiFrameThe DigiApiFrameBuffer object containing the data for which the checksum is to be calculated.

Lib::Digi::Core::Checksum::Checksum ( const uint8_t * data)
inline

Constructs a Checksum object and calculates the checksum for the given data.

This constructor takes a pointer to a data array, interprets the first part of the data as a Frame::Header, and calculates the checksum for the entire message.

Parameters
dataPointer to the data array. The data array must contain at least the size of Frame::Header plus the payload length specified in the header.
Note
The checksum is calculated by subtracting each byte of the payload from an initial value of 0xff.
Precondition
The data pointer must not be null.
