Lib::Xbee::Core::Frame0x09

Detailed Description

Represents a Queue Local AT Command Request frame (Frame 0x09) for XBee communication.

This class is used to construct and manage the data for a Queue Local AT Command Request frame, which is used to send a command to the local device. The frame type is identified as 0x09.
