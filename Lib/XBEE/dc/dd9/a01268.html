Lib::Digi::Core::Frame0x01

Detailed Description

Represents a frame with type 0x01 for transmitting data to a 16-bit address.

This class inherits from QueryBase and provides an implementation for converting the frame data into a Frame::Data object.

Note
The frame type is set to Frame::Type::TxRequest16bitAddr.
