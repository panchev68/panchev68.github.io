Detailed Description

Defines the UserProtocol class for handling XBee API commands.

This file contains the definition of the UserProtocol class, which provides methods for parsing and building XBee API commands. The class includes virtual methods for handling specific command types, which can be overridden by derived classes to implement custom behavior.
