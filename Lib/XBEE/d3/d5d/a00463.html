Lib::Xbee::Core::Frame0x17

Detailed Description

Represents a Remote AT Command Request frame (Frame 0x17) for XBee communication.

This class is used to construct and manage the data for a Remote AT Command Request frame, which is used to send a command to a remote device. The frame type is identified as 0x17.

uint8_t Lib::Xbee::Core::Frame0x17::remoteCommandOption = 0

Remote command option.

This variable holds the remote command option for the frame.

  • Bit 0: Disable ACK [0x01]
  • Bit 1: Apply changes on remote device [0x02]
  • Bit 2: Reserved
  • Bit 3: Reserved
  • Bit 4: Send the remote command securely [0x10]
    • Requires a secure session be established with the destination
