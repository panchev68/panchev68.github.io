Lib::Xbee::API::Service::ServiceBase

Detailed Description

Base class for services in the Xbee API.

The ServiceBase class provides the base functionality for services in the Xbee API. It includes methods for opening, closing, and updating the service, as well as callback methods that can be overridden by derived classes.
