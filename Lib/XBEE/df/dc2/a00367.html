Lib::Xbee::API::UserProtocol

Detailed Description

Provides methods for parsing and building XBee API commands.

The UserProtocol class provides methods for parsing and building XBee API commands. It includes virtual methods for handling specific command types, which can be overridden by derived classes to implement custom behavior.

enum Lib::Xbee::API::UserProtocol::Command

Enumeration of XBee API command types.

This enumeration defines the different types of XBee API commands that can be handled by the UserProtocol class. The command types include INPUTS, GET_PARAMETERS, OUTPUTS, and SET_PARAMETERS.
