Lib::Digi::API::ServerBaseabstract

Detailed Description

A base class for server applications, inheriting from ApplicationBase.

The ServerBase class provides the basic functionality for server applications, including starting the application, sending data, and handling task events.

virtual bool Lib::Digi::API::ServerBase::sendData ( const std::vector< uint8_t > & vector)
inlineoverrideprotectedvirtual

Sends data using the Core::Frame0x01 frame.

This function attempts to send the provided data vector using the Core::Frame0x01 frame. It first checks if the module is present. If the module is not present, it returns false. If the module is present, it sets the frame ID, destination address, and options, then assigns the provided data to the frame's RF data field. It then attempts to send the frame. If the frame is sent successfully, it increments the number of sent data messages and returns true. If the frame sending times out, it asserts a failure, increments the number of sent data messages with timeout, and returns false.

Parameters
vectorThe data to be sent.
Returns
true if the data was sent successfully, false otherwise.
