Lib::Digi::Core::ModemStatus

Detailed Description

Handles various modem status events.

The ModemStatus class provides a set of event handlers for different modem status changes. It uses std::function to allow users to define custom behavior for each event. The class is designed to work with a Module object and processes modem status frames to trigger the appropriate events.
