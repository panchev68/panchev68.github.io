Detailed Description

Optimization configuration for RTOS.

This file defines optimization-related macros, constants, and helper templates for RTOS performance tuning. It provides compile-time configuration for inlining, memory alignment, buffer sizing, and task/timer optimization, supporting different optimization levels for embedded and real-time systems.
