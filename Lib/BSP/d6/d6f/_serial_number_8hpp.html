Detailed Description

SerialNumber class for generating and formatting unique serial numbers.

Provides robust methods to generate a unique serial number using device UID and CityHash, supporting conversion to string and integer formats for embedded systems. Ensures type safety, input validation, and reliable access to UID memory, supporting serial number storage and retrieval with error handling and memory protection.
