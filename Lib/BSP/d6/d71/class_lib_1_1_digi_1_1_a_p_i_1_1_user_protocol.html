Detailed Description

Base class for user-defined protocol parsing and message building.

The UserProtocol class provides an abstract interface for parsing incoming messages and building outgoing messages according to user-defined protocol commands. It defines a set of protocol commands and callback methods for handling specific protocol actions. Derived classes can override these callbacks to implement custom protocol logic. The class ensures efficient message handling and type safety for Digi XBee API communication.
