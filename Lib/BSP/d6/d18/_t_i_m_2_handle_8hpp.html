Detailed Description

TIM handle wrapper for peripheral and channel management.

This file defines a lightweight wrapper for TIM_HandleTypeDef and channel management in STM32 HAL. It ensures type safety, valid handle usage, and facilitates direct access to TIM peripheral and channel. The wrapper is designed for embedded applications, supporting safe construction, comparison, and validity checks. It is intended for use in both real-time and non-real-time contexts, with careful separation of implementation details and interface contracts. The file adheres to C++17 standards and project documentation guidelines.
