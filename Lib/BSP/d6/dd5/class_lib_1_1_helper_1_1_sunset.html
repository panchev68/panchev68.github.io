Подробно описание

Sunset and moon phase calculation utility.

The Sunset class provides methods for calculating sunrise, sunset, and moon phase information for a specified geographic location and date. It supports official, nautical, civil, and astronomical sun event calculations, and computes times using precise astronomical algorithms. The class is suitable for embedded and real-time applications requiring solar and lunar event data, such as calendar utilities, astronomical tools, and environmental monitoring systems.
