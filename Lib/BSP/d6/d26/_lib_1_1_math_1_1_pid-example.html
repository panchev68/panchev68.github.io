Lib::Math::Pid

Example usage of the Pid controller for temperature regulation.

This example demonstrates how to configure and use the Pid controller to regulate the temperature of a simulated process. The controller is tuned for a simple first-order system and runs in a loop to maintain the setpoint. The output is clamped, and the integral term is protected against windup.

Example usage:
#include <iostream>
#include "Lib/Math/Pid.hpp"
int main() {
using Lib::Math::PidF;
// PID parameters: kp, ki, kd, sampleTime
PidF::Parameters params(2.0f, 0.5f, 1.0f, 0.1f, 0.0f, 100.0f);
PidF pid(params);
pid.setMode(true); // Enable automatic mode
float setpoint = 50.0f; // Target temperature
float process = 20.0f; // Initial temperature
float ambient = 20.0f; // Ambient temperature
float heaterPower = 0.0f;
for (int i = 0; i < 30; ++i) {
heaterPower = pid.compute(setpoint, process);
// Simulate process: simple first-order lag + disturbance
process += (heaterPower * 0.05f - (process - ambient) * 0.02f);
std::cout << "Step " << i
<< " | Setpoint: " << setpoint
<< " | Process: " << process
<< " | Output: " << heaterPower
<< std::endl;
}
return 0;
}
