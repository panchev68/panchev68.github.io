Подробно описание

TickCounter utility for FreeRTOS tick/time conversions and elapsed/timeout calculations.

This file provides a C++17 static utility class for converting between FreeRTOS ticks and milliseconds/seconds, calculating elapsed time, and checking timeouts. It encapsulates tick arithmetic, conversion logic, and provides safe, type-checked APIs for embedded use. The TickCounter class is designed for use in real-time and ISR contexts, supporting accurate time calculations and conversions with overflow-safe logic.
