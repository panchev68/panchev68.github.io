Detailed Description

Microsecond delay utility using ARM Cortex-M DWT.

This file provides the DelayUs class for precise microsecond delays using the Data Watchpoint and Trace (DWT) unit's cycle counter on ARM Cortex-M cores. The implementation configures the DWT and CoreDebug registers, calculates the required cycle count using the system clock frequency, and performs a bounded busy-wait loop. This utility is suitable for embedded applications requiring deterministic, accurate microsecond delays. It is designed for single-use delay instances, minimizing resource usage and side effects.
