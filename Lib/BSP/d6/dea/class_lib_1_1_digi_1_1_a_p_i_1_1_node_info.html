Detailed Description

XBee node information container.

The NodeInfo class encapsulates all relevant information about an XBee node, including MAC address, network info, discovery details, firmware/hardware versions, runtime monitor, and MAC diagnostic data. It provides nested types for each information category, supporting modular and extensible node management and diagnostics.
