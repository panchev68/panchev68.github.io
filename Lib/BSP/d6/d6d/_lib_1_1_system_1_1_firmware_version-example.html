Lib::System::FirmwareVersion

Example usage of FirmwareVersion class.

Demonstrates how to create a FirmwareVersion object, retrieve its value, sub-version, string representation, and build date.

Example usage:

#include "Lib/System/FirmwareVersion.hpp"
#include <iostream>
int main() {
Lib::System::FirmwareVersion version("1.2.3", 42);
std::cout << "Version value: " << version.getValue() << std::endl;
std::cout << "Sub-version: " << version.getSubVersion() << std::endl;
std::cout << "Version string: " << version.toString() << std::endl;
std::cout << "Build date: " << version.getBuildDate() << std::endl;
return 0;
}
