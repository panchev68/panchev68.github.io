Detailed Description

Timer template class declaration for TIM HAL component.

Declares the Timer template class, parameterized by Mode, for managing TIM peripherals in the HAL library. Integrates with Handle and Mode abstractions for type-safe, efficient TIM management. Ensure Mode.hpp is included before using this file. This file provides a generic interface for TIM peripheral management, supporting RAII ownership and future extension for callback/event integration. The Timer class template is designed for embedded applications, ensuring compatibility with C++17 standards and project documentation guidelines.
