Lib::BSP::Beep Class Reference

Detailed Description

A class to manage the beep functionality using a timer.

The Beep class provides an interface to configure and control a beep sequence using a timer. It allows setting the duration of each beep, the number of beeps, and the pause duration between beeps. The class manages the state transitions and timing using a state machine.

Note
This class inherits from DeviceManager::DeviceBase.
