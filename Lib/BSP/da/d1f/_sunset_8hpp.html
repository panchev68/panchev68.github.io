Detailed Description

Sunset and moon phase calculation utility.

This file defines the Sunset class, which provides functionality for calculating sunrise, sunset, and moon phase information for a given geographic location and date. The class supports various types of sun events (official, nautical, civil, astronomical) and computes times using astronomical algorithms. It is suitable for embedded and real-time applications requiring solar and lunar event data, such as calendar utilities, astronomical tools, and environmental monitoring systems.
