Detailed Description

Frame0x00 class for 64-bit Transmit Request.

Represents a Digi XBee API frame for transmitting serial payload data to a remote device using a 64-bit IEEE address. The frame supports host correlation, broadcast addressing, and transmission options. The response frame is Transmit Status (Frame 0x89).
