Lib::BSP::Display::DriverBase Class Referenceabstract

Detailed Description

Base class for display drivers.

The DriverBase class provides a common interface for initializing the display driver, transmitting blocks of data, handling transfer completion callbacks, and checking the transfer status.

Note
This class is part of the BSP (Board Support Package) for the display module.
