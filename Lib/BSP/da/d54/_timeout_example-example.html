TimeoutExample

Example usage of the Timeout class for managing timeouts in RTOS environments.

Example usage of the Timeout class for managing timeouts in RTOS environments.Demonstrates setting, checking, extending, and converting timeouts using tick and millisecond units.

Example usage:
#include "Timeout.hpp"
#include <iostream>
int main() {
RTOS::Timeout timeout = RTOS::Timeout::fromMilliseconds(500);
timeout.setMilliseconds(1000);
if (!timeout.isExpired()) {
std::cout << "Timeout not expired, remaining ms: " << timeout.getRemainingMs() << std::endl;
}
timeout.extendMs(500);
std::cout << "Extended timeout, new duration ms: " << timeout.getDurationMs() << std::endl;
if (timeout.willExpireWithinMs(2000)) {
std::cout << "Timeout will expire within 2 seconds." << std::endl;
}
timeout.disable();
if (timeout.isDisabled()) {
std::cout << "Timeout is disabled." << std::endl;
}
return 0;
}
