Подробно описание

FirmwareVersion class for firmware versioning and build date retrieval.

Provides version string, sub-version, and build date access. This class extends VersionBase and adds build date extraction using compile-time macros. It ensures type safety, input validation, and robust conversion between version formats for embedded firmware management.
