Lib::BSP::Backlight::Dimmer Class Reference

Detailed Description

Manages the dimming functionality of a Backlight display device.

The Dimmer class provides methods to control the brightness of a Backlight device. It allows setting up minimum and maximum brightness values, starting and stopping the dimming process, and handling tick events for smooth transitions.
