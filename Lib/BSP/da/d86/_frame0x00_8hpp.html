Frame0x00: 64-bit Transmit Request for Digi Core.

Provides the Frame0x00 class for sending serial payload data as an RF packet to a remote device with a 64-bit IEEE address. This class encapsulates Digi XBee API frame construction for transmitting data to a remote device using a 64-bit IEEE address. The response frame for this request is Transmit Status (Frame 0x89). The design ensures type safety, extensibility, and efficient integration with Digi XBee API infrastructure and RTOS environments. It supports flexible payloads and transmission options for embedded and general-purpose applications, facilitating reliable communication and host correlation.
