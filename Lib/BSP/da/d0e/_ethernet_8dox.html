Подробно описание

Ethernet library documentation file.

Provides detailed documentation for the Lib::Ethernet namespace, which contains Ethernet device abstractions, configuration types, status enums, and helpers for embedded and real-time applications. The namespace encapsulates all Ethernet-related types and utilities to ensure consistent configuration, management, and communication across the project. This documentation supports integration with Doxygen-generated reference manuals and improves discoverability and maintainability of Ethernet-related components.
