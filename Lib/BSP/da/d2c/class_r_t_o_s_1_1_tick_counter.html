Detailed Description

TickCounter utility for FreeRTOS tick/time conversions and elapsed/timeout calculations.

Provides static methods for converting between ticks and milliseconds/seconds, calculating elapsed time, and checking timeouts. Ensures type safety and overflow protection.

See also
https://github.com/panchev68-Library/IAR/blob/c7a278579c85aecf574831f2c2d77005a66d21c2/inc/Lib/RTOS/doc/TickCounter.md
