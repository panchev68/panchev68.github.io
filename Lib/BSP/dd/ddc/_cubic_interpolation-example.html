CubicInterpolation

Cubic Spline Interpolation Example

This example demonstrates how to use the CubicInterpolation class for smooth interpolation between data points in an embedded system. It shows adding points, setting boundary conditions, and performing interpolation.

Example usage:

#include "Lib/Math/Interpolation/CubicInterpolation.hpp"
int main() {
// Create interpolation object with natural boundary conditions
CubicInterpolationF interp;
// Add data points
interp.addPoint(0.0f, 0.0f);
interp.addPoint(1.0f, 1.0f);
interp.addPoint(2.0f, 4.0f);
interp.addPoint(3.0f, 9.0f);
// Interpolate at x = 1.5
float result = interp.interpolate(1.5f);
// Get derivative at x = 1.5
float deriv = interp.getDerivative(1.5f);
return 0;
}
