CubicInterpolation

Example demonstrating cubic spline interpolation for sensor data.

Example demonstrating cubic spline interpolation for sensor data.This example shows how to use the CubicInterpolation class to interpolate sensor readings at arbitrary input values. It initializes the interpolation table with sample data points, configures boundary conditions, and performs interpolation and derivative queries.

Example Usage
#include "Lib/Math/Interpolation/CubicInterpolation.hpp"
#include <iostream>
int main() {
using Interp = CubicInterpolationF;
Interp interp;
interp.addPoint(0.0f, 0.0f);
interp.addPoint(1.0f, 2.0f);
interp.addPoint(2.0f, 0.0f);
interp.setBoundaryConditions(Interp::BoundaryType::NATURAL);
float x = 1.5f;
float y = interp.interpolate(x);
float dy = interp.getDerivative(x);
std::cout << "Interpolated value at x = " << x << " is " << y << std::endl;
std::cout << "Derivative at x = " << x << " is " << dy << std::endl;
return 0;
}
