Detailed Description

LocalAtCommand class for Digi XBee modules.

Provides an interface for sending and receiving local AT commands to Digi XBee modules, including queue management, open/close state, and response handling. The LocalAtCommand class encapsulates command transmission, response event handling, and service lifecycle management, supporting robust AT command transactions for Digi XBee modules in embedded applications. The implementation emphasizes type safety, efficient resource management, and extensibility, allowing seamless integration with Digi XBee API infrastructure and RTOS environments.
