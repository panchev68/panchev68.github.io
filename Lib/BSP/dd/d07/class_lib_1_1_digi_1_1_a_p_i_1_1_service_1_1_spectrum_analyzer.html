Detailed Description

SpectrumAnalyzer service class.

The SpectrumAnalyzer class performs periodic energy scans and provides current, average, and maximum spectrum data arrays. It uses Digi's LocalAtCommand interface for communication and supports event-driven updates. The class is designed for embedded Digi modules, enabling robust spectrum analysis, error handling, and maintainable, type-safe service development.
