◆ setup()

virtual void Lib::BSP::Backlight::setup ( )
inlinevirtual

Sets up the backlight configuration and starts the PWM and dimmer timer.

This function initializes the timer capture/compare register (CCR) based on the specified timer channel in the configuration. It starts the PWM signal generation using the HAL_TIM_PWM_Start function. Additionally, it sets up and starts a dimmer timer to handle tick events for fading the backlight.

Note
If the timer channel is not one of TIM_CHANNEL_1, TIM_CHANNEL_2, TIM_CHANNEL_3, or TIM_CHANNEL_4, the function will assert and set the CCR pointer to nullptr.
If the PWM start or dimmer timer creation/start fails, the function will assert.
