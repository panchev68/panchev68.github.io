RTOS_Kernel

Example usage of RTOS::Kernel for task delay and tick count.

Demonstrates delaying a task and printing the current tick count using standard output.

Example usage:
#include <iostream>
#include "Lib/RTOS/Kernel.hpp"
int main() {
std::cout << "Delaying for 1000 ms..." << std::endl;
RTOS::delay(1000);
auto ticks = RTOS::getTicks();
std::cout << "Current tick count: " << ticks << std::endl;
return 0;
}
Kernel.hpp
Kernel abstraction for FreeRTOS task, tick, and memory management.

This example delays the current task for 1000 milliseconds and prints the tick count after the delay.

Expected output:
