SPI interface header file.

Provides declarations for SPI-related functionalities, including type-safe abstractions and integration points for STM32 HAL SPI operations. This file is designed to support robust, maintainable, and extensible SPI communication within embedded applications, facilitating seamless adaptation to evolving hardware requirements and system needs.
