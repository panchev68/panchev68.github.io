Подробно описание

Detects rising and falling edges in a sequence of data.

This file provides the FindPeaks class for detecting rising and falling edges in a sequence of integer data. The implementation uses a compact sliding window and peak weight calculation for efficient edge detection in embedded systems. The class maintains internal state to track peak values and their positions, enabling reliable detection of transitions in noisy or real-time data streams.
