◆ ILI9341

using Lib::BSP::Display::ILI9341 = Driver<Controller::ILI9341>

Alias for the Driver class template instantiated with the Controller::ILI9341 type.

This typedef provides a convenient alias for the Driver class template when using the ILI9341 display controller. It simplifies the code by avoiding the need to repeatedly specify the template parameter.

◆ ST7789

using Lib::BSP::Display::ST7789 = Driver<Controller::ST7789>

Alias for the Driver class template instantiated with Controller::ST7789.

This typedef provides a convenient alias for the Driver class template specialized with the ST7789 controller. It simplifies the usage of the driver for the ST7789 display controller by avoiding the need to specify the template parameter explicitly.
