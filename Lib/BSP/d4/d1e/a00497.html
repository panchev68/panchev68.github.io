Lib::BSP::XPT2046::Version1::Driver::Position Struct Reference

Detailed Description

Structure to represent the position on the touchscreen.

This structure holds the X and Y coordinates of a point on the touchscreen. The default values for both coordinates are set to -1, indicating an invalid or uninitialized position.
