Detailed Description

MessageBuffer class for FreeRTOS message buffer management.

The MessageBuffer class encapsulates the creation, destruction, and usage of FreeRTOS message buffers, providing a type-safe and RAII-compliant C++17 interface for both task and ISR contexts.
