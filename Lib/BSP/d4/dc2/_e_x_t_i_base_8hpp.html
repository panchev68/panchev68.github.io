Detailed Description

Base class for GPIO external interrupt (EXTI) management.

This file declares the EXTIBase class, which provides an abstract interface for GPIO external interrupt event
management and callback binding. The EXTIBase class enables user-defined callbacks for EXTI events, supports
singleton device management, and provides event dispatching for GPIO pins. It is designed to be used as a base
for GPIO input types that require EXTI event handling and integrates with the DeviceManagerBase infrastructure.
