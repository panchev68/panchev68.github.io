Lib::BSP::FMEM::SPI::DriverBase Class Referenceabstract

Detailed Description

Base class for SPI Flash Memory drivers.

This class provides a base implementation for SPI Flash Memory drivers. It includes methods for opening and closing the device, reading and writing data, and erasing the chip or sectors. It also provides information about the memory capacity, page size, and other parameters of the device.
