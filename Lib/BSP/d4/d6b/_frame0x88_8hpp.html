Local AT Command Response for Digi Core.

Provides the Frame0x88 class for parsing and representing local AT command responses from Digi XBee modules. This class encapsulates Digi XBee API frame construction for local AT command responses, supporting type safety, extensibility, and efficient integration with Digi XBee API infrastructure and RTOS environments. It enables reliable communication and host correlation for embedded and general-purpose applications.
