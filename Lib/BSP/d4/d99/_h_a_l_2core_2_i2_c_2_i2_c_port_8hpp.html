Detailed Description

I2CPort class definition and I2C peripheral abstraction.

This file defines the I2CPort class, a type-safe, event-driven wrapper for the STM32 HAL I2C peripheral. The class provides callback registration, memory and device operations, error and status handling, and integrates with RTOS event groups for asynchronous operation. It supports device manager registration for centralized management of I2C peripherals and enables robust, non-blocking I2C communication in embedded applications.
