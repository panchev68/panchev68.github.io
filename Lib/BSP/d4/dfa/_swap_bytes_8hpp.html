Подробно описание

Provides utility functions for byte swapping.

This file defines functions for swapping the byte order of various data types, including integral types and raw memory buffers. It provides optimized byte swapping operations for different integer types and memory buffers, essential for handling endianness conversion in embedded systems and network communications. The implementation ensures type safety, efficient conversion, and supports both value and pointer-based swapping for all standard fixed-width integer types.
