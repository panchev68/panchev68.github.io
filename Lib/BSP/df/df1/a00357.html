Lib::BSP::EncoderTimer Class Reference

Detailed Description

A class representing an encoder with a button using FreeRTOS.

This class provides an interface to read events from an encoder with a button using FreeRTOS. It uses a timer to read the encoder position and a button to read the button state.

enum class Lib::BSP::EncoderTimer::EventCode
strong

Enumeration of event codes.

This enumeration defines the possible event codes that can be generated by the encoder and button. The event codes include button push, button pull, button hold, clockwise rotation, and counter-clockwise rotation.

Note
The NONE event code is used to indicate that no event is available.
