Detailed Description

Module class for Digi XBee modules.

Manages Digi XBee module communication, initialization, hardware reset, and message sending with event hooks for message send/receive. The class provides event-driven hooks for message transmission and reception, encapsulates low-level communication, and supports robust lifecycle management for Digi XBee modules in embedded and general-purpose applications.
