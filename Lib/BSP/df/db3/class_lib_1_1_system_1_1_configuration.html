Подробно описание

Manages configuration data stored in OTP memory.

Provides methods to read and write configuration data to a specific address in OTP memory. Ensures data validity using a mask and provides methods to check if the configuration is empty or valid. The class encapsulates address calculation, bitset conversion, and OTP access logic for robust configuration management in embedded systems.
