Timer class specialization for ENCODER mode.

Specializes the Timer template for ENCODER mode, providing efficient management and value change hooks. This file defines the Timer class specialization for ENCODER mode, supporting STM32 TIM peripherals in encoder applications. It provides efficient management, value change hooks, and safe resource handling. The implementation uses modern C++17 features and standard libraries for safety, maintainability, and type safety. The class is intended for embedded applications, ensuring compatibility with STM32 HAL and project documentation guidelines.
