Подробно описание

C++ RAII wrapper for FreeRTOS mutex.

Manages lifecycle, safe creation, destruction, locking, unlocking, and status queries. Supports both normal and recursive mutex types for STM32/FreeRTOS systems.

Примери
RTOS::Mutex.
