Detailed Description

Mutex class for synchronization.

Provides a wrapper around FreeRTOS semaphores for mutex functionality, supporting normal and recursive mutexes. Ensures thread-safe access to shared resources.

Examples
RTOS::Mutex, and RTOS::MutexGuard.
