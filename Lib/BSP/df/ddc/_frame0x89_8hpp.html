Detailed Description

Transmit Status Response for Digi Core.

Provides the Frame0x89 class for parsing and representing transmit status responses from Digi XBee modules. This class encapsulates Digi XBee API frame construction for transmit status responses, supporting type safety, extensibility, and efficient integration with Digi XBee API infrastructure and RTOS environments. It enables reliable communication and host correlation for embedded and general-purpose applications.
