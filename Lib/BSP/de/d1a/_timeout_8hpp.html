Подробно описание

Timeout utility for managing timeouts and expiration logic in RTOS environments.

This file defines the Timeout class, which encapsulates timeout duration, start tick, and expiration logic for use with FreeRTOS tick counters. It provides APIs for setting, extending, checking, and converting timeouts, supporting both tick and millisecond units. The Timeout class is designed for safe, overflow-protected timeout management in embedded systems, supporting both immediate and disabled timeouts, and providing conversion helpers.
