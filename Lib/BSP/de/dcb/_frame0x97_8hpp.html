Frame 0x97: Remote AT Command Response for Digi Core.

Provides the Frame0x97 class for parsing and representing remote AT command responses from Digi XBee modules. This file defines the Frame0x97 type, which extracts frame ID, destination addresses, AT command, status, and parameters from Digi XBee API frames. The class is designed for embedded use, ensuring type safety and efficient parsing of binary data. It uses fixed-width integer types and standard containers for reliability and maintainability.
