Detailed Description

FirmwareVersion class for firmware versioning and build date retrieval.

Represents firmware version and build date. Inherits from VersionBase and adds build date extraction using compile-time macros. Provides conversion between version formats and build date retrieval for embedded firmware management. Ensures type safety, input validation, and robust conversion logic.
