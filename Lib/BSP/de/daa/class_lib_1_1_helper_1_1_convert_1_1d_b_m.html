Detailed Description

Utility class for dBM value conversions.

The dBM class provides static methods for converting dBM values to percentages. It supports both linear conversion for signed 8-bit and 16-bit dBM values, as well as lookup table-based conversion for digital dBM values. The class is designed for use in embedded systems and applications requiring efficient and type-safe dBM to percentage conversions.
