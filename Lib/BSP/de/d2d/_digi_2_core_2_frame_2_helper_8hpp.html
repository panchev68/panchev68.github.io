Подробно описание

Helper utilities for Digi Core Frame operations.

Provides conversion and vector manipulation utilities for Digi Core Frame, including byte swapping, serialization, and deserialization helpers. The Helper class enables efficient and type-safe manipulation of Digi XBee API frame data, supporting embedded and general-purpose applications. The design emphasizes correctness, extensibility, and resource efficiency, allowing seamless integration with Digi XBee API infrastructure and RTOS environments.
