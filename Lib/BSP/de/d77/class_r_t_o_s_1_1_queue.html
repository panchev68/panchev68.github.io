Detailed Description

template<typename T_ELEMENT_TYPE, size_t QUEUE_LENGTH = 1>
class RTOS::Queue< T_ELEMENT_TYPE, QUEUE_LENGTH >

RAII wrapper for FreeRTOS queues.

Provides type-safe operations for queue management, including creation, destruction, sending, receiving, and querying. Supports both dynamic and static allocation.

Template Parameters
T_ELEMENT_TYPEElement type, must be trivially copyable.
QUEUE_LENGTHMaximum number of elements in the queue.
Examples
RTOS::Queue.
