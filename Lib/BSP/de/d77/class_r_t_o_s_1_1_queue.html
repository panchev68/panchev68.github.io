Подробно описание

template<typename T, size_t QUEUE_LENGTH = 1>
class RTOS::Queue< T, QUEUE_LENGTH >

RTOS queue wrapper class.

Provides a type-safe, fixed-capacity queue abstraction over FreeRTOS queues, supporting both dynamic and static allocation, ISR-safe operations, and utility methods for queue management.

Шаблона Параметри
TElement type (must be trivially copyable).
QUEUE_LENGTHNumber of elements in the queue.
