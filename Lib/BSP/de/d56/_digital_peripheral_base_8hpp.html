Detailed Description

DigitalPeripheralBase class and related types.

This file defines the DigitalPeripheralBase class, which provides a base for digital peripheral management with SPI-based input and output handling, including configuration, update modes, and event callbacks. It includes nested types for configuration, input/output port abstraction, and SPI transfer logic. The class is designed for embedded systems using FreeRTOS and IAR toolchain.
