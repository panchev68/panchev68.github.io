Lib::System::System

Example usage of System class.

Demonstrates how to create and use the System class for managing system configuration, reset state, production and sale dates, serial number, and version information.

Example usage:

#include "System.hpp"
int main() {
Lib::System::System system(
"FW_1.2.3", 123,
"HW_A1", 456
);
system.restart();
return 0;
}
