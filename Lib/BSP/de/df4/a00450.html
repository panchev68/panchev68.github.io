Lib::BSP::W25Q128JVSIQ Class Reference

Detailed Description

W25Q128JVSIQ flash memory driver class.

This class provides methods to initialize, read from, write to, and erase the W25Q128JVSIQ flash memory chip. It also includes functions to manage power states and retrieve chip information.

const Info & Lib::BSP::W25Q128JVSIQ::getInfo ( )
inline

Retrieves the chip information for the W25Q128JVSIQ flash memory.

This function reads the JEDEC ID from the chip and populates the Info structure with the chip manufacturer and memory capacity details. If the memory density is recognized (0x18 for 128Mb), it calculates and sets the memory capacity, number of pages, page size, sector size, sector count, and pages per sector. If the memory density is not supported, it asserts an error.

Returns
const W25Q128JVSIQ::Info& A reference to the populated Info structure.
