Detailed Description

Touch screen driver class for XPT2046.

This file contains the implementation of the touch screen driver for the XPT2046 controller. The driver supports configuration and initialization of the touch screen, reading touch positions, and setting the screen rotation.

Note
In the new driver version, the touch screen push pin (nirq) must be configured as EXTI (falling edge) and interrupt enabled.
