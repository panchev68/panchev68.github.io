Detailed Description

Represents an error with states and metadata.

The Error class manages error states, associates errors with groups, and provides unique identification. It supports state transitions, group assignment, and metadata retrieval for error handling in embedded systems. The class enables activation, clearing, resetting, enabling, disabling, and state change tracking, allowing integration with error dispatchers and observers for comprehensive error management.
