Detailed Description

ADC DMA Port implementation.

This file defines the specialization of the Lib::HAL::Port template for DriverMode::DMA, providing DMA-based ADC port functionality. It implements the interface for ADC operations using DMA, including setup, release, start, stop, and read methods. The class is derived from AdcTransparentPortBase and is intended for use in embedded systems with FreeRTOS and TouchGFX.
