Lib::BSP::EEPROM::I2C::DriverBase Class Referenceabstract

Detailed Description

Base class for I2C EEPROM drivers.

This class provides a base implementation for I2C EEPROM drivers. It includes methods for opening and closing the device, reading and writing data, and retrieving the size of the EEPROM chip.
