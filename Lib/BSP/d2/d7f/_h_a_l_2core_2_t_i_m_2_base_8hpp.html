Detailed Description

STM32 Timer (TIM) base class and device manager abstraction.

Provides the Base class for STM32 TIM peripherals, supporting callback registration, device management, and event handling. Integrates with DeviceManagerBase for robust, type-safe device tracking and callback dispatch. This file defines the base class for STM32 TIM peripherals, supporting callback registration, device management, and event handling. It integrates with DeviceManagerBase for robust, type-safe device tracking and callback dispatch. The implementation uses modern C++17 features and standard libraries for safety, maintainability, and type safety. The class is intended for embedded applications, ensuring compatibility with STM32 HAL and project documentation guidelines.
