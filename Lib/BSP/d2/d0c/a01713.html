Detailed Description

USB serial port interface.

Provides methods for USB serial communication, including open/close, send, reset, baud rate management, and event callbacks. Encapsulates UART port management, USB reset pin control, RTOS task handling, and message buffering for efficient and reliable USB serial data transfer.
