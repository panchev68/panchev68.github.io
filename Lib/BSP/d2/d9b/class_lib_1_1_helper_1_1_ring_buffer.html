Detailed Description

template<typename T, size_t T_SIZE>
class Lib::Helper::RingBuffer< T, T_SIZE >

Fixed-size ring buffer for FIFO storage.

The RingBuffer class implements a circular buffer of fixed capacity for storing elements of type T. It provides efficient push and pop operations, as well as methods to check if the buffer is full or empty, retrieve the number of stored elements, and clear the buffer. This class is suitable for embedded and real-time systems where bounded, non-blocking data queues are required.
