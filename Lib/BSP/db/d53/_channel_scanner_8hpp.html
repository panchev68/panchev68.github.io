Detailed Description

ChannelScanner service class.

Provides functionality to scan channels and monitor connection quality for embedded Digi modules. The ChannelScanner class manages channel scanning, connection quality updates, error handling, and task management using FreeRTOS. It enables robust monitoring of channel quality and supports extensibility for custom scanning logic and diagnostics. Designed for embedded systems, it ensures reliable channel scanning, error reporting, and event-driven updates, supporting maintainable and type-safe service development.
