Detailed Description

Frame0x8a.hpp: Modem Status Response for Digi Core.

Provides the Frame0x8a class for parsing and representing modem status responses from Digi XBee modules. This class encapsulates Digi XBee API frame construction for modem status responses, supporting type safety, extensibility, and efficient integration with Digi XBee API infrastructure and RTOS environments. It enables reliable communication and host correlation for embedded and general-purpose applications.
