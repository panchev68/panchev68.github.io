Detailed Description

ADC channel configuration and management.

This file defines the AdcChannel class for configuring and managing ADC channels using the STM32 HAL library. The class provides a type-safe interface for ADC channel setup through a nested Config class, enabling structured initialization and encapsulation of channel parameters for embedded applications.
