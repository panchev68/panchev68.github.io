Подробно описание

Implements cubic spline interpolation for STM32 embedded systems.

This file defines the CubicInterpolation class template, which provides efficient cubic spline interpolation between data points for embedded applications. The implementation supports both natural and clamped boundary conditions, using a tridiagonal matrix algorithm for spline coefficient calculation. Optimized for memory usage and computation speed, it is suitable for signal processing, sensor fusion, and control systems.
