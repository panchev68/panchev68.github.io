Detailed Description

template<typename T = double>
class Lib::Math::Filter::Kalman< T >

Kalman filter for generic numeric types. The Kalman class implements a simple Kalman filter for smoothing and estimating sensor values. It supports configuration of process noise, sensor noise, and initial error, and provides methods for filtering, resetting, and parameter adjustment. Internal calculations use double precision for improved accuracy.
