Detailed Description

Hexadecimal Conversion Utilities.

This file defines template specializations for converting various integer types to hexadecimal and BCD representations. The Hex class provides type-safe conversion methods for 8-bit, 16-bit, and 64-bit unsigned integers, supporting conversion to hexadecimal strings, BCD formats, and BCD strings with optional leading zero suppression. The implementation is suitable for embedded systems and applications requiring efficient and robust hexadecimal and BCD conversions.
