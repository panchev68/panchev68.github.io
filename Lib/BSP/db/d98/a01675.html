Detailed Description

RTOS stream buffer wrapper class. The StreamBuffer class encapsulates a FreeRTOS stream buffer handle and provides methods for buffer creation, destruction, data transmission, reception, and status queries. It ensures safe resource management and offers utility functions for buffer utilization and state.
