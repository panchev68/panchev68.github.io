Detailed Description

Utility class for ASCII character operations.

The Ascii class provides static methods for checking ASCII character properties, such as printability and validity, as well as conversions between ASCII characters and their numeric values. It supports detection of decimal and hexadecimal digits and conversion between hexadecimal ASCII characters and their numeric representations. The class is designed for use in embedded systems and applications requiring efficient and type-safe ASCII character manipulation.
