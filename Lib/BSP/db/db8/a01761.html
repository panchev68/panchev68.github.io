Kalman

Example usage of the Kalman filter for smoothing sensor data. This example demonstrates how to configure and use the Kalman filter to process noisy sensor readings.

Example Usage
#include "Lib/Math/Filter/Kalman.hpp"
#include <iostream>
int main() {
Lib::Math::Filter::Kalman<float>::Config config;
config.processNoise = 0.0001f;
config.sensorNoise = 0.01f;
config.estimatedError = 1.0f;
config.value = 0.0f;
Lib::Math::Filter::Kalman<float> filter(config);
float noisyData[] = { 1.0f, 1.2f, 0.9f, 1.1f, 1.05f };
for (auto value : noisyData) {
float filtered = filter(value);
std::cout << "Input: " << value << ", Filtered: " << filtered << std::endl;
}
return 0;
}
