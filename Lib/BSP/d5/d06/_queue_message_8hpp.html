Подробно описание

QueueMessage class for Digi XBee communication.

Provides a fixed-size message buffer with convenient conversion to std::vector. This class enables efficient storage and transfer of message data in embedded Digi XBee applications, supporting robust communication and interoperability with standard containers. The design ensures type safety, buffer bounds enforcement, and easy integration with other system components.
