Detailed Description

Interface for SPI port communication.

The SPIPort class provides an abstract interface for SPI (Serial Peripheral Interface) communication. It defines a set of pure virtual functions that must be implemented by any derived class to handle SPI communication, including opening and closing the port, sending and receiving data, and performing simultaneous send/receive operations.
