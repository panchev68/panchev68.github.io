Detailed Description

Base class for Digi API services.

The ServiceBase class provides a common interface and lifecycle management for all service classes in the Digi API. It supports open/close state management, update cycles, and integration with service managers. The class is designed for embedded Digi modules, enabling robust service management, error handling, and maintainable, type-safe service development.
