Detailed Description

EventGroup class and event bits utilities for RTOS event synchronization.

This file provides the EventGroup class and related types for managing event groups in FreeRTOS. It supports bitwise event signaling, waiting, and synchronization with configurable bit width depending on the FreeRTOS tick type. The class encapsulates creation, destruction, and manipulation of event bits, including ISR-safe operations and timeout-based waiting. It is designed for use in embedded systems with FreeRTOS and provides a type-safe, C++-style interface.
