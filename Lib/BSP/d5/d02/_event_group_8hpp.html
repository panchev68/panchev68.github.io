Detailed Description

EventGroup class for FreeRTOS event group management.

This file defines the EventGroup class, which provides a type-safe, modern C++17 wrapper for FreeRTOS event groups. It enables bitwise event signaling, waiting, and synchronization with strong type safety, compile-time validation, and convenient utility methods for embedded and real-time systems.
