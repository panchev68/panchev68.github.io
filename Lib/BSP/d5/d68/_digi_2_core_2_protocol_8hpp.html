Подробно описание

Protocol class for Digi XBee API frames.

Provides parsing, validation, and message processing for Digi XBee API frames, including frame construction, checksum verification, and event dispatching. The Protocol class encapsulates robust frame handling for Digi XBee modules, supporting efficient and type-safe communication in embedded applications. The implementation emphasizes correctness, resource efficiency, and extensibility, allowing seamless integration with Digi XBee API infrastructure and RTOS environments.
