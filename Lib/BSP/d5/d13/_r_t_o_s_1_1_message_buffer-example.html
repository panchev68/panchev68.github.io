RTOS::MessageBuffer

Example usage of RTOS::MessageBuffer

This example demonstrates how to create, send, and receive messages using the RTOS::MessageBuffer class.

Example usage:

#include "Lib/RTOS/MessageBuffer.hpp"
int main() {
RTOS::MessageBuffer buffer;
constexpr size_t BUFFER_SIZE = 128;
if (!buffer.create(BUFFER_SIZE)) {
// Handle error
return -1;
}
const char message[] = "Hello, FreeRTOS!";
buffer.send(message, sizeof(message), 10);
char receiveBuffer[32] = {};
buffer.receive(receiveBuffer, sizeof(receiveBuffer), 10);
buffer.destroy();
return 0;
}
