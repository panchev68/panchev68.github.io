Detailed Description

Base class for Digi API services.

Provides a common interface and lifecycle management for all service classes in the Digi API. This class defines the interface and lifecycle management for all service classes, ensuring consistent behavior and extensibility for Digi API service implementations. It supports open/close state management, update cycles, and integration with service managers. Designed for embedded Digi modules, enabling robust service management, error handling, and maintainable, type-safe service development.
