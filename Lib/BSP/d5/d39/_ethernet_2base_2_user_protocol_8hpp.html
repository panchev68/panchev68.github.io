Подробно описание

Ethernet user protocol class.

Provides a class for representing and parsing Ethernet user protocol messages. Supports message construction, parsing, and callback handling for commands and payloads. Ensures type safety, bounds checking, and input validation for embedded networking applications. Includes support for custom callbacks and error handling.
