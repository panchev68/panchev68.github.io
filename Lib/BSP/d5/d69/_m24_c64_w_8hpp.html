Подробно описание

EEPROM M24C64W I2C driver class for STM32 BSP.

This file provides the M24C64W EEPROM I2C driver for STM32 BSP, enabling high-level access to the EEPROM device with support for page and chip size configuration. It implements open, close, read, write, and chip size query operations using 16-bit addressing.
