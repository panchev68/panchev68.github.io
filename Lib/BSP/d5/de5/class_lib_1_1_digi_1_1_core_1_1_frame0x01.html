Detailed Description

Frame0x01 class for 16-bit Transmit Request.

Represents a Digi XBee API frame for transmitting serial payload data to a remote device using a 16-bit network address. The frame supports host correlation, broadcast addressing, and transmission options. The response frame is Transmit Status (Frame 0x89). Deprecated for legacy use; use Frame 0x10 for new designs.
