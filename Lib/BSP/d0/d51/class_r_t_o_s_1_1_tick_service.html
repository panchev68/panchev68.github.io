Подробно описание

TickService provides periodic tick event dispatching to registered clients.

TickService manages a list of ClientBase-derived clients, dispatching tick events at a configurable interval. Clients are registered/unregistered automatically via their constructors/destructors. The service runs in a dedicated task and ensures thread safety using a mutex. Supports dynamic client management and priority control.
