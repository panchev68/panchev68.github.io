Provides a utility class for converting percentages to dBM values.

This file defines the Percent class, which offers a static method for converting percentage values (0-100) to dBM values. The class is designed for embedded systems and applications requiring efficient percentage to dBM conversions. The implementation ensures type safety and input validation, making it suitable for robust conversion logic in performance-critical environments.
