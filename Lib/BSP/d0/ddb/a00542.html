Lib::BSP::Voltmeter::Base Class Reference

Detailed Description

The base class for voltmeter devices.

This class provides the base functionality for voltmeter devices, including reading voltage values from an ADC and converting them to a voltage value. The class is a DeviceBase and AdcPort, and it also includes an interpolator and integrator for processing the ADC values.
