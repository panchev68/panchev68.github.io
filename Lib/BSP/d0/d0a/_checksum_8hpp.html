Подробно описание

Checksum calculation for Digi API frames.

Provides the Checksum class for calculating and representing Digi API message checksums. This file defines the Checksum type, which computes the checksum for Digi XBee API frames using either an ApiFrameBuffer or a raw data pointer. The class ensures type safety and efficient calculation, supporting robust embedded communication. It uses fixed-width integer types and validates input sizes for reliability and maintainability.
