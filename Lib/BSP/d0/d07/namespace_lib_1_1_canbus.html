Detailed Description

Namespace for CAN bus library classes and functions.

The Lib::Canbus namespace contains all abstractions, helpers, and types related to the CAN bus library. It provides APIs for CAN message handling, node management, diagnostics, and event-driven communication. All CAN bus-related code should reside within this namespace to ensure modularity and maintainability.
