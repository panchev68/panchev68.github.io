Frame0x8b.hpp: Extended Transmit Status Response for Digi Core.

Provides the Frame0x8b class for parsing and representing extended transmit status responses from Digi XBee modules. This class encapsulates Digi XBee API frame construction for extended transmit status responses, supporting type safety, extensibility, and efficient integration with Digi XBee API infrastructure and RTOS environments. It enables reliable communication and host correlation for embedded and general-purpose applications.
