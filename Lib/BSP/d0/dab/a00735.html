Detailed Description

DeviceManager class for centralized device registration and tick event handling. The DeviceManager class manages a list of device objects derived from DeviceBase. It provides automatic registration and deregistration of devices, setup routines, and periodic tick event dispatching via an RTOS timer. The singleton pattern ensures a single instance manages all devices in the BSP layer. DeviceBase is an abstract base class for all managed devices.
