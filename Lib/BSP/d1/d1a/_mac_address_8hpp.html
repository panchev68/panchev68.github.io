Detailed Description

MAC Address class.

This file defines the MacAddress class for representing and manipulating MAC addresses in embedded networking applications. It provides construction from arrays, automatic generation based on device serial number, conversion to and from string representations, validation, and access methods. The class ensures type safety, bounds checking, and input validation.
