Подробно описание

Jagged array container for variable-length rows.

This file defines the JaggedArrayBase template class, which provides a container for managing jagged arrays (arrays of arrays with variable-length rows). The class supports adding, removing, and accessing elements, as well as clearing and retrieving the underlying storage. It is suitable for embedded and real-time applications where flexible, non-rectangular data structures are required. The implementation uses std::vector for dynamic storage and ensures type safety and bounded access.
