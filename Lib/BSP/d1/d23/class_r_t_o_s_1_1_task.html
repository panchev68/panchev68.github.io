Detailed Description

Task class for FreeRTOS thread management.

Provides a C++17 abstraction for FreeRTOS task creation, control, and notification. Supports safe resource management, priority and stack configuration, and notification APIs.

See also
https://github.com/panchev68-Library/IAR/blob/3.03/inc/Lib/RTOS/doc/Task.md
