Detailed Description

Compact and efficient date/time utility classes.

Provides classes for representing, manipulating, and formatting date and time values, including packing and unpacking to 32-bit integers. The implementation ensures type safety, correctness, and efficient resource usage for embedded and general-purpose applications. It supports validation, formatting, and day-of-week calculation, enabling easy integration with various systems requiring compact date/time representation.
