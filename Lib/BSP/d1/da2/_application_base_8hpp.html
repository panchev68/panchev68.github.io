Detailed Description

ApplicationBase class for Digi API applications.

Provides a common interface and lifecycle management for Digi API client/server applications, including service registration, module management, and message handling. The ApplicationBase class encapsulates service initialization, connection management, and message processing, supporting robust client and server implementations for Digi XBee modules in embedded and general-purpose applications. The implementation emphasizes type safety, efficient resource management, and extensibility, allowing seamless integration with Digi XBee API infrastructure and RTOS environments. It supports event-driven updates, robust error handling, and maintainable, type-safe application development for Digi modules.
