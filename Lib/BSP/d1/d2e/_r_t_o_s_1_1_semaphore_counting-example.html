RTOS::SemaphoreCounting

Example usage of SemaphoreCounting for resource pooling.

This example demonstrates how to create and use a counting semaphore for synchronizing access to a shared resource pool in a FreeRTOS-based STM32 embedded application. The semaphore is initialized with a maximum count of 5 and an initial count of 3, representing three available resources out of five.

Example usage:
#include "Lib/RTOS/SemaphoreCounting.hpp"
void taskFunction() {
RTOS::SemaphoreCounting semaphore(5, 3);
if (semaphore.create()) {
// Try to take a resource
if (semaphore.take(100)) {
// Access shared resource
// ...
semaphore.give(); // Release resource
}
}
}
