RTOS::SemaphoreCounting

Example usage of SemaphoreCounting.

Demonstrates how to create and use a counting semaphore in FreeRTOS.

Example usage:

#include "Lib/RTOS/SemaphoreCounting.hpp"
void taskFunction(void* params) {
RTOS::SemaphoreCounting semaphore(5, 2);
if (semaphore.create()) {
// Use semaphore in task
if (!semaphore.isEmpty()) {
// Take semaphore
}
if (!semaphore.isAtMaxCount()) {
// Give semaphore
}
}
}
