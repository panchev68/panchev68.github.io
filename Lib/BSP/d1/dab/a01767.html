Pid

Example demonstrating the use of the Pid class for controlling a process variable. This example shows how to configure a PID controller, set its parameters, and compute the control output for a simulated process variable. The controller is switched to automatic mode, and the output is printed for several iterations to illustrate its response to a changing input.

Example Usage
#include "Lib/Math/Pid.hpp"
#include <iostream>
int main() {
Lib::Math::PidF::Parameters params(2.0f, 0.5f, 1.0f, 0.1f, -10.0f, 10.0f);
Lib::Math::PidF pid(params);
pid.setMode(true); // Enable automatic mode
float setpoint = 5.0f;
float input = 0.0f;
for (int i = 0; i < 10; ++i) {
float output = pid.compute(setpoint, input);
std::cout << "Step " << i << ": Input=" << input << " Output=" << output << std::endl;
input += output * 0.1f; // Simulate process response
}
return 0;
}
