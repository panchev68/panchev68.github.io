Detailed Description

QueryBase.hpp: Base class for Digi Core query frames.

Provides the QueryBase class, which serves as the base for all Digi Core send requests. This class defines a virtual conversion operator to Frame::Data for use with Digi API frame builders. It ensures type safety, extensibility, and efficient integration with Digi XBee API infrastructure and RTOS environments. The design supports flexible payloads and transmission options for embedded and general-purpose applications, facilitating reliable communication and host correlation.
