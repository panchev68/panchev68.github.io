Detailed Description

Time zone representation and conversion utilities.

This file defines the TimeZone class, which provides comprehensive functionality for representing time zones, handling offsets, and converting local times to UTC minutes. It includes support for daylight saving time rules, offset calculations, and local-to-UTC time conversions, making it suitable for embedded systems and applications requiring robust time zone management.
