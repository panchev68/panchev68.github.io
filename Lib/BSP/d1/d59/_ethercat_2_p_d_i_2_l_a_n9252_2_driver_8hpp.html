Подробно описание

LAN9252 EtherCAT slave controller base class declaration.

Provides initialization, reset, and communication methods for the LAN9252 EtherCAT slave controller chip. This file defines the Base class, which encapsulates SPI and GPIO management, chip initialization, reset, and register access for the LAN9252 EtherCAT slave controller. The class supports CSR and PRAM register operations, burst mode transfers, and chip state management for robust EtherCAT communication.
