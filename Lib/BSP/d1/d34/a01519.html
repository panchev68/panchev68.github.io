Detailed Description

template<class T>
class Lib::Helper::JaggedArrayBase< T >

Jagged array base container for variable-length rows.

The JaggedArrayBase class template provides a container for managing jagged arrays, allowing dynamic addition, removal, and access of elements. It uses std::vector for storage and supports safe access and modification of elements. The class is type-safe and suitable for embedded systems and real-time applications requiring flexible data structures.

Template Parameters
TType of elements stored in the jagged array.
