Detailed Description

template<typename T_VALUE, typename T_INPUT = T_VALUE>
class Lib::Math::CubicInterpolation< T_VALUE, T_INPUT >

Cubic spline interpolation class for embedded applications.

The CubicInterpolation class template provides smooth cubic interpolation between data points using natural or clamped boundary conditions. It uses a tridiagonal matrix solution for efficient spline coefficient calculation and supports up to MAX_POINTS data points. All internal state variables are initialized and updated according to standard cubic spline equations.

Template Parameters
T_VALUENumeric type for values (float or double).
T_INPUTNumeric type for input coordinates (typically same as T_VALUE).
