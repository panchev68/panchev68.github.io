Подробно описание

template<typename T = float>
class Lib::Math::Pid< T >

PID controller for control systems.

Implements a discrete-time PID controller with anti-windup, output clamping, and derivative kick prevention. Supports both automatic and manual modes with bumpless transfer. The controller is suitable for embedded and real-time applications, providing safe parameter validation and bounded control outputs.

Шаблона Параметри
TNumeric type for calculations (typically float or double).
