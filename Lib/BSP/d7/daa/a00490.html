Lib::BSP::Termometer::DTH::Driver< Type::DTH22 > Class Reference

Detailed Description

DTH22 Driver class.

This class provides functionalities to interface with the DTH22 temperature and humidity sensor. It includes methods to read the temperature and humidity values from the sensor and manage the sensor state.

Template Parameters
TypeThe type of the sensor (DTH22).
