RTOS::TickService

Example usage of TickService and ClientBase.

This example demonstrates how to implement a tick event client by inheriting from RTOS::TickService::ClientBase and starting the tick service with a custom interval.

Example usage:

#include "Lib/RTOS/TickService.hpp"
class MyTickClient : public RTOS::TickService::ClientBase {
public:
void handleTickEvent() override {
// Handle tick event here.
}
};
int main() {
MyTickClient client;
RTOS::TickService::getInstance().start(10); // Start with 10 ms interval.
// Application main loop...
return 0;
}
