Detailed Description

Frame0x08 class for Local AT Command Request.

Represents a Digi XBee API frame for querying or setting AT command parameters on the local device. Any parameter set with this frame applies immediately. For queued changes, use Frame 0x09. Querying a parameter returns its current value in the response frame (0x88).
