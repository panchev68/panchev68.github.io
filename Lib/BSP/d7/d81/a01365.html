Detailed Description

template<size_t T_SIZE>
class Lib::Helper::BufferStatic< T_SIZE >

Manages a static memory buffer.

The BufferStatic class provides a fixed-size buffer using stack allocation, suitable for performance-critical or memory-constrained environments. It inherits from Buffer and initializes the buffer with a statically allocated array.
