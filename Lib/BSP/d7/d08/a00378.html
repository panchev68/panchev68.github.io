Lib::BSP::Ethercat::PDI::LAN9252::Base Class Reference

Detailed Description

Base class for the LAN9252 EtherCAT slave controller.

This class provides methods for initializing, resetting, and communicating with the LAN9252 chip. It includes functions for reading and writing registers, handling PRAM and CSR operations, and managing the state of the chip.
