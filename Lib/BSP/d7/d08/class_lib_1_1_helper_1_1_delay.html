Detailed Description

Utility class for microsecond delays using DWT cycle counter.

This class provides static methods to perform busy-wait delays in microseconds and nanoseconds. It is non-instantiable and uses the ARM DWT for cycle-accurate timing. The delay is calibrated based on the current HCLK frequency.
