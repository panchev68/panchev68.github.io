Lib::BSP::Ethercat::PDI::LAN9252::Interface Class Reference

Detailed Description

Interface class for the LAN9252 EtherCAT Slave Controller.

This class provides methods to interact with the LAN9252 EtherCAT Slave Controller using SPI. It includes methods to open, close, select, and deselect the device, as well as read and write data to and from the device using SPI.
