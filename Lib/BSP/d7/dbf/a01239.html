Detailed Description

Frame0x09 class for Queue Local AT Command Request.

Represents a Digi XBee API frame for queuing AT command parameters on the local device. Queued values are not applied until a Local AT Command Request (0x08) or AC command is issued. Querying behaves as Local AT Command Request (0x08). The response frame is Local AT Command Response (0x88).
