Detailed Description

Provides string manipulation utilities for case conversion, trimming, and array search.

This file defines the StringHelper class, which offers static methods for converting string case, trimming whitespace, and searching for strings within null-terminated arrays. The implementation uses standard algorithms and input validation to ensure safe and efficient string operations. Suitable for embedded and general-purpose applications requiring robust string handling.
