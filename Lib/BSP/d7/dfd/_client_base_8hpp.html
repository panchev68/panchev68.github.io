Detailed Description

CAN bus client service class.

ClientBase class for Digi XBee API.

Provides client-side CAN bus communication, including message sending, diagnostics, and event handling. Inherits from ServiceBase and specializes for client mode. Enables sending messages, handling received events, and error processing for CAN bus clients in embedded systems. Designed for extensibility and robust communication, supporting diagnostics and reliable message transmission.

Provides a base class for Digi XBee client applications, including message handling, connection management, and task lifecycle integration. The ClientBase class encapsulates message processing, connection state transitions, and task lifecycle hooks, supporting robust client implementations for Digi XBee modules in embedded and general-purpose applications. The implementation emphasizes type safety, efficient resource management, and extensibility, allowing seamless integration with Digi XBee API infrastructure and RTOS environments.
