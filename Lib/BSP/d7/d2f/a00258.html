Detailed Description

DigitalPeripheralBase class.

This class serves as the base class for digital peripheral ports. It provides configuration structures, update modes, and virtual functions for port initialization, updates, and SPI handle retrieval.

The DigitalPeripheralBase class contains nested structures for configuration, including inputs and outputs, and an enumeration for update modes. It also defines a PortBase class with pure virtual functions that must be implemented by derived classes. Additionally, it provides protected virtual functions for handling pre-update, post-update, and test output events, which must be implemented in the owner class.

Note
The class also contains a private friend class Test and a template function for task execution based on the update mode.

Initializes the digital peripheral.

This function initializes the digital peripheral by setting up the inputs and outputs.

This function sets up the digital peripheral by calling the setup functions for inputs and outputs. It then selects the operating mode based on the SPI port configuration and creates a task to run the appropriate task function for the selected mode.
