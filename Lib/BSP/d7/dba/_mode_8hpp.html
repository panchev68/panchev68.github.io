Detailed Description

TIM mode enumeration for STM32 TIM peripherals.

This file defines the Mode enumeration and helpers for STM32 TIM peripherals, supporting type-safe selection, compile-time enumeration, and clear code semantics. It is intended for embedded STM32 HAL applications, ensuring compatibility with C++17 standards and project documentation guidelines.
