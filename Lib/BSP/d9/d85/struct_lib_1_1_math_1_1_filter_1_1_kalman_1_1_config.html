template<typename T = double>
struct Lib::Math::Filter::Kalman< T >::Config

Configuration parameters for Kalman filter.

The Config struct holds process noise, sensor noise, estimated error, and initial value. It allows fine-tuning of filter behavior for different sensor and signal characteristics.
