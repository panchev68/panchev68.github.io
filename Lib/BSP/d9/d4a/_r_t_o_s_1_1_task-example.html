RTOS::Task

Example usage of RTOS::Task for FreeRTOS thread management.

Demonstrates how to create, start, suspend, resume, and notify a FreeRTOS task using the RTOS::Task class.

Example usage:

#include "Lib/RTOS/Task.hpp"
void workerFunction() {
// Task logic here
for (int i = 0; i < 10; ++i) {
// Simulate work
RTOS::Task::delay(100);
}
}
int main() {
RTOS::Task myTask("Worker", 256, RTOS::Task::DEFAULT_TASK_PRIORITY + 1);
RTOS::Task::Handle handle = myTask.start(workerFunction);
if (myTask.isValid()) {
myTask.suspend();
RTOS::Task::delay(500);
myTask.resume();
}
return 0;
}
