Подробно описание

DeviceBase abstract base class for managed devices.

DeviceBase provides the interface for device setup and tick event handling. Devices are automatically registered with the DeviceManager upon construction and deregistered upon destruction. Derived classes must implement the handleTickEvent method to respond to tick events.
