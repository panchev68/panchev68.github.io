TimeZone

Example usage of TimeZone class.

Demonstrates how to create a TimeZone object, set its properties, and convert local time to UTC minutes.

Example usage:

#include "Lib/Helper/TimeZone.hpp"
int main() {
TimeZone tz("Europe/Sofia", 120); // 120 minutes offset
tz.setName("Europe/Berlin");
tz.setOffsetMinutes(60);
int utcMinutes = tz.convertLocalToUtcMinutes(2024, 6, 1, 12, 0, 0);
// Use utcMinutes for further processing
return 0;
}
