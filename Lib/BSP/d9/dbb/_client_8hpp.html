Подробно описание

Wiznet TCP client base class for Ethernet communication.

Provides connection management, message handling, and diagnostic tracking for embedded TCP clients. Integrates with Wiznet W5500 driver, FreeRTOS, and custom Node/DataHandle types. Ensures type safety, extensibility, and resource management for client operations. This class supports connection state management, message handling, diagnostic counters, and LED callbacks. It is designed for extensibility and integration with embedded networking stacks and RTOS environments.
