Detailed Description

Provides precise microsecond delays using the DWT cycle counter.

The DelayUs class enables accurate microsecond delays by configuring the ARM Cortex-M DWT unit and calculating the required cycle count based on the system clock frequency. Intended for use in embedded systems where precise timing is required; should not be used in interrupt or real-time critical paths due to its blocking nature.
