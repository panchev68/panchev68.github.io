Lib::BSP::DeviceManager Class Reference

Detailed Description

DeviceManager class.

Device Manager class

Organizes an array of devices on which the handleTickEvent function is launched every 1 ms. Before using the class, it must be initialized with the setup function. It starts a timer that serves the device updates.
