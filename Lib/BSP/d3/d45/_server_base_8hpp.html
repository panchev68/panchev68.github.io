Detailed Description

CAN bus server service specialization for SERVER mode.

XBee server base class for Digi XBee API.

Specializes the Service class template for Mode::SERVER, providing server-side CAN bus communication, node scanning, and diagnostics.

This file provides the ServerBase class, a robust base for XBee server implementations. The ServerBase class manages message handling, connection management, and a transaction state machine for Digi XBee API communication. It supports efficient message transmission, reception, and connection state transitions, ensuring reliable operation in embedded and general-purpose applications. The implementation emphasizes type safety, performance, and extensibility, allowing derived classes to customize protocol logic and event handling. Designed for modularity and maintainability, it integrates seamlessly with Digi XBee API infrastructure and RTOS environments.
