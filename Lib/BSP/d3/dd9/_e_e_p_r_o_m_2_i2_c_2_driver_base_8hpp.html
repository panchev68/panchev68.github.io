Detailed Description

Base class for I2C EEPROM drivers.

This file defines the DriverBase class, providing common functionality for interacting with I2C EEPROM devices over an I2C interface. It includes configuration, open/close, read/write, and address handling for EEPROM chips, supporting both 8-bit and 16-bit memory addressing.
