Frame0x17: Remote AT Command Request for Digi Core.

Provides the Frame0x17 class for querying or setting AT command parameters on a remote device. This class encapsulates Digi XBee API frame construction for remote AT command requests, supporting secured remote configuration through Secure Session. For parameter changes to take effect, set the Apply Changes option or send an AC command to the remote device. The response frame for this request is Remote AT Command Response (Frame 0x97). The design ensures type safety, extensibility, and efficient integration with Digi XBee API infrastructure and RTOS environments. It supports flexible payloads and transmission options for embedded and general-purpose applications, facilitating reliable communication and host correlation.
