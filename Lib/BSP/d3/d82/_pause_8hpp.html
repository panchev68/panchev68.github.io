Pause interface for Digi XBee API.

This file provides the Pause base class, an abstract interface for pause and resume services in Digi XBee modules. The Pause class enables derived classes to implement custom logic for pausing and resuming operations, supporting modular and extensible control of Digi XBee API components. Designed for maintainability and integration with Digi XBee infrastructure, it ensures robust resource management and type safety in embedded and general-purpose applications.
