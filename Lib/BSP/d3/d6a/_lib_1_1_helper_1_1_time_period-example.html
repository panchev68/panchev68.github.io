Lib::Helper::TimePeriod

Example of using TimePeriod for measuring elapsed ticks.

This example demonstrates how to use the TimePeriod class to measure the elapsed time between two events using FreeRTOS ticks. The measured tick count is printed to the standard output.

Example usage:
#include <Lib/Helper/TimePeriod.hpp>
#include <cstdio>
#include <thread>
#include <chrono>
int main() {
Lib::Helper::TimePeriod timer;
timer.start();
// Simulate some work with a delay
std::this_thread::sleep_for(std::chrono::milliseconds(150));
uint32_t elapsedTicks = 0;
if (timer.stop(elapsedTicks)) {
std::printf("Elapsed ticks: %lu\n", static_cast<unsigned long>(elapsedTicks));
} else {
std::printf("Timer was not started.\n");
}
return 0;
}
