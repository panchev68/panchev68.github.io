Подробно описание

ResponseEvents class for Digi XBee API protocol response events.

Provides event hooks for all supported Digi XBee API response frames, enabling flexible handling of Digi XBee API responses in embedded and general-purpose applications. The ResponseEvents class encapsulates event-driven mechanisms for each supported response frame type, allowing users to register custom handlers for received packets, AT command responses, transmit status updates, modem status changes, and more. This design promotes extensibility, type safety, and efficient integration with Digi XBee API infrastructure and RTOS environments.
