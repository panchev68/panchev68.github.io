Подробно описание

Header.hpp for Digi Core Frame Xbee API header struct.

Provides the packed header structure for Xbee API frames, including start delimiter and length fields. The Header struct is used for parsing and constructing Digi XBee API frames, ensuring correct framing and length encoding for communication with Digi modules. This design supports efficient, type-safe integration with Digi XBee API infrastructure and RTOS environments, promoting maintainability and extensibility in embedded applications.
