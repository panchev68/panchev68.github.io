RTOS::Mutex

Example usage of RTOS::Mutex and MutexGuard.

Demonstrates creation, locking, and scoped management of a FreeRTOS mutex using the C++ RAII wrapper.

Example usage:
#include "Mutex.hpp"
void exampleTask() {
RTOS::Mutex mutex(RTOS::Mutex::MutexType::NORMAL);
if (mutex.create()) {
{
MutexGuard guard(mutex);
if (guard) {
// Critical section protected by mutex.
}
} // Mutex automatically unlocked here.
mutex.destroy();
}
}
