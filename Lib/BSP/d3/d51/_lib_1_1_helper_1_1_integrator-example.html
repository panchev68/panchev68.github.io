Lib::Helper::Integrator

Example usage of the Integrator template class for computing a moving average.

Example usage of the Integrator template class for computing a moving average.

Example usage:

#include "Lib/Helper/Integrator.hpp"
int main() {
Lib::Helper::Integrator<int, 4> integrator;
integrator.addValue(10);
integrator.addValue(20);
integrator.addValue(30);
integrator.addValue(40);
int avg = integrator.computeResult(); // avg == 25
integrator.addValue(50);
avg = integrator.computeResult(); // avg == 35
integrator.resetWindow();
avg = integrator.computeResult(); // avg == 0
return 0;
}

This example demonstrates how to use the Integrator class to calculate a moving average over a window of four integer values. After adding four values, the average is computed. When a fifth value is added, the oldest value is replaced, and the average is updated. The window can be reset at any time.
