Example usage of Crc16 class for CRC-16/CCITT-FALSE calculation.

Example usage of Crc16 class for CRC-16/CCITT-FALSE calculation.

Example usage:
#include "Lib/Helper/crc16.hpp"
#include <vector>
#include <iostream>
int main() {
std::vector<uint8_t> buffer = {0x01, 0x02, 0x03, 0x04};
Lib::Helper::Crc16 crc(buffer);
uint16_t value = crc;
std::cout << "CRC-16: " << std::hex << value << std::endl;
return 0;
}
