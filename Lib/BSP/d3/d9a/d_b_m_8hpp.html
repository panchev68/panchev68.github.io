Подробно описание

Provides conversion functions for dBM values.

This file defines the dBM class, which offers utility functions for converting dBM values to percentages using both linear and lookup table methods. The class is non-instantiable and provides only static methods, making it suitable for embedded systems and applications requiring efficient dBM to percentage conversions.
