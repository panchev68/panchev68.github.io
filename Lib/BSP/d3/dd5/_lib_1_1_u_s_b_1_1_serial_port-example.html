Lib::USB::SerialPort

Example usage of SerialPort for USB communication.

Demonstrates how to configure, open, send data, and close a SerialPort instance for USB serial communication in an embedded STM32 system.

Example usage:

#include <Lib/USB/SerialPort.hpp>
#include <Lib/HAL/UART.hpp>
#include <Lib/HAL/GPIO.hpp>
HAL::UART uartRx;
HAL::UART uartTx;
PinOutput pinNRST(GPIOA, 1);
int main() {
Lib::USB::SerialPort::Config config{uartRx, uartTx, pinNRST};
Lib::USB::SerialPort serialPort(config);
if (serialPort.open()) {
uint8_t data[] = {0x55, 0xAA};
serialPort.send(data, sizeof(data));
serialPort.close();
}
return 0;
}
