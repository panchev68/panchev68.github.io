RingBuffer

Example demonstrating usage of the RingBuffer class for integer FIFO storage.

Example demonstrating usage of the RingBuffer class for integer FIFO storage.This example shows how to push and pop elements in a fixed-size ring buffer.

#include "Lib/Helper/RingBuffer.hpp"
#include <iostream>
int main() {
Lib::Helper::RingBuffer<int, 4> buffer;
buffer.push(10);
buffer.push(20);
buffer.push(30);
int value = 0;
while (buffer.pop(value)) {
std::cout << "Popped: " << value << std::endl;
}
return 0;
}
