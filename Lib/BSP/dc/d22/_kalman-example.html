Kalman Filter Example

This example demonstrates how to use the Kalman filter to smooth sensor data.

Example usage:

#include "Lib/Math/Filter/Kalman.hpp"
int main() {
Lib::Math::Filter::Kalman<double> filter({0.00001, 0.004, 1.0, 0.0});
double sensorValue = 10.5;
double filtered = filter(sensorValue);
return 0;
}
