Detailed Description

Base class for CAN bus services.

Provides start/stop, send/receive, diagnostics, and event handling for CAN messages. Inherits from Lib::HAL::CAN::CanPortBase and uses FreeRTOS for task and queue management. The CanServiceBase class enables robust CAN bus communication, diagnostics tracking, and extensibility for custom service implementations. Designed for embedded systems, it ensures reliable message transmission, error handling, and event processing, supporting maintainable and type-safe CAN bus service development.
