Lib::BSP::DigitalPeripheralBase::Inputs Class Reference

Detailed Description

DigitalPeripheralBase::Inputs class.

This class provides an interface for handling input data for the digital peripheral. It includes functions for setting up the inputs, updating the inputs, and getting the input value. The class also includes an event for when the input value changes.
