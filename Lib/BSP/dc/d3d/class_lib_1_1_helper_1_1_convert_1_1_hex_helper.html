Detailed Description

Utility class for hexadecimal and BCD conversions.

The HexHelper class provides static methods for converting between hexadecimal, ASCII, and BCD formats. It supports conversion of 4-bit values to ASCII hexadecimal characters, conversion of 8-bit and 16-bit unsigned integers to BCD format, and conversion of ASCII hexadecimal characters to their corresponding 4-bit values.
