Подробно описание

XBee message container class for Digi Core.

Provides the Message class for storing and managing fixed-size XBee frame data. This type supports assignment from buffers and conversion to std::vector, ensuring efficient and type-safe handling of message data for embedded communication. The container uses fixed-width integer types and standard containers for reliability and maintainability. It is designed for robust embedded use, with bounded operations and input validation.
