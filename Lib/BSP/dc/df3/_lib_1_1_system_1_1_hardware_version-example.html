Lib::System::HardwareVersion

Example usage of HardwareVersion.

Demonstrates how to create and use the HardwareVersion class for hardware version and sub-version management in embedded systems.

Example usage:

#include "Lib/System/HardwareVersion.hpp"
#include <iostream>
int main() {
Lib::System::HardwareVersion hwVersion("HW-1.2", 5);
std::cout << "Hardware Version: " << hwVersion.toString() << std::endl;
std::cout << "Version Value: " << static_cast<uint32_t>(hwVersion) << std::endl;
std::cout << "Sub-version: " << hwVersion.getSubVersion() << std::endl;
return 0;
}
