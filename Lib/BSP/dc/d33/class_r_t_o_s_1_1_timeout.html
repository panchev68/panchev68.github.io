Detailed Description

Timeout class for managing tick-based timeouts in FreeRTOS.

Provides methods for setting, extending, resetting, and querying timeouts in ticks, milliseconds, and seconds. Supports disabled and immediate timeouts, and offers conversion and status utilities.
