SemaphoreCounting

Example usage of RTOS::SemaphoreCounting for resource pooling and multi-token synchronization in FreeRTOS.

Example usage of RTOS::SemaphoreCounting for resource pooling and multi-token synchronization in FreeRTOS.This example demonstrates how to create a counting semaphore, acquire and release tokens, and query the current count and capacity. It shows how to initialize a counting semaphore with a maximum and initial count, take and give tokens, and check fullness or emptiness for robust resource management in a real-time STM32 application.

Example usage:
#include "Lib/RTOS/SemaphoreCounting.hpp"
#include <cstdint>
int main() {
// Create a counting semaphore with max count 5 and initial count 2
RTOS::SemaphoreCounting semaphore(5, 2);
if (!semaphore.create()) {
// Failed to create semaphore
return -1;
}
// Take a token from the semaphore
if (semaphore.take()) {
// Token acquired, do work
// ...
// Release (give) the token back
semaphore.give();
}
// Query current count and capacity
auto count = semaphore.getCount();
auto remaining = semaphore.getRemainingCapacity();
// Check if semaphore is full or empty
if (semaphore.isAtMaxCount()) {
// All tokens available
} else if (semaphore.isEmpty()) {
// No tokens available
}
return 0;
}
