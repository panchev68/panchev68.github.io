Подробно описание

Moon phase calculation utility.

The MoonPhase class provides methods for calculating the phase of the moon for a given date. It computes the phase index and name using astronomical algorithms and allows retrieval of the phase name and index. This class is suitable for embedded and real-time applications requiring lunar phase information, such as calendar utilities and astronomical tools.
