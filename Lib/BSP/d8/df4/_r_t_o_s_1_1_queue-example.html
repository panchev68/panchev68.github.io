RTOS::Queue

Example usage of RTOS::Queue with standard output.

Demonstrates creation, sending, receiving, and querying of a FreeRTOS queue using the RTOS::Queue wrapper.

Example usage:

#include <iostream>
#include "Lib/RTOS/Queue.hpp"
int main() {
RTOS::Queue<int, 4> queue;
if (!queue.create()) {
std::cout << "Failed to create queue." << std::endl;
return 1;
}
int value = 42;
if (queue.send(value)) {
std::cout << "Sent value: " << value << std::endl;
}
int received = 0;
if (queue.receive(received)) {
std::cout << "Received value: " << received << std::endl;
}
std::cout << "Messages waiting: " << queue.messagesWaiting() << std::endl;
queue.destroy();
return 0;
}
