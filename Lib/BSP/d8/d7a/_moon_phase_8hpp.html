Detailed Description

Moon phase calculation utility.

This file defines the MoonPhase class, which provides functionality for calculating the phase of the moon for a given date. The class computes the moon phase index and name using astronomical algorithms and supports retrieval of the phase name and index. It is suitable for embedded and real-time applications requiring lunar phase information, such as calendar utilities and astronomical tools.
