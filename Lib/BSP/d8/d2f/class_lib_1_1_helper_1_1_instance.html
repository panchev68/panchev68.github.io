Подробно описание

template<typename T>
class Lib::Helper::Instance< T >

Singleton instance helper for type T.

Provides a singleton instance of type T, constructed with default constructor. Ensures only one instance exists per type. The instance is heap-allocated and accessible via conversion operator.

Шаблона Параметри
TType for singleton instance.
