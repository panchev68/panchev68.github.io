RTOS::Timer

Example usage of RTOS::Timer class.

Demonstrates creation, configuration, and usage of FreeRTOS software timers via the RTOS::Timer abstraction.

Example usage:
#include "Lib/RTOS/Timer.hpp"
void onTimeout() {
// Timer callback logic
}
int main() {
// Create a one-shot timer with 1000 ms period
auto timer = RTOS::Timer::createMs("MyTimer", onTimeout, 1000, RTOS::Timer::Mode::ONE_SHOT);
if (timer && timer->start()) {
// Timer started successfully
}
// ... application logic ...
return 0;
}
