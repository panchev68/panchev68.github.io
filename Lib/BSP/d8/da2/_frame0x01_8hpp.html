Frame0x01: 16-bit Transmit Request for Digi Core.

Provides the Frame0x01 class for sending serial payload data as an RF packet to a remote device with a 16-bit network address. This class encapsulates Digi XBee API frame construction for transmitting data to a remote device using a 16-bit network address. The response frame for this request is Transmit Status (Frame 0x89). Deprecated for legacy use; use Frame 0x10 for new designs. The design ensures type safety, extensibility, and efficient integration with Digi XBee API infrastructure and RTOS environments. It supports flexible payloads and transmission options for embedded and general-purpose applications, facilitating reliable communication and host correlation.
