Detailed Description

template<class T>
class Lib::Helper::JaggedArrayBase< T >

JaggedArrayBase class.

Base class for managing a collection of pointers to objects of type T. Provides methods for adding, removing, and accessing elements.

Template Parameters
TThe type of elements stored in the array.
