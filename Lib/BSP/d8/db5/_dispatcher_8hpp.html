Подробно описание

Manages error instances and their states.

Provides a singleton dispatcher for managing error instances, tracking their states, and notifying observers on status changes. The Dispatcher class is responsible for maintaining a collection of error objects, monitoring their states, and invoking registered callbacks when error status changes occur. It integrates with RTOS tick services to periodically check and update error states, ensuring timely notification and management of error conditions within the system. The class supports error grouping, observer notification, and efficient state tracking for robust error handling in embedded applications.
