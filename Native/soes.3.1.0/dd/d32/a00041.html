Detailed Description

Header file for File over EtherCAT (FoE) protocol implementation.

This file contains the definitions and function declarations for handling File over EtherCAT (FoE) protocol operations. FoE is a simple protocol used for file transfer over EtherCAT.

The functions provided in this file allow for sending and receiving files over an EtherCAT network, handling FoE requests and responses, and managing FoE error handling.
