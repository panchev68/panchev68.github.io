◆ sizeOfPDO()

uint16_t sizeOfPDO ( uint16_t index,
int * nmappings,
_SMmap * mappings,
int max_mappings )

Calculate the size in Bytes of RxPDO or TxPDOs by adding the objects in SyncManager SDO 1C1x.

A list of mapped objects is created for fast lookup of dynamically mapped process data. The max size of the list (max_mappings) can be set to 0 if dynamic processdata is not supported.

The output variable nmappings is set to 0 if dynamic processdata is not supported. It is set to the number of mapped objects if dynamic processdata is supported, or -1 if the mapping was incorrect.

Parameters
[in]index= SM index
[out]nmappings= number of mapped objects in SM, or -1 if mapping is invalid
[out]mappings= list of mapped objects in SM
[out]max_mappings= max number of mapped objects in SM
Returns
size of RxPDO or TxPDOs in Bytes.
