Detailed Description

Header file for EtherCAT over EtherCAT (EoE) protocol implementation.

This file contains the definitions and function declarations for handling the EtherCAT over EtherCAT (EoE) protocol. EoE allows Ethernet frames to be tunneled over EtherCAT, enabling standard Ethernet communication over an EtherCAT network.

The functions and definitions provided in this file facilitate the initialization, processing, and management of EoE frames within an EtherCAT slave device.

Note
This file is part of the EtherCAT Slave Stack Code (SSC) and is intended for use with EtherCAT slave devices.

void ESC_eoeprocess ( void )

Processes the EtherCAT over EtherType (EoE) protocol.

This function handles the processing of the EoE protocol, which is used for transmitting Ethernet frames over EtherCAT. It is responsible for managing the EoE communication, including sending and receiving Ethernet frames encapsulated in EtherCAT frames.

Main EoE receive function checking the status on current mailbox buffers carrying data, distributing the mailboxes to appropriate EOE functions depending on requested frametype.
