Detailed Description

A union representing the EEPROM configuration.

ECAT EEPROM configuration area data structure

The EEPROM configuration area is 16 bytes long and is used to store the configuration of the EtherCAT slave device. The configuration area is divided into 8 16-bit words. The configuration area is used to store the following information:

  • PDI control register
  • PDI configuration register
  • Sync impulse length
  • PDI configuration 2 register
  • Configured station alias
  • Reserved
  • Checksum
