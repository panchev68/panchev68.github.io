Flyweight Design Pattern

Въведение

Flyweight (Лекотоварен) е структурен дизайн патърн, който позволява спестяване на памет чрез споделяне на общи части от състоянието между множество обекти. Вместо всеки обект да съхранява цялото си състояние, някои части от състоянието се споделят между обектите.

Пример

Ето един пример за имплементация на Flyweight патърна в C++:

#include <iostream>
#include <unordered_map>
#include <memory>
// Flyweight клас
class Flyweight {
public:
Flyweight(const std::string& sharedState) : sharedState_(sharedState) {}
void operation(const std::string& uniqueState) const {
std::cout << "Flyweight: Displaying shared (" << sharedState_ << ") and unique (" << uniqueState << ") state.\n";
}
private:
std::string sharedState_;
};
// Flyweight фабрика
class FlyweightFactory {
public:
std::shared_ptr<Flyweight> getFlyweight(const std::string& sharedState) {
if (flyweights_.find(sharedState) == flyweights_.end()) {
flyweights_[sharedState] = std::make_shared<Flyweight>(sharedState);
}
return flyweights_[sharedState];
}
private:
std::unordered_map<std::string, std::shared_ptr<Flyweight>> flyweights_;
};
int main() {
FlyweightFactory factory;
std::shared_ptr<Flyweight> flyweight1 = factory.getFlyweight("SharedState1");
flyweight1->operation("UniqueState1");
std::shared_ptr<Flyweight> flyweight2 = factory.getFlyweight("SharedState1");
flyweight2->operation("UniqueState2");
std::shared_ptr<Flyweight> flyweight3 = factory.getFlyweight("SharedState2");
flyweight3->operation("UniqueState3");
return 0;
}

В този пример, Flyweight класът съдържа споделено състояние, което се използва от множество обекти. FlyweightFactory класът управлява създаването и споделянето на Flyweight обекти. В main функцията се създават няколко Flyweight обекта с различно уникално състояние, но със споделено състояние.
