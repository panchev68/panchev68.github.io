Copy Constructor in C++

В тази страница ще разгледаме копиращия конструктор в C++.

Какво е копиращ конструктор?

Копиращият конструктор е специален конструктор, който се използва за създаване на нов обект като копие на съществуващ обект. Той се извиква, когато нов обект се инициализира с друг обект от същия клас.

Синтаксис на копиращ конструктор:

class MyClass {
public:
MyClass(const MyClass& other) {
// Тяло на копиращия конструктор
}
};

Пример за копиращ конструктор:

#include <iostream>
using namespace std;
class MyClass {
public:
int x;
MyClass(int val) : x(val) {}
// Копиращ конструктор
MyClass(const MyClass& other) : x(other.x) {
cout << "Копиращ конструктор е извикан" << endl;
}
};
int main() {
MyClass obj1(10);
MyClass obj2 = obj1; // Копиращият конструктор се извиква тук
cout << "obj1.x: " << obj1.x << endl; // Output: obj1.x: 10
cout << "obj2.x: " << obj2.x << endl; // Output: obj2.x: 10
return 0;
}

В този пример, когато obj2 се инициализира с obj1, копиращият конструктор се извиква и създава нов обект като копие на obj1.

Заключение:

Копиращият конструктор е важен механизъм в C++, който позволява създаването на нови обекти като копия на съществуващи обекти. Той е полезен за управление на ресурси и избягване на проблеми с дълбокото и плиткото копиране.
