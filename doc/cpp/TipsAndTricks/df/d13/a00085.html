Flyweight Design Pattern

Въведение

Патернът Flyweight е структурен дизайн патерн, който позволява спестяване на памет чрез споделяне на общи части от състоянието между множество обекти. Това е особено полезно, когато имаме голям брой обекти, които споделят значителна част от своето състояние.

Пример

Ето един пример за имплементация на патерна Flyweight в C++:

#include <iostream>
#include <unordered_map>
#include <string>
// Flyweight клас
class Flyweight {
public:
Flyweight(const std::string &sharedState) : sharedState_(sharedState) {}
void operation(const std::string &uniqueState) const {
std::cout << "Flyweight: Displaying shared (" << sharedState_ << ") and unique (" << uniqueState << ") state.\n";
}
private:
std::string sharedState_;
};
// FlyweightFactory клас
class FlyweightFactory {
public:
Flyweight *getFlyweight(const std::string &sharedState) {
if (flyweights_.find(sharedState) == flyweights_.end()) {
flyweights_[sharedState] = new Flyweight(sharedState);
}
return flyweights_[sharedState];
}
~FlyweightFactory() {
for (auto &pair : flyweights_) {
delete pair.second;
}
}
private:
std::unordered_map<std::string, Flyweight *> flyweights_;
};
int main() {
FlyweightFactory factory;
Flyweight *fw1 = factory.getFlyweight("SharedState1");
fw1->operation("UniqueStateA");
Flyweight *fw2 = factory.getFlyweight("SharedState1");
fw2->operation("UniqueStateB");
Flyweight *fw3 = factory.getFlyweight("SharedState2");
fw3->operation("UniqueStateC");
return 0;
}

Този код демонстрира имплементация на патерна Flyweight. Класът Flyweight представлява обект, който споделя част от своето състояние. Класът FlyweightFactory създава и управлява обектите Flyweight. В main() функцията се създават няколко обекта Flyweight, които споделят част от своето състояние.

Този код ще изведе следното: Flyweight: Displaying shared (SharedState1) and unique (UniqueStateA) state. Flyweight: Displaying shared (SharedState1) and unique (UniqueStateB) state. Flyweight: Displaying shared (SharedState2) and unique (UniqueStateC) state.
