Factory Method Design Pattern

Въведение

Фабричният метод е създаващ дизайн патърн, който дефинира интерфейс за създаване на обекти, но позволява на подкласовете да променят типа на създаваните обекти. С този патърн, класът делегира инстанцирането на обекти на подкласовете.

Пример

Ето един пример за имплементация на Фабричния метод в C++:

#include <iostream>
#include <memory>
// Продукт
class Product {
public:
virtual ~Product() = default;
virtual void use() const = 0;
};
// Конкретен продукт A
class ConcreteProductA : public Product {
public:
void use() const override {
std::cout << "Using ConcreteProductA\n";
}
};
// Конкретен продукт B
class ConcreteProductB : public Product {
public:
void use() const override {
std::cout << "Using ConcreteProductB\n";
}
};
// Създател
class Creator {
public:
virtual ~Creator() = default;
virtual std::unique_ptr<Product> factoryMethod() const = 0;
void someOperation() const {
std::unique_ptr<Product> product = factoryMethod();
product->use();
}
};
// Конкретен създател A
class ConcreteCreatorA : public Creator {
public:
std::unique_ptr<Product> factoryMethod() const override {
return std::make_unique<ConcreteProductA>();
}
};
// Конкретен създател B
class ConcreteCreatorB : public Creator {
public:
std::unique_ptr<Product> factoryMethod() const override {
return std::make_unique<ConcreteProductB>();
}
};
int main() {
std::unique_ptr<Creator> creatorA = std::make_unique<ConcreteCreatorA>();
creatorA->someOperation();
std::unique_ptr<Creator> creatorB = std::make_unique<ConcreteCreatorB>();
creatorB->someOperation();
return 0;
}

В този пример, Product е интерфейсът, който дефинира метода use. ConcreteProductA и ConcreteProductB са конкретни продукти, които имплементират този интерфейс. Creator е абстрактен клас, който декларира фабричния метод factoryMethod. ConcreteCreatorA и ConcreteCreatorB са конкретни създатели, които имплементират фабричния метод, за да създават съответно ConcreteProductA и ConcreteProductB.

Този патърн позволява на подкласовете да променят типа на създаваните обекти, като променят само фабричния метод.
